# 00.17 C035地皮价格权威边界记录 v1
> 回写日期：2026-06-18

## 裁决

- 9.2商品清单是地皮基础价唯一权威源。
- 8.3只维护“拥有8块地皮”成就带来的800星辉折扣价。
- 9.1只维护难度倍率和最终结算顺序。
- 1.1只维护地皮购买、揭示与开发行为，不再复制价格数字。

## 结算顺序

```text
9.2基础价
-> 8.3成就折扣（若已解锁）
-> 9.1商店购买难度倍率
-> 最终应付星辉
```

特殊地皮与普通地皮使用同一价格流程；侦察费用是独立操作，不属于地皮购买价格。
