# 00.18 C020数值权威与受控摘要治理记录 v1
> 回写日期：2026-06-18

## 裁决

- 同一数值只能有一个权威源；其他文件只描述流程、引用权威源或保留可自动校验的受控摘要。
- 受控摘要不得引入新公式、新字段或覆盖权威值；无法自动校验的重复完整数值应删除并改为引用。
- DeepSeek可以维护受控摘要和执行校验，但数值权威边界、公式与玩法含义由核心框架设计者裁决。

## 权威边界

| 机制 | 唯一权威源 | 其他文件允许内容 |
|------|-----------|----------------|
| 建筑基础数值 | 6.1建筑数值表 | 建筑详情中的快速摘要，必须通过自动校验 |
| 建筑星辉解锁价 | 9.2商品清单 | 建筑详情frontmatter中的受控摘要 |
| NPC基础值、职业倍率、多职业倍率、属性适配公式 | 6.2 NPC数值表 | 3.1/3.3/3.5/3.6仅维护规则、流程和映射 |
| 游戏时间与昼夜阶段 | 1.3时间系统规则 | 其他系统仅引用时间事件和阶段 |
| 怪物、敌袭与防御规则 | 7.2怪物与防御系统 | 6.3仅维护难度参数及其引用关系 |
| 局内任务目标与奖励 | 10.2任务清单 | UI和流程文件只展示或引用 |
| 跨局挑战与成就解锁 | 8.3挑战模式与成就解锁系统 | 与局内任务含义重叠的部分留给C015/C037裁决 |
