# 垃圾机制

### 垃圾产生
- [x] 垃圾产生来源：生产建筑运行、NPC死亡
- [x] 单位时间垃圾产生量：每座运行中的建筑产生 **1 垃圾/游戏日** + 每名NPC产生 **0.5 垃圾/游戏日**
  - 建筑闲置（无工人）不产生垃圾
  - NPC死亡时额外产生 1 垃圾
- [x] 总垃圾产生公式：运行建筑数 + NPC数×0.5 / 游戏日
- [x] 垃圾处理参考：1名清洁工可处理约 5-8 座运行建筑的垃圾产出

### 垃圾不处理的负面效果
- [x] 垃圾未处理时：每日占用全局存储空间（垃圾=存储类资源，上限500）
- [x] 垃圾堆积超过200：NPC满意度每日 -1%（全员工作效率 -1%，可累积至 -20%）
- [x] 垃圾堆积超过400：NPC满意度每日 -2%，疾病概率 +5%/日
- [x] 垃圾满500后：溢出垃圾丢弃，但负面效果持续累积
- [x] 垃圾清零后：满意度负面效果立即消除，疾病概率次日恢复

### 垃圾处理场
- [x] 处理模式：4种模式可手动切换
  1. **直接销毁**：垃圾直接清除，无产出，处理速度最快
  2. **资源回收**：垃圾转化为少量随机资源（木材 / 石头 / 金属之一）
  3. **堆肥制肥**：垃圾转化为肥料，可加速农田产出
  4. **燃料提炼**：垃圾转化为燃料，可提升熔炉房效率
- [x] 垃圾处理场单周期处理量：**6 垃圾/游戏日**（1工人基础值）
- [x] 各模式处理效率与产出数值：

| 处理模式 | 处理速度 | 每1垃圾产出 | 备注 |
|---------|---------|------------|------|
| A-销毁 | 10 垃圾/日 | 无 | 最快清除，无收益 |
| B-回收 | 6 垃圾/日 | 回收1随机资源(木材40%/石头40%/金属20%) | 资源循环 |
| C-堆肥 | 6 垃圾/日 | 产出1肥料（可使1座农田下次产出+50%）+ 每5垃圾额外产出1染料 | 农业增益+染料副产 |
| D-提炼 | 6 垃圾/日 | 产出0.3燃料 | 工业增益 |

### 存储规则
- [x] 垃圾可堆积：是
- [x] 垃圾堆积上限：500（全局仓库内）
- [x] 垃圾超出上限后：溢出自动丢弃
- [x] 垃圾存储不影响其他资源的分类上限

### 与存储告警联动
- [x] 垃圾达400时触发存储黄色告警（同全局存储80%）
- [x] 垃圾达500时触发存储红色告警（同全局存储90%）
- [x] 详见 **4.3 存储规则** § 存储告警系统
