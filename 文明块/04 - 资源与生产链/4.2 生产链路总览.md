# 生产链路总览

## 生产链路
- 树场 → 木材
- 矿场 → 矿石
- 熔炉房：矿石 → 金属
- 农田 → 食物
- 水井 → 水
- 垃圾处理场（多模式可切换）：
  - 模式A：垃圾 → 直接销毁（无产出，处理速度快）
  - 模式B：垃圾 → 随机回收资源（木材/石头/金属之一，少量）
  - 模式C：垃圾 → 肥料（可加速农田产出）
  - 模式D：垃圾 → 燃料（可提升熔炉房效率）

## 生产链路（交叉依赖）
<!-- 基础资源生产 -->
- 树场 → 木材 → 建造/维修
- 采掘场 → 矿石+石头 → 建造/加工
- 农田 → 食物 → NPC消耗
  └─ 运行消耗：0.5水/日（灌溉）
- 水井 → 水 → NPC消耗+农田灌溉+疗愈所
- 熔炉房：矿石+燃料(1/日) → 金属
  └─ 运行消耗：1燃料/日
- 农舍：6食物/工人/日（取代农田的高效农业）
  └─ 运行消耗：1水/日

<!-- 垃圾与回收系统 -->
- 每座运营建筑 → 1垃圾/日 → 垃圾处理场
- 垃圾处理场（多模式）：
  ├→ 模式A：垃圾直接销毁
  ├→ 模式B：垃圾→随机回收资源(木/石/金属)
  ├→ 模式C：垃圾→肥料→加速农田
  └→ 模式D：垃圾→燃料→供应熔炉房/太阳灯

<!-- 科技与进阶 -->
- 研究站 → 科技点 → 科技树解锁
  └─ 运行消耗：0.5食+0.5水/工人/日
- 疗愈所 → 延长寿命+防疾病
  └─ 运行消耗：1食+1水/日
- 工坊 → 相邻建筑耐久损耗-50%
- 管道 → 跨层资源自动传输

<!-- 防御系统 -->
- 哨塔+守卫 → 夜间消灭怪物
- 城墙 → 阻挡怪物路径(飞行怪除外)
- 陷阱 → 一次性消灭经过怪物

## 消耗链路
- NPC生存消耗：0.5 食物 + 0.5 水 / 游戏日 / 人
- 建筑建造消耗：木材、石头、金属
- 建筑运行消耗：详见 6.1 建筑运行消耗表
