# 生产链路总览

## 生产链路
- 树场 → 木材
- 矿场 → 矿石
- 熔炉房：矿石 → 金属
- 农田 → 食物
- 水井 → 水
- 垃圾处理场（多模式可切换）：
  - 模式A：垃圾 → 直接销毁（无产出，处理速度快）
  - 模式B：垃圾 → 随机回收资源（木材/石头/金属之一，少量）
  - 模式C：垃圾 → 肥料（可加速农田产出）
  - 模式D：垃圾 → 燃料（可提升熔炉房效率）

## 生产链路（交叉依赖）
<!-- 基础资源生产 -->
- 树场 → 木材 → 建造/维修
- 采掘场 → 矿石+石头 → 建造/加工
- 农田 → 食物 → NPC消耗
  └─ 运行消耗：0.5水/日（灌溉）
- 水井 → 水 → NPC消耗+农田灌溉+疗愈所
- 熔炉房：矿石+燃料(1/日) → 金属
  └─ 运行消耗：1燃料/日
- 农舍：6食物/工人/日（取代农田的高效农业）
  └─ 运行消耗：1水/日

<!-- 垃圾与回收系统 -->
- 每座运营建筑 → 1垃圾/日 → 垃圾处理场
- 垃圾处理场（多模式）：
  ├→ 模式A：垃圾直接销毁
  ├→ 模式B：垃圾→随机回收资源(木/石/金属)
  ├→ 模式C：垃圾→肥料→加速农田
  └→ 模式D：垃圾→燃料→供应熔炉房/太阳灯

  └─ 运行消耗：0.5食+0.5水/工人/日
- 疗愈所 → 延长寿命+防疾病
  └─ 运行消耗：1食+1水/日
- 工坊 → 相邻建筑耐久损耗-50%
- 管道 → 跨层资源自动传输

<!-- 防御系统 -->
- 哨塔+守卫 → 夜间消灭怪物
- 城墙 → 阻挡怪物路径(飞行怪除外)
- 陷阱 → 一次性消灭经过怪物

## 燃料生产链

### 主要链路
垃圾 → 垃圾处理场模式D → 燃料

### 备用链路
2木材 → 树场应急炭化 → 1燃料
- 标准时间：1游戏日
- 工人：1名
- 解锁：树场建成后立即可用
- 科技要求：无
- 燃料输入：无
- 金属输入：无

### 链路定位
- 垃圾处理场是主要燃料来源，提供常规燃料供应。
- 树场应急炭化是备用来源，用于应急恢复和断供后恢复。
- 该路线的建筑建设（树场：10木材）和运行均不依赖燃料链产物。
- 两条路线产出同一种"燃料"资源。
- 木材不能直接替代燃料；必须先经过树场应急炭化才能成为燃料。
- 备用路线不需要燃料启动，可在无燃料时运行。
- C027树场应急炭化同样遵循本规则。

## 通用生产批次与连续生产

### 适用范围
适用于工坊、熔炉房、垃圾处理场、树场应急炭化及其他采用"输入资源→生产时间→批次产物"的生产建筑。若单个建筑详情存在明确特殊规则，特殊规则优先；未明确部分继续使用本通用规则。

### 生产槽
每个生产建筑默认维护一个生产槽、一个选中配方、一个当前批次、一个批次产物缓冲。多个独立生产区域的建筑（如垃圾处理场多模式）每个区域分别维护自己的生产槽状态。

### 单批与连续模式
单批生产：玩家主动开始一批，完成后生产槽回到空闲，不得自动启动下一批。
连续生产：玩家开启连续生产后，当前没有批次时自动尝试启动，每批完成后逐批重新检查条件，条件恢复后自动继续。玩家关闭连续模式只阻止后续批次，当前批次继续完成。

### 状态机
每个生产槽使用以下状态：未配置、空闲、等待条件、生产中、批次暂停、产物待转移。

- 未配置：没有选择配方，不能开始生产。
- 空闲：已选配方、无活动批次、无待转移产物。
- 等待条件：连续模式已开启但无法启动新批次，记录具体原因（缺少输入/缺少工人/建筑不可运行/配方暂不可用），没有活动批次和锁定输入。
- 生产中：当前批次已创建，输入已锁定，工作量正在推进。
- 批次暂停：批次存在但暂时不能继续（工人离岗、睡眠、建筑停用等），保留配方、进度和锁定输入。
- 产物待转移：批次工作完成，输入已消耗，产物在缓冲中，未全部转入存储。

### 批次原子启动
全部条件满足时原子创建：已选有效配方→无活动批次→无待转移产物→建筑可运行→满足最低工人条件→输入充足→输入可完整锁定。任一失败则不得创建半批次。

### 输入锁定
批次开始时锁定输入（非永久消耗），锁定资源继续占用原存储容量，不得被其他系统使用。批次完成时锁定输入转为已消耗。取消批次时解除锁定，资源恢复可用。

### 工作量与效率
每个配方定义"基础所需工作量"。批次保存已完成工作量和基础所需工作量。效率变化保留已完成工作量，只影响未来推进速度，不重置进度，不改变固定批次产量。

### 工人和运行条件变化
批次开始前无工人则等待条件。批次中工人离开则进入批次暂停，保留进度和锁定。工人更换只影响更换后效率。

### 输入不足
当前批次输入已锁定不受影响。输入不足只影响下一批启动。连续模式下进入等待条件，输入恢复后自动继续。

### 批次完成
已完成工作量达标时原子执行：标记完成→锁定输入正式消耗→生成一次配方产物→写入产物缓冲→记录事务→清除进度→尝试转入存储。

### 产物缓冲
每个生产槽最多容纳一个完整批次产物。成功转入存储后允许连续模式启动下一批。空间不足时保留在缓冲，状态为产物待转移，不启动下一批。

### 连续生产下一批
每批完成后逐批重新检查条件并重新锁定输入，不得预锁多批。

### 关闭连续模式
当前批次继续完成，完成后不启动下一批，最终进入空闲。

### 取消当前批次
生产中或批次暂停时可取消。确认后：清除批次和进度→解除输入锁定→不生成产物。连续模式不自动重启。

### 配方切换
未配置/空闲/等待条件可立即切换。生产中或批次暂停时默认"完成当前批次后切换"，也可"立即取消并切换"。产物待转移时等待转移后才切换。

### 建筑移动、升级、拆除和摧毁
移动时暂停保留批次；升级时暂停保留批次和进度；拆除前必须确认取消批次；摧毁时批次终止，进度清零，锁定解除。

### 存档与读档
保存生产槽完整状态。读档按顺序恢复，不得重复扣料或产出。

### 旧存档迁移
有明确旧进度和输入状态的迁移为活动批次。无法确认的恢复为空闲。不自动生成输入或产物。

## 消耗链路
- NPC生存消耗：0.5 食物 + 0.5 水 / 游戏日 / 人
- 建筑建造消耗：木材、石头、金属
- 建筑运行消耗：详见 6.1 建筑运行消耗表
