### 四、建筑外观辨识元素（17种建筑各自特征）

#### 🏠 房屋
- **外观特征**：方正小屋+斜顶瓦片屋顶+窗户透暖光+小烟囱
- **颜色**：米白色墙壁+棕木色屋顶
- **内部3分区**：睡眠区（双层床）/起居区（方桌凳子）/储物区（储物箱）

#### 🌾 农田
- **外观特征**：翻过的褐色土地+绿色作物行+稻草人+木栅栏围边
- **颜色**：土棕色地面+绿色作物
- **内部3分区**：播种区（种子袋锄头）/生长区（3阶段作物模型）/收获区（镰刀收获篮）

#### 💧 水井
- **外观特征**：圆形石砌井口+木制绞盘+绳索+水桶
- **颜色**：灰色石砌+棕色木架
- **内部2分区**：提水区（绞盘绳索）/蓄水区（石砌蓄水池）

#### 🌲 树场
- **外观特征**：小块林地+3棵不同高度树木+树苗+木桩+斧头架
- **颜色**：深绿树冠+棕色树干
- **内部3分区**：育苗区（小树苗）/成树区（可砍大树）/伐木区（斧头架木屑堆）

#### ⛏️ 采掘场
- **外观特征**：露天矿坑+矿道洞口+铁轨矿车+矿石堆
- **颜色**：灰色岩石+棕色支撑木架+银色轨道
- **内部3分区**：矿道口（黑暗洞口）/矿车轨（铁轨矿车）/分拣区（筛网分拣台）

#### 🔥 熔炉房
- **外观特征**：石砌大房子+高烟囱冒烟+炉口火焰光+铁砧
- **颜色**：深灰色石墙+红色/橙色火焰光
- **内部3分区**：原料区（矿石燃料堆）/熔炼区（大型熔炉+风箱+铁砧+淬火池）/成品区（金属锭堆）

#### ☀️ 太阳灯
- **外观特征**：高金属灯杆+顶部3颗发光晶体+反射罩+底部燃料槽
- **颜色**：银灰色金属+暖黄色发光晶体
- **高度**：2倍标准高度（4m）
- **内部2分区**：灯体区（发光晶体）/供能区（燃料槽）

#### 🏡 农舍
- **外观特征**：玻璃棚顶温室+水泵+打包台，比普通房屋大
- **颜色**：透明玻璃顶+米白墙+绿色植物
- **内部3分区**：温室区（多层种植架）/灌溉中枢（水泵水阀）/加工区（清洗槽打包台）

#### 🗑️ 垃圾处理场
- **外观特征**：带4个不同功能区的厂房+烟囱+传送带+堆肥箱
- **颜色**：暗绿色+灰色工业风格
- **内部4分区**：销毁区（焚烧炉）/回收区（传送带机械臂）/堆肥区（堆肥箱）/提炼区（蒸馏釜）

#### 📦 仓库
- **外观特征**：大跨度库房+宽大门+屋顶天窗+地面堆放资源
- **颜色**：深棕色木墙+灰色石基
- **内部2分区**：散货区（地面堆放）/货架区（多层货架+梯子）

#### 🗼 哨塔
- **外观特征**：**细高塔形**，3倍高度，底部武器架+中层螺旋楼梯+顶层360°瞭望台+探照灯
- **颜色**：灰蓝色石墙+木制垛口+金色探照灯光束
- **尺寸**：1×1×3（特殊细高）

#### 🪤 陷阱
- **外观特征**：**扁平贴地**，金属夹齿装置+触发踏板+连接链
- **颜色**：铁灰色+红色触发标记
- **尺寸**：0.5×0.5×0.3（极扁平）

#### 🔬 研究站
- **外观特征**：略带天文穹顶的建筑+望远镜+书架+试管架
- **颜色**：银灰+淡紫+金色点缀
- **尺寸**：2×2×1.5（略扁，有穹顶弧度）

#### 🔧 管道
- **外观特征**：**垂直细管**，金属材质+法兰盘连接口+管内彩色粒子流动
- **颜色**：银灰色金属
- **尺寸**：0.5×0.5×2（细高）
- **粒子颜色**：蓝色=水、棕色=木、灰色=石、金色=金属

#### 🏥 疗愈所
- **外观特征**：带红十字标志+长凳等候区+病床+药柜
- **颜色**：白色+红色十字+淡蓝色窗
- **内部3分区**：候诊区（长凳叫号牌）/治疗区（诊察床药柜）/康复区（病床输液架）

#### 🧱 城墙
- **外观特征**：**扁长墙体**，石砖堆叠面+垛口+支撑扶壁
- **颜色**：灰色石砖+棕色木制支撑
- **尺寸**：2×0.5×1.5（扁长）
- **特殊**：有撞击痕纹理（被攻击后出现）

#### ⚒️ 工坊
- **外观特征**：木工+金工混合工坊，有锯台、铁砧、工作台
- **颜色**：木棕色+金属灰
- **内部3分区**：木工区（锯台刨子）/金工区（小型铁砧打磨机）/装配区（工作台零件盒）

### 一、地皮与网格系统

```
┌────┬────┬────┬────┬────┐
│ A1 │ A2 │ A3 │ A4 │ A5 │  ← 一行5格地皮
├────┼────┼────┼────┼────┤
│ B1 │ B2 │ B3 │ B4 │ B5 │    世界地图 5×4 = 20格
├────┼────┼────┼────┼────┤
│ C1 │ C2 │ C3 │ C4 │ C5 │    初始拥有B2/B3/C2/C3
├────┼────┼────┼────┼────┤
│ D1 │ D2 │ D3 │ D4 │ D5 │
└────┴────┴────┴────┴────┘
```

#### 放置基础规则
| 规则 | 说明 |
|------|------|
| **地皮坐标** | 固定网格系统，每格地皮对应一个2m×2m底座 |
| **建筑对齐** | 建筑必须对齐网格线 |
| **放置范围** | 只能在自己的地皮上放置（初始4块，更多需购买） |
| **未勘探地皮** | 不可建造，必须先勘探（1工人+1游戏日） |
| **每格最多建筑** | 1座（占满整格）+ 可同时放小块装饰（0.5×0.5） |
| **旋转** | 建筑可旋转（90°增量），所有方向功能相同 |
| **不可重叠** | 两座建筑不可占用同一格空间 |

### 二、堆叠放置规则

```
        第N+1层 ┌──────┐
                │ 建筑  │  ← 上层建筑
                │ 重量W │
        第N层   └──────┘
              ┌──────────┐
              │  下层建筑  │  ← 下层需承受上层重量
              │  承重值C  │     条件：C ≥ W
              └──────────┘
```

#### 垂直堆叠规则
| 规则 | 值/说明 |
|------|---------|
| **最大堆叠层数** | 64层 |
| **堆叠对齐** | 允许悬空（建筑不需要严格正上方对齐，但必须在下层建筑承重范围内） |
| **承重条件** | 下层承重值 ≥ 上层所有建筑重量之和 |
| **重量公式** | `重量 = (木材×1 + 石头×2 + 金属×3) ÷ 10`（四舍五入取整） |
| **高度暴露** | 按1-7、8-15、16-31、32-64层分区；取消通用效率惩罚，高层增加封顶的自然耐久损耗，并提供屋顶农业与哨塔视野收益（数值见1.2） |
| **中间层拆除** | 允许，但上层进入30分钟悬空状态，超时崩塌（不返还资源） |

### 三、光照与遮挡放置规则

```
   ☀️ 太阳灯 (第N层)
    ↓ 向下覆盖3层
    ↓
   🌾 农田 (第N-1层)   ← 被太阳灯覆盖可夜间产出
   🌾 农田 (第N-2层)
   🌾 农田 (第N-3层)
   ❌ 不可达 (第N-4层)
```

| 规则 | 说明 |
|------|------|
| **光照遮挡** | 正上方1格内存在任何实体建筑即完全遮挡 |
| **太阳灯覆盖** | 向下覆盖3层（含自身所在层），水平覆盖本格+相邻十字4格 |
| **多层太阳灯** | 效果叠加（2个灯可覆盖6层深度） |
| **依赖阳光的建筑** | 农田、农舍需要光照才能产出（夜晚/被遮挡时停产） |

### 四、相邻加成放置规则

```
最佳布局示例（水平同层）：
┌──────┬──────┬──────┐
│ 水井  │ 农田  │ 农田  │  ← 水井+农田相邻：各+15%
├──────┼──────┼──────┤
│ 农田  │ 采掘场 │ 熔炉房│  ← 采掘场+熔炉房相邻：各+20%
├──────┼──────┼──────┤
│ 哨塔  │ 哨塔  │ 哨塔  │  ← 哨塔围成防御圈：各+10%攻击范围
└──────┴──────┴──────┘
```

| 相邻规则 | 效果 |
|---------|------|
| **同类型相邻** | 相同建筑上下左右相邻，每座+10%效率，上限+30% |
| **采掘场+熔炉房** | 双方+20%效率 |
| **水井+农田** | 双方+15%效率 |
| **哨塔形成防御圈** | 每座+10%攻击范围 |
| **上下层相邻** | EA阶段暂不设加成（通过管道连通资源） |

### 五、同类建筑重复惩罚

```
第1座同类建筑：100%效率  ✅
第2座同类建筑：80%效率   ⚠️
第3座同类建筑：60%效率   ⚠️
第4座及以上：40%效率     ❌
```
