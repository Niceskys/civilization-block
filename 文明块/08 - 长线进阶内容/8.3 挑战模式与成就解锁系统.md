# 挑战模式与成就解锁
> 方案C：成就树 + 每完成一个成就解锁一个新的难度选项，形成"通关→解锁→再挑战"循环

---

## 一、每日/每周挑战

### 挑战刷新规则
- [x] **每日挑战**：每游戏日刷新1个（游戏中一天=20分钟现实，等于玩家每20分钟一个新挑战）
- [x] **每周挑战**：每5游戏日刷新1个
- [x] 挑战入口：HUD顶部"🏆 挑战"按钮（快捷键T，与科技树共用T键切换子页）

### 每日挑战池（示例）
| 挑战名 | 目标 | 奖励 | 难度 |
|--------|------|------|------|
| 效率至上 | 单游戏日内产出20份食物 | 30星辉 | 简单 |
| 建筑狂人 | 单游戏日内建造3座建筑 | 40星辉 | 简单 |
| 无伤之夜 | 夜晚0建筑被怪物击破 | 50星辉 | 普通 |
| 人口激增 | 单游戏日内诞生2名婴儿 | 80星辉 | 普通 |
| 全职业运转 | 同时有5种不同职业NPC在工作 | 100星辉 | 困难 |
| 零恐惧日 | 当日结束时恐惧值仍为0 | 120星辉 | 困难 |

## 二、成就树系统

### 成就树结构
```
一层成就（新手期）→ 二层成就（成长期）→ 三层成就（巅峰期）
       ↓                      ↓                    ↓
   解锁基础选项            解锁中级选项          解锁终极选项
```

### 一层成就（新手期，达成后解锁基础难度选项）
| 成就 | 解锁选项 | 效果 |
|------|---------|------|
| 🏆 建造第一座房屋 | 开局加速 | 新游戏初始木材+20 |
| 🏆 人口达到5人 | 初始人口+1 | 新游戏初始NPC +1 |
| 🏆 击败第一只怪物 | 轻松开局 | 新游戏初始星辉+50 |
| 🏆 完成第一个主线任务 | 蓝图记忆 | 新游戏初始直接解锁矿场（无需购买） |

### 二层成就（成长期，达成后解锁中级难度选项）
| 成就 | 解锁选项 | 效果 |
|------|---------|------|
| 🏆 人口达到20人 | 人口上限+25 | 人口上限从50→75 |
| 🏆 堆叠达到10层 | 堆叠狂人 | 堆叠层数上限从64→128 |
| 🏆 击败3次BOSS | BOSS倍率 | 新游戏可选"BOSS翻倍+掉落×3"模式 |
| 🏆 拥有8块地皮 | 地皮折扣 | 商店地皮价格从1000→800星辉 |
| 🏆 累计建造30座建筑 | 建筑大师 | 同种建筑效率惩罚从40%→50%（第4座起50%，非40%） |

### 三层成就（巅峰期，达成后解锁终极难度选项）
| 成就 | 解锁选项 | 效果 |
|------|---------|------|
| 🏆 人口达到50人（满） | 传奇开端 | 新游戏可选"开局1名NPC带随机传奇特质" |
| 🏆 击败全部3种BOSS | 百战英雄 | 新游戏可选"初始自带3个BOSS令牌" |
| 🏆 完全挖掘2块地皮地下10层 | 地质学家 | 新游戏可选"地下挖掘坍塌概率-50%" |
| 🏆 拥有过5名不同传奇特质NPC | 天命所归 | 新游戏可选"初始NPC传奇概率从5%→15%" |
| 🏆 游戏时间达100游戏日 | 不朽文明 | 新游戏可选"初始人口+3 + 全科技Lv1" |
| 🏆 累计染色50座建筑 | 色彩大师 | 解锁隐藏配色"彩虹渐变"（随机变色动画） |

## 三、成就解锁的难度选项面板

### 新游戏设置界面新增"解锁选项"标签
- [x] 新游戏时，在难度选择旁新增"解锁选项"面板
- [x] 已通过成就解锁的选项显示为绿色✅，可选
- [x] 未解锁的选项显示为灰色🔒，显示解锁条件
- [x] 每个选项可独立开关（玩家可选择启用/不启用）

### 选项分类
| 类别 | 选项示例 |
|------|---------|
| 🔧 资源加成 | 开局木材+20、初始星辉+50 |
| 👥 人口加成 | 初始NPC+1、人口上限+25 |
| 🏗️ 建造加成 | 矿场初始解锁、堆叠上限128层、地皮折扣 |
| ⚔️ 战斗加成 | BOSS翻倍模式、开局BOSS令牌 |
| 🎲 随机加成 | 开局传奇NPC、传奇概率提升、地下安全挖掘 |
| 🎨 外观加成 | 隐藏彩虹配色、雕像初始解锁 |
