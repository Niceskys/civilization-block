# 外部世界与贸易路线系统
> 方案C：被动消息 + 贸易路线

---

## 一、外部消息推送

### 生成规则
- [x] 每3-5游戏日自动生成1条"远方城镇的消息"
- [x] 消息显示在事件日志中（棕色📜图标标记）
- [x] 消息纯氛围，不影响游戏机制

### 消息池示例
| 消息类型 | 示例 |
|---------|------|
| 🏙️ 城市消息 | "据说东边的"铁锤城"刚刚建成了第50层高塔" |
| 🧭 旅行者见闻 | "来自南方的商人说，沙漠深处有一种会发光的矿石" |
| 📜 历史消息 | "古籍记载，这片土地上曾有古代文明留下的地下遗迹" |
| ⚠️ 警示消息 | "北边的城镇最近遭遇了可怕的暴风雨，损失惨重" |
| 💡 提示消息 | "信使带来消息：有一种新的建筑技术正在北方流行" |
| 🎭 文化消息 | "远方城镇的居民今天庆祝他们的建城纪念日" |

## 二、贸易路线系统

### 开通规则
- [x] 可通过商店购买"贸易路线开通许可"：100星辉（一次性永久）
- [x] 开通后每 5 游戏日获得一次外部特产
- [x] 特产自动送达全局仓库（带运输队动画）

### 贸易路线等级
| 等级 | 升级条件 | 每5日获得 | 额外效果 |
|------|---------|----------|---------|
| Lv1 (基础) | 开通 | 随机资源×20（食物/木材/石头之一） | - |
| Lv2 | 拥有5座不同建筑 | 随机资源×30 + 5星辉 | - |
| Lv3 | 人口≥15 | 随机资源×40 + 10星辉 | 低概率获得染料/肥料 |
| Lv4 | 拥有≥3种不同职业 | 随机资源×50 + 15星辉 | 稳定获得1染料或1肥料 |
| Lv5 (满级) | 人口≥25 + 累计贸易10次 | 随机资源×60 + 25星辉 + 1BOSS令牌碎片 | 极低概率获得随机传奇特质书 |

### 升级消耗
| 升级 | 消耗 | 说明 |
|------|------|------|
| Lv1→Lv2 | 50星辉 | 扩大商路规模 |
| Lv2→Lv3 | 100星辉+10木材 | 修建驿站 |
| Lv3→Lv4 | 200星辉+10金属 | 组建护卫队 |
| Lv4→Lv5 | 500星辉+20金属+3BOSS令牌 | 建立永久商路要塞 |

### 贸易路线视觉表现
- [x] 开通时：地图边缘出现一条发光的商路连线
- [x] 每次特产送达：一队小推车从地图边缘进入→走向仓库→资源入库
- [x] 升级时：商路连线变得更亮更宽，增加装饰性路标
- [x] 升级到Lv5时：商路终点出现一座小型要塞装饰
