# 堆叠交互规则

### 光照遮挡
- [x] 光照遮挡判定标准：正上方1格内存在任何实体建筑即完全遮挡（含非标准尺寸建筑）
- [x] 太阳灯生效是否需要消耗资源：是
  - 消耗：1 燃料/游戏日
- [x] 太阳灯最多向下覆盖层数：3 层（含自身所在层）
- [x] 太阳灯水平覆盖范围：本格 + 相邻4格（上下左右十字形）
- [x] **多层太阳灯是否效果叠加**：是（2个太阳灯可覆盖6层深度，效果累加不衰减）

### 相邻加成/惩罚
- [x] 水平相邻加成（同层）：

#### 同种建筑水平相邻加成
- 0座有效同种邻居：+0%。
- 1座有效同种邻居：总加成+10%。
- 2座有效同种邻居：总加成+25%。
- 3座及以上有效同种邻居：总加成+40%。
- 只计算同层上下左右四个正交方向。
- 对角和上下层不计算。
- 总加成上限为40%。
- 每座建筑独立计算自己的邻居。
- 移动、重新放置、完成施工、拆除或摧毁后立即刷新。
- 与同种数量倍率采用乘算，数量倍率权威规则见2.1。

- [x] 采掘场相邻熔炉房：双方+20%效率
- [x] 水井相邻农田：双方+15%效率
- [x] 哨塔相邻（连线围成防御圈）：每座+10%攻击范围
- [x] **上下层相邻加成**：暂不设上下层相邻加成（EA阶段可通过管道实现资源连通）

### 跨层资源连通
- [x] 底层建筑与上层建筑的资源是否自动连通：否
  - 默认情况下，每层建筑资源独立
  - 需建造"管道"建筑才能在上下层之间自动传输资源
- [x] 管道连接规则：
  - 一根管道连接上下相邻2层建筑（输入口在下层，输出口在上层）
  - 多层堆叠需逐层铺设管道
  - 管道不占用建筑工人槽位
  - 管道方向一旦建好不可更改（需拆除重建）

### 管道建筑
- 功能：连接同一块地皮的上下层，使资源自动流通
- 建造成本：5木材+2金属
- 建造条件：需在上下层之间有空位（管道尺寸0.5×0.5×2，不占用全格）
- 连接范围：仅连接直接相邻的上下2层
