### 快速摘要
- **建造成本**：10木材 | **建造时长**：1游戏日（20游戏分钟）
- **首次基础生产建筑加速**：0.4游戏日（8游戏分钟）；与农田、水井共享一次加速机会；如果该机会已被农田或水井使用，则使用正常施工时长
- **工人上限**：2人 | **产出**：3木材/工人/游戏日
- **运行消耗**：无
- **耐久度**：500 | **承重值**：3
- **升级**：Lv2(+5木+5石×1.25) → Lv3(+10木+10石+2金×1.50，解锁自动运转) → Lv4(+20木+20石+5金×2.00)

### 应急炭化
| 项目 | 数值 |
|------|------|
| 输入 | 2木材 |
| 输出 | 1燃料 |
| 标准生产时间 | 1游戏日 |
| 工人需求 | 1名 |
| 科技要求 | 无 |
| 燃料输入 | 无 |
| 金属输入 | 无 |

- 应急炭化是零燃料备用来源，用于垃圾处理场中断后的应急恢复。
- 垃圾处理场模式D仍是主要燃料来源。
- 必须由玩家手动选择"应急炭化"配方，树场不自动消耗木材。
- 熔炉房和太阳灯缺燃料时，不会自动调用树场应急炭化，也不能直接燃烧木材。
- 配方运行期间占用对应生产能力（树场工人）和1名工人。
- 效率影响生产速度，不改变每批固定产量（1燃料/批）。
- 连续生产边界留给C024。
- 木材必须先经过应急炭化才能成为燃料。
