# 工坊

> 📋 所有数值详见 **6.1 建筑数值总表**
> 🎨 视觉动画详见 **5.3 建筑详情UI**
> 🔧 维修机制详见 **2.1 建筑通用规则** § 耐久维护

### 快速摘要
- **建造成本**：15木材+8石头+3金属 | **建造时长**：1.5游戏日
- **工人上限**：1人 | **效果**：相邻建筑耐久损耗-50%
- **运行消耗**：无
- **耐久度**：550 | **承重值**：4
- **升级**：Lv2(+5木+5石，减耗-60%) → Lv3(+10木+10石+2金，减耗-70%) → Lv4(+20木+20石+5金，减耗-80%)

### 维护机制
- 工坊工人自动维修相邻（上下左右前后6格）受损建筑
- 维修资源从全局库存自动扣除
- 可手动拖拽工人调整优先维修目标

### 简易炭化
| 项目 | 数值 |
|------|------|
| 输入 | 2木材 |
| 输出 | 1燃料 |
| 标准生产时间 | 1游戏日 |
| 工人需求 | 1名 |
| 科技要求 | 无 |
| 工坊等级要求 | 基础等级 |
| 是否需要燃料启动 | 否 |

- 简易炭化是备用燃料生产路线，垃圾处理场仍是主要燃料来源。
- 玩家必须手动打开工坊生产界面，选择"简易炭化"配方，并确认开始生产。
- 工坊不会自动消耗仓库中的木材来生产燃料。
- 太阳灯和熔炉房缺燃料时，不会自动调用工坊的简易炭化，也不能直接燃烧木材。
- 木材必须先经过工坊简易炭化，才能成为燃料。
- 实际生产速度受现有建筑效率与工人效率规则影响。
- 完成一批固定生成1燃料，效率改变生产速度而非批次产出数量。
- 连续生产、输入扣除时点、暂停、满仓、取消、配方切换和存档边界，等待C024统一回写。
