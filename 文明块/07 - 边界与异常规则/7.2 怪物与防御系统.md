# 怪物与防御系统

## 怪物生成机制
- [x] 生成时间：夜晚时段（每个游戏日后半段10分钟）
- [x] 生成方式：随机触发 + 人口/恐惧值影响
  - 基础触发概率：30%/夜
  - 总人口≥10：概率 +10%
  - 总人口≥20：概率 +20%
  - 恐惧值≥50：概率 +20%
  - 恐惧值≥80：概率 +40%，可能出现精英怪物
  - 最大触发概率：90%

- [x] **动态难度调节**（Flow Theory机制）：
  - 如果玩家连续3夜无建筑被破坏 → 下夜怪物生成数量 +20%
  - 如果玩家连续2夜有建筑被破坏 → 下夜怪物生成数量 -20%
  - 强度调节上限：原始数量×50%（最多降低到50%或提升到150%）
  - BOSS威胁值系统不受动态调节影响（BOSS仍由威胁值独立控制）
  - 玩家可在设置中关闭"动态调节"（关闭后按固定值生成）
  - 动态调节在每一夜开始时重新计算
- [x] 怪物种类（6种）：
  - **普通怪物**：基础型，攻击建筑每秒消耗耐久10-30
  - **迅捷怪**：移动速度×2，HP为普通1/2，容易突破防线直击建筑（第5游戏日起出现）
  - **装甲怪**：HP×3，移动速度×0.5，需哨塔攻击3次/陷阱无法秒杀（第10游戏日起出现）
  - **自爆怪**：死亡时对周围3×3范围建筑造成50耐久伤害（第15游戏日起出现）
  - **飞行怪**：无视城墙和陷阱，直接飞越攻击建筑，仅哨塔守卫可攻击（第20游戏日起出现）
  - **精英怪物**：攻击力×2+范围伤害，恐惧值≥80时概率出现
- [x] **怪物掉落系统（所有怪物均掉落物品）**：
  | 怪物种类 | 击杀掉落 | 掉落概率 | 备注 |
  |---------|---------|---------|------|
  | 普通怪物 | 1星辉 | 50% | 基础掉落 |
  | 迅捷怪 | 2星辉 | 60% | 移速快难命中但收益稍高 |
  | 装甲怪 | 3星辉 + 1石头 | 80% | 皮厚但掉落稳定 |
  | 自爆怪 | 5星辉 | 30% | 需远程击杀避免爆炸，高风险高回报 |
  | 飞行怪 | 4星辉 + 随机资源 | 70% | 无视城墙的代价是掉落更丰富 |
  | 精英怪物 | 10-20星辉 + 1金属 + 概率掉落BOSS碎片 | 100% | 必定掉落，碎片集齐可合成特殊道具 |
- [x] 掉落物自动飞入全局仓库（带资源粒子飞行动画）
- [x] BOSS击杀掉落：详见下文§BOSS威胁值系统

- [x] 怪物行为：
  - 生成在地皮边缘，向中心建筑移动
  - 优先攻击最近的非防御建筑
  - 如果遇到哨塔/守卫，优先与守卫战斗
  - 飞行怪无视地面障碍直接攻击后方建筑
  - 城墙可阻挡除飞行怪外的所有怪物（需破坏城墙后才能通过）

## BOSS威胁值系统（风险递增式）
- [x] **BOSS生成规则**：不按固定时间，而是根据玩家的"威胁值"递增概率
- [x] **威胁值计算**（每游戏日开始前计算）：
  | 来源 | 每项增加威胁值 |
  |------|--------------|
  | 每座建筑（不含陷阱/管道） | +1 |
  | 恐惧值≥50 | +10 |
  | 恐惧值≥80 | +20 |
  | 每10层堆叠（跨所有地皮累计） | +5 |
  | 人口≥20 | +15 |
  | 拥有≥3个不同职业 | +5 |
- [x] **威胁值与BOSS出现**：
  - 威胁值 < 30：BOSS出现概率0%（前30日几乎无BOSS）
  - 威胁值 30-60：每夜 BOSS出现概率 5%
  - 威胁值 61-100：每夜 BOSS出现概率 15%
  - 威胁值 > 100：每夜 BOSS出现概率 30%
- [x] **BOSS类型**（随游戏日解锁）：
  | 名称 | 外观 | 出现条件 | HP | 攻击力 | 特殊能力 | 击杀掉落 |
  |------|------|---------|-----|--------|---------|---------|
  | 🐉 石甲巨兽 | 4方块体型，全身棕色石甲 | 威胁值≥30 | 普通×8 | 普通×2 | 冲撞城墙2倍伤害 | 50星辉+5金属+1BOSS令牌 |
  | 🦅 暗影蝠王 | 2方块体型，紫黑色蝙蝠翼 | 威胁值≥60+飞行怪已出现 | 普通×5 | 普通×1.5 | 飞行+召唤2只小飞行怪 | 80星辉+随机资源×10+BOSS令牌 |
  | 👹 混沌核心 | 单方块核心，周身彩色粒子漩涡 | 威胁值≥90+第40日后 | 普通×12 | 普通×3 | 每30秒释放一次冲击波(全建筑-20耐久) | 150星辉+10金属+古代数据碎片(用于科技树) |
- [x] **BOSS战规则**：
  - BOSS出现时全屏红色警告+慢镜头+BOSS名称显示
  - BOSS不会瞬移，从地皮边缘缓慢向中心推进
  - 所有哨塔自动优先攻击BOSS
  - 陷阱对BOSS无效（BOSS体型太大，无法触发）
  - 城墙对BOSS可阻挡15秒（石甲巨兽可单独再阻挡15秒）
  - BOSS战倒计时3分钟：3分钟内未击杀→BOSS自动撤退，但摧毁3座建筑
- [x] **BOSS击杀奖励**：
  - 掉落的 **BOSS令牌** 集齐3个可兑换：1块新地皮 或 解锁隐藏建筑"神像"
  - **古代数据碎片** 集齐5个可解锁科技树隐藏支线（全效率+25%）
  - 所有掉落物自动入全局仓库
- [x] **击败BOSS后威胁值重置为0**（从头累积）

## 防御建筑
### 哨塔
- 工人上限：2名守卫
- 守卫在哨塔内自动攻击范围内的怪物
- 攻击范围：哨塔所在格 + 周围2格半径
- 基础伤害：1守卫=秒杀1只普通怪物/2秒，精英怪物需4秒
- 2名守卫伤害翻倍
- **瞄准模式**：全屏切入哨塔→可手动点击怪物指定攻击部位→守卫优先攻击该部位

### 陷阱
- 一次性消耗品，放置在怪物路径上
- 怪物经过时自动触发，瞬间消灭1只怪物（包括精英）
- 触发后陷阱消失，需重新建造
- 同一格可放置多个陷阱
- 陷阱对BOSS无效（BOSS体型过大无法触发）

## NPC死亡与怪物
- 怪物击杀NPC：全体恐惧值 +10
- 守卫击杀怪物：全体恐惧值 -5（每次击杀）
- NPC被怪物杀死的尸体产生垃圾 +1

- 夜幕降临：天空从白天色调→深蓝→全黑渐变(10秒过渡)

## 防御策略建议
- 在地皮外围建造城墙形成第一道防线（飞行怪除外）
- 城墙内侧部署哨塔（2格范围覆盖城墙前沿）
- 设置2-4名夜班勇敢性格守卫负责夜间防御
- 在外围铺设陷阱（密集铺设可秒杀装甲怪以外的怪物）
- 保持恐惧值低于50可大幅降低怪物入侵概率
- 随游戏进程推进，逐步升级防御体系→城墙+哨塔围成防御圈
- **BOSS应对**：保持威胁值低于30可避免BOSS出现。如需挑战BOSS获取掉落奖励，主动积累威胁值至60+诱使BOSS出现
