# 地皮基础规则

### 地皮数量与获取
- [x] 初始玩家拥有地皮数量：4 个
- [x] 地皮获取方式与解锁条件：商店购买、达成成就、任务直接奖励
- [x] 解锁单块地皮的非货币消耗（资源/时间）：4 人开发真实时间半小时
- [x] 商店购买单块地皮的消耗成本：**1000 星辉**

### 地皮属性
- [x] 单块地皮的标准尺寸（单位：格/米）：单块地皮占用一格
- [x] 地皮坐标规则：固定网格，可以旋转和移动
- [x] 地皮之间是否有相邻加成/限制：是（地皮间有加成和限制）
- [x] 相邻加成具体规则：
  - **同类型加成**：相同建筑相邻（上下左右），每座+10%效率，上限+30%
  - **特定组合奖励**：特定配对有额外加成
    · 采掘场相邻熔炉房：双方+20%效率
    · 水井相邻农田：双方+15%效率
    · 哨塔相邻（连线围成防御圈）：每座+10%攻击范围

### 地皮操作
- [x] 地皮是否可被拆除/回收：不可拆除，不可回收
- [x] 可向上建造，可向下挖掘
- [x] 地皮获取方式：手动解锁 / 商店购买 / 达成条件自动解锁

### 地皮勘探系统
- [x] 每块新解锁/购买的地皮在首次使用前，必须经过**勘探阶段**
- [x] 勘探消耗：1名空闲工人 + 1游戏日
- [x] 勘探期间地皮上显示进度条（工人+勘探帐篷动画）
- [x] 勘探完成后随机发现以下之一：

  | 勘探结果 | 概率 | 效果 |
  |---------|------|------|
  | 🟢 无障碍物 | 60% | 可正常建造，无额外奖励 |
  | 🟡 野生资源堆 | 25% | 一次性+10木材/石头（随机） |
  | 🟠 小型遗迹 | 10% | 一次性+30星辉 |
  | 🔴 古代宝藏 | 4% | 一次性+80星辉+2金属 |
  | 💎 特殊地脉 | 1% | 该地皮获得永久特殊类型（见下文） |

- [x] 未勘探的地皮显示"未勘探"标记，不可建造，不可堆叠
- [x] 已勘探完成的地皮显示结果图标（不消失，永久标记）

### 初始地皮特殊规则
- [x] 新存档生成时，初始2×2区域的B2、B3、C2、C3四块地皮默认视为已完成基础勘探
- [x] 玩家进入新存档后可以直接在这四块地皮上放置和建造建筑，不需要分配NPC或等待勘探时间
- [x] 初始地皮不会显示"未勘探"标记，也不会重复产生勘探奖励
- [x] 该例外只适用于新存档的上述四块初始地皮。后续获得、购买或解锁的所有地皮，仍必须按照本章的正式勘探规则完成勘探，完成前不可建造或堆叠

### 特殊地皮类型（勘探1%概率发现 + 世界地图预设少量）
- [x] 世界中预设存在少量特殊地皮（数量=总地皮数×5%）
- [x] 特殊地皮在商店中购买时**灰色显示**（提示"未知类型地皮"），价格不变（1000星辉），购买后才揭晓类型
- [x] 玩家也可以花200星辉"派遣侦察队"提前揭晓该地皮的特殊类型（侦察消耗0.5游戏日，无需工人）

| 特殊类型 | 效果 | 外观标记 |
|---------|------|---------|
| 🌾 富饶地皮 | 该地皮上所有农田/农舍产出 +30% | 绿色麦穗图标 |
| ⛏️ 矿脉地皮 | 该地皮上所有采掘场/熔炉房产出 +30% | 灰色矿石图标 |
| 🌬️ 高风地皮 | 该地皮上所有哨塔攻击范围 +1格 | 蓝色旋风图标 |
| 🕊️ 祥和地皮 | 该地皮上建筑中的NPC寿命消耗 -20% | 白色鸽羽图标 |
| 💧 水源地皮 | 该地皮上所有水井产出 +30% | 蓝色水滴图标 |
| 🔥 熔岩地皮 | 该地皮上所有熔炉房产出 +50%，但建筑耐久日损耗×1.5 | 红色火焰图标 |

- [x] 特殊地皮的加成**永久有效**，与建筑升级/科技/性格加成**乘法叠加**
- [x] 每块地皮最多拥有1种特殊类型（不可叠加）
