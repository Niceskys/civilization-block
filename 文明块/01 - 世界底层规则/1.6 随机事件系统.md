# 随机事件系统
> 方案D：天气 + 周期性危机 + 随机商人/流浪NPC/地陷
> 扩展B：正向爆点（超稀有正面事件）

---

## 一、天气系统

### 天气类型与概率
| 天气 | 概率 | 持续时长 | 效果 |
|------|------|---------|------|
| ☀️ 晴天 | 55% | 3-8游戏日 | 无影响（默认状态） |
| ☁️ 阴天 | 25% | 2-5游戏日 | 农田/农舍产量 -10%，太阳能灯夜间覆盖范围-1层 |
| 🌧️ 阴雨 | 15% | 1-3游戏日 | 农田/农舍产量 -30%，建筑耐久日损耗×2，树场木材产量 +15% |
| ⛈️ 暴风雨 | 5% | 0.5-1游戏日 | 农田停产，建筑耐久日损耗×3，所有NPC工作效率 -20%，树木有概率被吹倒(随机破坏一座木建筑) |

### 天气切换规则
- [x] 每游戏日结束时判定天气是否切换（20%基础概率）
- [x] 暴风雨结束后强制转为阴天持续至少1日
- [x] 天气切换时有全屏视觉效果（阴天变暗、雨天粒子效果）
- [x] 天气图标显示在HUD左上角

### 天气与怪物
- [x] 暴风雨夜晚：怪物生成概率 +30%
- [x] 阴雨夜晚：怪物生成概率 +10%

## 二、周期性危机事件

### 危机类型
| 危机 | 触发条件 | 效果 | 持续时长 |
|------|---------|------|---------|
| 🦠 瘟疫爆发 | 每15-30游戏日随机 | 全体NPC寿命消耗×2，工作效率-40% | 3-5游戏日 |
| 🦗 虫灾 | 每10-25游戏日随机(仅影响有农田/农舍时) | 所有农田/农舍产出降至25%，树场-30% | 2-3游戏日 |
| 🔥 火灾 | 每20-40游戏日随机 | 随机点燃1座木质建筑(房屋/农田/树场/农舍)，需立即维修否则蔓延 | 0.5游戏日(蔓延前) |

### 危机应对
- [x] **瘟疫**：拥有疗愈所→持续时间减半(1.5-2.5日)，效率惩罚减半(-20%)
- [x] **虫灾**：消耗2水→手动驱虫(缩短1日)。哨塔守卫可协助驱虫
- [x] **火灾**：消耗3水灭火。工坊工人自动参与灭火。无灭火→火势蔓延至相邻建筑(每0.5日蔓延1座)
- [x] 危机期间游戏顶部显示红色危机横幅 + 倒计时

### 危机触发规则
- [x] 同一时间只触发一种危机
- [x] 危机结束后有10游戏日冷却期
- [x] 危机触发概率随游戏日递增：前30日无危机，30-60日概率减半，60日后正常

## 四、流浪NPC加入

### 触发
- [x] 每12-25游戏日随机触发
- [x] 加入条件：有空余床位 + 食物库存≥5 + 水库存≥5
- [x] 不满足条件→流浪者离开，显示提示"流浪者路过，但城镇无法接纳"

### 流浪NPC属性
- [x] 随机职业（初始四职业：农民/伐木工/挑水工/矿工），属性随机(中等偏上)
- [x] 自带1份食物+1份水
- [x] 自带随机正面特质（如"勤劳+15%效率"或"坚韧-20%恐惧敏感"）
- [x] **有2%概率流浪NPC自带传奇特质**（见3.1传奇特质系统）

## 五、地陷事件

### 触发
- [x] 每40-60游戏日随机触发（稀有事件）
- [x] 效果：随机1格地皮变为"塌陷"状态
  - 塌陷地皮上建筑直接摧毁（掉落50%建造资源）
  - 塌陷地皮不可建造，需消耗10石头+5木材→"填埋修复"恢复
  - 修复需1游戏日 + 1工人施工
- [x] 地陷不影响相邻地皮
- [x] 塌陷地皮在3D视图中显示为凹陷深色坑洞+裂缝粒子

## 六、正向爆点事件 💥（超稀有正面事件）
> 概率约2%/游戏日，独立于所有其他事件判定，可与其他事件同时触发

### 正向爆点事件列表
| 事件名称 | 触发概率 | 效果 | 视觉表现 |
|---------|---------|------|---------|
| 💎 地下矿脉发现 | 0.4%/日 | 瞬间获得 +50矿石，该地皮下方出现可开采矿脉图标 | 地面金光爆裂→矿石粒子喷泉→全局通知 |
| 🏺 古代遗物发掘 | 0.3%/日 | 瞬间获得 +1000星辉，NPC全体+10好感度 | 金色光柱从天而降→星辉粒子飞散 |
| 🧑‍🎓 天才学徒加入 | 0.3%/日 | 1名自带**传奇特质**的成年NPC加入（随机职业） | 传送门打开→NPC走出+🌟特效 |
| 🌾 极品丰收 | 0.4%/日 | 所有农田/农舍下一次产出 ×300%（仅一次） | 金色麦浪+天使光特效笼罩农田 |
| 🏭 机械觉醒 | 0.2%/日 | 所有熔炉房/采掘场下一次产出 ×200%，且不消耗燃料 | 金属光泽笼罩工业建筑 |
| ⚡ 灵感迸发 | 0.2%/日 | 研究站当前科技点产出 ×500%（一次性） | 紫色电光罩住研究站 |
| 🛡️ 精英守卫 | 0.1%/日 | 所有哨塔守卫立即获得一个临时传奇特质"庇佑"（全属性+10，持续5游戏日） | 金色光环笼罩所有哨塔 |
| 💰 天降财宝 | 0.1%/日 | 随机获得3种不同类型资源各 +50，星辉 +300 | 彩色礼花→资源图标雨落下 |

### 触发规则
- [x] 每个正向爆点事件触发时，弹出全屏金色横幅 + 音效
- [x] 事件记录自动写入事件日志
- [x] 同一游戏日最多触发1个正向爆点事件
- [x] 正向爆点事件的概率不受难度影响（所有难度一致）
- [x] **动态概率调节**（Flow Theory机制）：当玩家连续3日资源赤字（任意一种核心资源食物/水/木材为0）时，正向爆点概率从2%提升至5%/日。困境持续越久概率越高（每多1日赤字+1%，上限10%/日）。资源恢复正数后重置为2%

### 季节性限时活动
- [x] 每30游戏日为一个"季节"
- [x] 季节名称按顺序循环：春芽季→夏焰季→秋实季→冬寂季
- [x] 每个季节持续30日，结束后自动进入下一季节
- [x] 季节切换时HUD显示横幅"🌸 春芽季开始了！持续30日"

### 季节专属内容
| 季节 | 限时挑战 | 限定装饰 | 限定资源包 |
|------|---------|---------|-----------|
| 🌸 春芽季 | 30日内种植10座农田 | 樱花树装饰图纸 | 开局种子包+20食物 |
| 🔥 夏焰季 | 30日内击败1次BOSS | 火焰旗帜装饰图纸 | 高温燃料包+10燃料 |
| 🍂 秋实季 | 30日内收获总计200食物 | 金色丰收雕塑图纸 | 丰收礼包+50食物+20星辉 |
| ❄️ 冬寂季 | 30日内无NPC死亡 | 冰晶路灯装饰图纸 | 保暖物资包+20木材+10石头 |

- [x] 限时挑战在季节开始时自动接取（不计入正常任务上限）
- [x] 完成限时挑战获得：该季节的**限定装饰图纸**（永久解锁，错过等一年）
- [x] 限定资源包在季节开始时自动发放一次
- [x] 限定装饰图纸在季节结束后无法再获得（直到下一年同一季节轮回）
