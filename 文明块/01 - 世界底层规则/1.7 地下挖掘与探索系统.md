# 地下挖掘与探索系统
> 方案C：深层地下城（10层）+ 工具层级联动 + 加速坍塌难度

---

## 一、基础规则

- [x] 每块地皮可向下挖掘，深度上限：10 层
- [x] 挖掘方式：建造"挖掘入口"（消耗5木材+3铁锭，占该地皮1格）
- [x] 挖掘入口建好后，该地皮出现"向下楼梯"图标，点击可切换视角至地下层
- [x] 地上建筑与地下层互不干扰（地上承重不受地下挖掘影响）
- [x] **必须使用对应等级的工具才能开始挖掘对应层**（见下表）
- [x] 工具在挖掘过程中持续消耗耐久（每层消耗约30%耐久，请提前备好足够工具）

---

## 二、挖掘流程（增加难度版）

### 工具要求与挖掘消耗（修正版）
| 深度 | 所需工具 | 所需时间 | 工具耐久消耗 | 坍塌风险 | 加固消耗 |
|------|---------|---------|------------|---------|---------|
| 第1层 | 铁镐（或更高级） | 1游戏日 | 30% | 0% | - |
| 第2层 | 铁镐 | 1.5游戏日 | 30% | 5% | 5木材+1铁锭 |
| 第3层 | 铁镐 | 2游戏日 | 30% | 10% | 5木材+2铁锭 |
| 第4层 | **铜镐**（铁镐不可用） | 2.5游戏日 | 30% | 15% | 8木材+2铁锭 |
| 第5层 | 铜镐 | 3游戏日 | 40% | 22% | 8木材+3铁锭 |
| 第6层 | 铜镐 | 3.5游戏日 | 40% | 30% | 10木材+4铁锭 |
| 第7层 | **水晶镐**（铜镐不可用） | 4游戏日 | 40% | 38% | 10木材+5铁锭+1水晶碎片 |
| 第8层 | 水晶镐 | 4.5游戏日 | 50% | 45% | 12木材+6铁锭+2水晶碎片 |
| 第9层 | 水晶镐 | 5游戏日 | 50% | 50% | 15木材+8铁锭+2水晶碎片 |
| 第10层 | **晶能镐**（水晶镐可用但效率-50%） | 6游戏日 | 60% | 60% | 20木材+10铁锭+3水晶碎片 |

### 坍塌处理（增强版）
- [x] 每次开始挖掘时判定坍塌概率
- [x] 坍塌发生时：工人自动逃出（不受伤），该层挖掘进度**重置至0%**
- [x] 使用的工具耐久消耗**不返还**（白白消耗）
- [x] 加固支撑：消耗对应材料后→该层坍塌概率**减半**（向下取整）
- [x] 加固后视觉效果：该层墙壁出现木制+金属支撑架
- [x] 加固操作在挖掘开始前进行（挖掘开始后不可中途加固）

### 工具层级锁定
- [x] **铁镐**：可挖掘第1-3层（第4层开始不可用，强行挖掘无效，系统提示"工具不够锋利"）
- [x] **铜镐**：可挖掘第4-6层
- [x] **水晶镐**：可挖掘第7-9层
- [x] **晶能镐**：可挖掘第10层（水晶镐也能挖第10层，但效率-50%，即需要12游戏日）
- [x] 工具可在挖掘途中更换（暂停挖掘→收回旧工具→分配新工具→继续）

- [x] 不同地皮的地下独立计算（每块地皮都有10层可挖）

|------|---------|---------|-----|------|
| 第1-3层 | 10% | 洞穴爬虫 | 普通×0.5 | 星辉×0.5 |
| 第4-6层 | 30% | 深渊蠕虫（1-2只） | 普通×0.8 | 星辉×1 + 概率矿石 |

| 第10层 | 10% | **隐藏BOSS：晶脉幼体**（仅首次挖掘第10层） | 普通×5 | 必掉1晶能宝石+回声文本 |

### 战斗规则
- [x] 地下遭遇怪物时：挖掘暂停，进入小规模战斗
- [x] 战斗方式：工人自动反击（伤害=该NPC力量属性×0.5%）
- [x] 玩家可手动"紧急撤离"（工人安全返回地面，但该层进度**清空至0%**，工具耐久消耗不变）
- [x] 工人死亡风险：与地面相同，死后产生1垃圾
- [x] 哨塔不参与地下战斗

## 五、挖掘难度说明

```
D1-D3：新手区（铁镐即可，基本无风险）
   ↓
D4-D6：进阶区（需铜镐，坍塌概率开始显著，建议加固）
   ↓
D7-D9：高危区（需水晶镐，坍塌概率极高，必须加固）
   ↓
D10：终极区（建议晶能镐，否则效率减半，有隐藏BOSS可能）
```

### 设计意图
- [x] 玩家不能"随便就挖到地下10层"，必须先发展科技+工具制造
- [x] 第4层开始需要铜镐 → 必须先建矿脉挖掘点+熔炉产出铜锭→制造铜镐
- [x] 第7层需要水晶镐 → 必须先积累水晶碎片，再经过工坊制造
- [x] 第10层晶能宝石只能地下挖掘获得 → 形成闭环：挖前几层→得晶能宝石→造晶能镐→挖第10层
- [x] 第10层必出1晶能宝石，保证玩家不会永远卡死

## 六、地下视角与交互
- [x] 点击"挖掘入口" → 视角切入地下（同全屏切入建筑内部机制）
- [x] 地下视角：垂直切面视图，显示已挖掘层数+当前挖掘进度
- [x] 当前工具状态显示（工具名+剩余耐久进度条）
- [x] 加固状态显示（已加固/未加固）
- [x] 每层挖掘进度条显示在层右侧（0%~100%）
- [x] 已挖掘完成的层显示资源图标（已采空）
- [x] 当前正在挖掘的层显示工人动画（挥镐+碎石粒子）
- [x] ESC/双击地面 → 返回地面视角

### 与其他系统的联动
- [x] 第10层隐藏建筑"地下金库"：装饰性建筑，提供+1000额外全局存储
- [x] 晶能宝石与2.4工具制造系统联动
- [x] 水晶碎片与科技树系统联动
- [x] 回声文本与8.6世界观剧情联动
