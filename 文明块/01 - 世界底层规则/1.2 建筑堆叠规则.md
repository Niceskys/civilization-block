# 建筑堆叠基础规则

### 堆叠上限
- [x] 单块地皮最大堆叠层数上限：64 层

### 堆叠对齐
- [x] 堆叠对齐规则：允许悬空（建筑不需要严格正上方对齐）

### 承重系统
- [x] 建筑承重规则（下层建筑是否有承重上限）：有承重上限
  - 每层建筑的承重能力取决于建筑类型（见 6.1 建筑数值表 承重值列）
  - 上层建筑总重量不得超过下层承重上限
- [x] **建筑重量计算公式**：重量 = 建造成本 ÷ 10（四舍五入取整）
  - 木材权重1、石头权重2、金属权重3：重量 = (木材×1 + 石头×2 + 金属×3) ÷ 10
  - 例：房屋10木材 → (10×1)÷10 = 1 重量
  - 例：熔炉房15石头+5金属 → (15×2+5×3)÷10 = (30+15)÷10 = 4.5 → 5 重量
  - 例：农田5木材+1水 → (5×1)÷10 = 0.5 → 1 重量
  - 例：陷阱5木材+5金属 → (5×1+5×3)÷10 = (5+15)÷10 = 2 重量
- [x] **堆叠验证规则**：下层承重值 ≥ 上层所有建筑重量之和
  - 例：房屋(承重3)在上方建熔炉房(重量5) → 3 < 5 → **不可建造**
  - 例：城墙(承重6)在上方建熔炉房(重5)+房屋(重1) → 6 ≥ 6 → ✅ 可建造
- [x] 所有17种建筑重量已在6.1中计算可用，开发者按公式自行计算

### 拆除规则
- [x] 建筑拆除规则：拆除返还 75% 建造资源
- [x] 能否单独拆除中间层建筑：是
- [x] **拆除中间层后，上层建筑的处理逻辑**：
  - 拆除后上层建筑进入"悬空"状态
  - 玩家有30现实分钟时间在中间层放入足够承重值的建筑
  - 拆除时弹出警告横幅"⚠️ 上层建筑悬空！请在30分钟内放入承重建筑"
  - 倒计时悬浮在HUD顶部（红色计时器）
  - 超过30分钟仍未修复 → 上层建筑逐一崩塌（每次间隔5秒，从上往下崩塌）
  - 崩塌时NPC自动逃出变为空闲状态，建筑摧毁（**不返还资源**）
  - 成功放入承重建筑后 → 悬空标记解除，倒计时消除
- [x] 拆除时该建筑的工人：自动变为空闲状态，进入空闲NPC列表

### 高度影响
- [x] 堆叠高度是否带来额外影响：是
  - 从第 8 层起开始出现温度和风力影响
  - **温度影响**：每增加1层，建筑效率 -2%，适合种植类建筑
  - **风力影响**：每增加1层，建筑耐久消耗速度 +3%
  - 1-7层：无任何惩罚（安全区域）
- [x] 具体高度影响的数值（已定稿）：
  - 第8层：效率-2%，耐久消耗+3%
  - 第16层：效率-18%，耐久消耗+27%
  - 第32层：效率-50%，耐久消耗+75%
  - 第64层（上限）：效率-114%，耐久消耗+171%（无法正常运作）
