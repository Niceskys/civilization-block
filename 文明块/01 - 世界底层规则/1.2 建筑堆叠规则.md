# 建筑堆叠基础规则

### 堆叠上限
- [x] 单块地皮最大堆叠层数上限：64 层

### 堆叠对齐
- [x] 堆叠对齐规则：允许悬挑偏移（建筑不需要严格正上方对齐），但正常建造时底面至少50%的占用格必须与有效支撑顶面接触；完全无支撑不属于合法建造。

### 承重系统
- 本节是堆叠层数、支撑关系、重量计算、承重验证、悬空与崩塌的唯一规则权威源。
- 建筑承重值和建造成本数值由6.1建筑数值表维护；2.2只维护堆叠空间交互。
- [x] 建筑承重规则（下层建筑是否有承重上限）：有承重上限
  - 每层建筑的承重能力取决于建筑类型（见 6.1 建筑数值表 承重值列）
  - 上层建筑总重量不得超过下层承重上限
- [x] **建筑重量计算公式**：重量 = 建造成本 ÷ 10（四舍五入取整）
  - 木材权重1、石头权重2、金属权重3：重量 = (木材×1 + 石头×2 + 金属×3) ÷ 10
  - 铁锭归入金属类，权重3；水不计入结构重量，权重0。
  - 新增建造资源必须先在本节声明重量类别和权重；未声明时不得自行套用或忽略。
  - 例：房屋10木材 → (10×1)÷10 = 1 重量
  - 例：熔炉房15石头+5金属 → (15×2+5×3)÷10 = (30+15)÷10 = 4.5 → 5 重量
  - 例：农田5木材+1水 → (5×1)÷10 = 0.5 → 1 重量
  - 例：陷阱5木材+5金属 → (5×1+5×3)÷10 = (5+15)÷10 = 2 重量
- [x] **堆叠验证规则**：下层承重值 ≥ 上层所有建筑重量之和
  - 例：房屋(承重3)在上方建熔炉房(重量5) → 3 < 5 → **不可建造**
  - 例：城墙(承重6)在上方建熔炉房(重5)+房屋(重1) → 6 ≥ 6 → ✅ 可建造
- [x] 18种建筑的建造成本和承重值由6.1提供，重量由本节公式派生计算
- [x] 6.1中承重值为`-`表示该对象不具备承重资格，不得作为其他建筑的支撑层；它仍可按自身放置规则依附或占用空间
- [x] **三维支撑接触规则**：
  - 上层建筑底面格与下方建筑最高占用层的顶面格逐格接触；同一支持者的多个接触格合并为一条支撑关系。
  - 正常建造最低有效接触比例为50%，所需格数向上取整；例如3格底面至少2格、4格底面至少2格。
  - 多个建筑共同支撑时，上层建筑自重与其收到的全部上层载荷按各支持者的接触格数量成比例分配。
  - 载荷使用1000倍整数定点单位；比例除不尽时按余数从大到小分配，余数相同按稳定ID升序，载荷总量不得因取整增减。
  - 建筑承重值约束它收到的全部上层载荷，不包含自身重量；自身重量会继续向直接支持者传递。
  - 地面是显式基础支撑，`BaseLayer = 0`的合法实体不生成隐藏建筑或虚构承重节点。
  - 施工任务占用空间并向下传递自身重量，但完工前不具备承重资格；不得在未完工建筑上继续向上建造。
  - 拆除造成的临时无支撑只允许通过下述难度相关修复状态存在，不能作为普通建造或模组绕过最低接触比例的入口。

### 拆除规则
- [x] 正常拆除的资源返还比例以2.1建筑通用规则为唯一权威源
- [x] 能否单独拆除中间层建筑：是
- [x] **拆除中间层后，上层建筑的处理逻辑**：
  - 拆除后上层建筑进入"悬空"状态
  - 玩家拥有按6.3难度表确定的修复宽限；普通难度在1倍速下约30现实分钟，暂停停止、2倍速加快
  - 拆除时弹出警告横幅并显示权威剩余游戏Tick换算后的倒计时，不把客户端墙钟作为裁决依据
  - 倒计时悬浮在HUD顶部（红色计时器）
  - 超过当前难度宽限仍未修复 → 上层建筑按6.3规定的间隔逐一崩塌，从最高占用层向下，同层按稳定ID升序
  - 崩塌时NPC自动逃出变为空闲状态，建筑摧毁（**不返还资源**）
  - 成功放入承重建筑后 → 悬空标记解除，倒计时消除
- [x] 拆除时该建筑的工人：自动变为空闲状态，进入空闲NPC列表

### 高度影响

高度影响采用“高层暴露分区”，不再按每层无限叠加通用效率惩罚。垂直建造本身是核心发展路线，高层提供农业日照和防御视野收益，同时增加可预测且封顶的自然维护压力。

#### 分区判定

- 以建筑当前占用的最高层作为高度层；普通1层建筑使用其所在层，跨多层建筑使用顶部层。
- 建筑移动或重新放置完成后立即重算；施工预览只显示预计效果，不提前生效。
- 高度分区只取一个，不跨分区累加。

| 最高占用层 | 高度分区 | 自然耐久损耗 | 农田/农舍高层日照 | 哨塔视野 |
|-----------|---------|-------------|------------------|---------|
| 1-7层 | 遮蔽区 | +0% | +0%产出 | +0格射程 |
| 8-15层 | 迎风区 | +5% | +5%产出 | +1格射程 |
| 16-31层 | 高风区 | +10% | +10%产出 | +1格射程 |
| 32-64层 | 天际区 | +15% | +15%产出 | +2格射程 |

#### 效果边界

- 取消旧规则中的通用建筑效率负数；任何建筑都不会仅因高度而停产或出现负效率。
- 自然耐久损耗倍率为`基础自然耐久损耗 × (1 + 分区增幅)`，只影响随时间产生的自然损耗，不放大怪物攻击、灾害、拆除、崩塌或脚本伤害。
- 高层日照只适用于农田和农舍，并且仅在白天、建筑顶部无遮挡、当前满足正常光照条件时生效；太阳灯可以恢复基础生产，但不能在夜晚伪造高层日照奖励。
- 哨塔视野只增加攻击射程，不增加伤害、射速或弹药效率；射程仍受遮挡和目标合法性规则约束。
- 非农田、非农舍、非哨塔建筑不获得高度产出或射程奖励，只承担对应分区的自然维护压力。
- 分区、当前收益和维护增幅必须写入建筑效率拆解或状态详情，不能作为隐藏修正。
