# 游戏目标与胜负规则

### 核心玩法
- [x] 核心玩法模式：纯沙盒无目标
  - 本游戏无预设胜利条件
  - 玩家自由定义目标（发展最大城市、研究全部科技、生存最久等）

### 失败条件
- [x] 全体NPC死亡是否直接失败：是（按难度执行恢复或结束，见7.1）
- [x] 资源清零是否直接失败：否（资源为0时NPC仍可存活，但无法建造新建筑）
- [x] 其他失败条件：无（沙盒模式仅NPC全灭为终止条件）

### 成就与里程碑
- [x] 是否有成就/里程碑系统：是
  - 核心里程碑（见10.2任务清单）：
    1. 十人村落（人口10）
    2. 二十人小镇（人口20）
    3. 垂直之城（堆叠10层）
    4. 全职业就绪（8种职业齐全）
    5. 地皮大亨（8块地皮）
    6. 钢铁丛林（建造50座建筑）
    7. 百年文明（游戏时间50游戏日）
