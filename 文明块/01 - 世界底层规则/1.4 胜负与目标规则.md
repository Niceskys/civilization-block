# 游戏目标与胜负规则

### 核心玩法
- [x] 核心玩法模式：开放式沙盒，无强制胜利条件
  - 游戏可以提供发展篇章、城市委托、里程碑和可选叙事结局，但它们是方向、记录或奖励，不是继续游玩的许可条件。
  - 玩家可以隐藏推荐、拒绝委托、忽略里程碑或在叙事选择后继续经营原存档。
  - 建筑、基础生产、NPC管理、基础防御、科技和探索不得仅因玩家未完成任务而永久锁死；具体建筑购买和科技前置仍按各自正式规则执行。
  - 玩家可以自由定义目标，例如建设最高城市、经营特定产业、照顾居民、探索地下或长期生存。

### 三层目标结构

| 层级 | 作用 | 强制性 | 失败与过期 |
|------|------|-------|-----------|
| 发展篇章 | 教会完整生存与建造循环 | 自动记录，可隐藏，不阻断操作 | 不失败、不过期 |
| 城市委托 | 提供玩家主动选择的短中期目标 | 手动接取，可放弃 | 普通委托不过期；放弃无惩罚 |
| 里程碑 | 记录城市长期成就并提供纪念 | 自动记录，无需追踪 | 不失败、不过期 |

- 随机事件任务可以随事件结束而失效，但失败不得扣除已有资产、降低永久能力或造成永久内容绝版。
- UI不得使用“必须完成才能正常游戏”暗示可选目标，也不得在玩家关闭追踪后反复自动开启。

### 失败条件
- [x] 全体NPC死亡是否直接失败：仅极难难度直接结束本局；简单、普通、困难进入“人口崩溃”暂停恢复状态（详见7.1）
- [x] 资源清零是否直接失败：否（资源为0时NPC仍可存活，但无法建造新建筑）
- [x] 其他失败条件：无

### 人口崩溃与本局结束
- 人口崩溃不等于存档损坏，也不删除城市、资源、建筑或历史记录。
- 简单、普通、困难难度允许按7.1恢复；恢复完成前模拟保持暂停。
- 极难难度在全体NPC死亡结算完成后进入本局结束状态，玩家仍可查看总结、读取旧存档或新建存档，但不能继续推进该局模拟。

### 成就与里程碑
- [x] 是否有成就/里程碑系统：是
  - 核心里程碑（见10.2任务清单）：
    1. 十人村落（人口10）
    2. 二十人小镇（人口20）
    3. 垂直之城（堆叠10层）
    4. 全职业就绪（8种职业齐全）
    5. 地皮大亨（8块地皮）
    6. 钢铁丛林（建造50座建筑）
    7. 百年文明（游戏时间50游戏日）

里程碑是对玩家已经创造之事的确认，不构成通关清单。完成全部或完全不追踪，都不改变当前存档继续游玩的资格。
