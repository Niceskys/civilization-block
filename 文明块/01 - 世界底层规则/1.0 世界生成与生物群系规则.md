# 世界生成与生物群系规则
> 适用如《Minecraft》的随机种子+程序化地形生成

---

## 一、随机种子系统

### 种子生成规则
- [x] 玩家创建新游戏时，可输入**种子码**（字母/数字，最长32位）
- [x] 种子码为空则自动生成随机种子（基于系统时间戳SHA-256截断）
- [x] 同一种子码每次生成**完全相同**的世界布局（包括：地皮类型排列、特殊地皮位置、地下层资源分布）
- [x] 种子码不区分大小写
- [x] 种子码在加载界面显示，支持分享

### 种子对哪些内容生效
| 生成内容 | 种子影响 | 固定内容 |
|---------|---------|---------|
| 初始4块地皮的生物群系类型 | ✅ 完全随机 | - |
| 商店可购买地皮的生物群系类型排列 | ✅ 种子决定排列 | 总地皮数固定（共20块） |
| 特殊地皮（6种）的位置 | ✅ 种子决定 | 总数量固定（≈总地皮数×5%） |
| 地下10层每层的资源内容 | ✅ 种子决定 | 每层总量固定（见1.7） |
| 初始资源量 | ❌ 固定按难度 | - |
| 初始NPC属性 | ❌ 每次独立随机 | - |
| 天气/事件序列 | ❌ 运行时随机 | - |
| 怪物生成 | ❌ 运行时随机 | - |

## 二、生物群系系统（共7种）

### 生物群系生成算法（仿Minecraft）
- [x] 使用Perlin噪声算法生成温度图+湿度图（分辨率：1格地皮=1采样点）
- [x] **温度值** 0.0~1.0（0=极寒，1=极热）
- [x] **湿度值** 0.0~1.0（0=极干，1=极湿）
- [x] 温度+湿度交叉映射到7种生物群系
- [x] 初始4块地皮的生物群系保证可生存（不会全部是极端群系）

### 生物群系映射表
| 生物群系 | 温度范围 | 湿度范围 | 视觉色调 | 特殊地皮出现概率 |
|---------|---------|---------|---------|----------------|
| 🏔️ 雪山 | 0.0~0.2 | 0.3~1.0 | 白色冷色调 | 8%（祥和+高风为主） |
| 🌲 寒带森林 | 0.2~0.4 | 0.4~1.0 | 深绿+灰 | 5%（均衡分布） |
| 🏞️ 平原 | 0.4~0.6 | 0.3~0.7 | 翠绿 | 4%（富饶为主） |
| 🌊 海滨 | 0.4~0.7 | 0.7~1.0 | 蓝绿渐变 | 6%（水源+富饶为主） |
| 🌄 高原 | 0.3~0.5 | 0.1~0.4 | 土黄+棕 | 7%（矿脉+高风为主） |
| 🌴 雨林 | 0.6~1.0 | 0.7~1.0 | 深绿+彩花 | 5%（均衡分布） |
| 🏜️ 沙漠 | 0.7~1.0 | 0.0~0.3 | 黄沙色 | 3%（熔岩为主，极少富饶） |

## 三、世界地图布局

### 地图结构
- [x] 世界地图为 **5×4 网格**（20块地皮），初始玩家拥有中心4块（位置B2/B3/C2/C3）
- [x] 地图边界不可扩展（世界大小固定）
- [x] 未解锁的地皮以灰色迷雾覆盖，购买后揭晓

### 典型地图示例（种子"ABC123"）
```
  A1🏜️  A2🌴  A3🌊  A4🏔️  A5🌲
  B1🌲  B2🏞️  B3🏞️  B4🌊  B5🏜️
  C1🏞️  C2🏞️  C3🏞️  C4🌄  C5🌴
  D1🌴  D2🌄  D3🏔️  D4🏜️  D5🌊
```
- B2/B3/C2/C3 = 初始4块，出生在平原（默认基准群系）
- 玩家向不同方向扩展会进入不同生物群系
- 种子不同则排列完全不同

### 生物群系对游戏策略的影响
- [x] **沙漠开局最困难**：水-10+食物-10，但石头+15 → 适合高手挑战
- [x] **平原开局最平衡**：无加减 → 适合新手
- [x] **雨林开局资源最丰富**：木材+15食物+5，但垃圾速度+50% → 需要快速建造垃圾处理场
- [x] **高原开局防御优势**：哨塔+1格 → 防怪物入侵更容易
- [x] **随机种子保证可玩性**：初始4块地皮的生物群系组合保证至少1块是平原或雨林（可产出食物）

## 四、新游戏流程

### 新游戏创建步骤
1. 输入种子（可选，空=随机）
2. 选择难度（简单/普通/困难/极难）
3. 选择已解锁的成就选项（如有）
4. 选择传承遗产+遗物（如有）
5. **预览世界地图**（显示5×4网格，初始4块高亮，其余灰色）
   - 灰色地皮显示轮廓+编号
   - 无法提前知道未解锁地皮的群系类型
   - 可通过点击灰色地皮花费200星辉侦察（见1.1特殊地皮侦察）
6. 确认 → 游戏开始

### 初始界面加载
- 显示种子码 + 世界地图缩略图
- 显示当前生物群系名称 + 该群系效果摘要
- 如果出生在非平原群系，显示特殊提示
  - "你降落在🌴雨林中 → 木材充足但注意垃圾处理"
  - "你降落在🏜️沙漠中 → 这将是一场生存挑战！"

### 生物群系视觉表现
- 每个生物群系有不同色调的天空盒+地皮纹理+环境粒子
- 雪山：白色地面+飘雪粒子
- 雨林：深绿地面+雨滴粒子+彩色花朵
- 沙漠：黄色地面+热浪扭曲
- 海滨：蓝色水边+波浪声
- 背景音乐随群系切换不同变奏
