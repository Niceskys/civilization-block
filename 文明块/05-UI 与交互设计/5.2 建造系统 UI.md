# 建造系统UI

### 建造面板
- [x] 建造面板打开方式：快捷键B / 底部建造按钮
  - 悬浮式面板，可拖拽
- [x] 建筑分类方式：全部 / 生产类 / 居住类 / 功能类
  - 生产类：农田、水井、树场、采掘场、熔炉房、农舍
  - 居住类：房屋
  - 防御类：哨塔、陷阱、城墙
  - 科技类：研究站、工坊
  - 民生类：疗愈所
  - 功能类：太阳灯、垃圾处理场、仓库、管道

### 建造交互
- [x] 建造预览：显示3D虚影
  - 合法位置：绿色虚影
  - 非法位置：红色虚影 + 显示非法原因
- [x] 建造时显示消耗资源预览：是
  - 显示建造成本、建造时长、所需工人
- [ ] 其他交互：👉 待补充

### 拆除/移动功能
- [x] 拆除功能：单独拆除按钮，确认二次提示
  - 拆除返还75%建造资源
- [x] 移动功能：点击移动按钮→选择目标位置→工人搬运

### 升级交互
- [x] 点击已解锁升级的建筑→"升级"按钮
- [x] 显示升级消耗资源和预计产出提升
