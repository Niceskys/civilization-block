# 建造系统UI

### 建造面板
- [x] 建造面板打开方式：快捷键B / 底部建造按钮
  - 悬浮式面板，可拖拽

### 建造面板一级分类
- 建造面板一级Tab固定为5个，顺序固定，不因存档、难度、解锁顺序或最近使用动态重排，不因某分类暂时没有可建建筑而消失，不因教程高亮而改变位置，存档读档后顺序不变，不允许玩家拖拽改变一级Tab顺序。
- 未解锁分类的Tab保持可见，点击后显示该分类的已解锁内容；未解锁建筑进入折叠区域；没有已解锁建筑时显示简短解锁说明，不弹出阻断窗口。
- 五个一级Tab为：
  - **常用**：默认Tab。包含新手推荐、最近使用、玩家收藏、当前任务推荐等动态区域。没有内容的区域不显示空标题；同一建筑只显示一张卡片；建筑卡仍属于原正式分类；常用页不会改变建筑数据分类；常用页排序不得改变其他Tab。
  - **生存**：直接维持NPC生存和基础居住的建筑，如房屋、农田、水井、仓库、疗愈所等。
  - **生产**：采集、加工和制造建筑，如树场、矿场、矿脉挖掘点、熔炉房、工坊等。
  - **设施**：公共设施、环境支持和科技功能建筑，如管道、太阳灯、垃圾处理场、研究站等。装饰内容不单独占用一级Tab，使用设施Tab内的"装饰"二级筛选或通过5.5定义的独立装饰入口访问。
  - **防御**：城墙、哨塔、陷阱等正式防御建筑。
- 不得再新增"全部""最近""收藏""装饰""管道""科技""高级""未解锁""推荐""临时活动分类"等一级Tab。这些功能必须通过常用页内容分区、二级筛选、搜索、独立装饰入口、建筑卡标签、未解锁折叠区实现，不得继续扩张一级Tab数量。

### 建筑卡显示
- 每个分类中按以下层级分区或排序：已解锁且当前可建、已解锁但资源不足、已解锁但当前条件不满足、未解锁。
- 未解锁建筑默认折叠在"未解锁"区域中。玩家可以展开查看建筑名称、基础用途、解锁条件、所属科技或阶段。不得显示完整后期隐藏剧情信息。
- 资源不足不等于未解锁。位置不合法不等于未解锁。

### 搜索与二级筛选
- 搜索框独立于一级Tab。搜索时可以跨5个分类搜索，结果显示正式分类标签。清除搜索后返回之前选中的Tab。搜索不新增"搜索结果"一级Tab，不改变Tab顺序。
- 二级筛选可以包括：可建造、资源不足、未解锁、装饰、管线、任务推荐、收藏。二级筛选不得伪装成一级Tab。

### 当前任务推荐
- 主线要求建造某建筑时，常用Tab可以出现"当前任务推荐"卡片，卡片仍显示其正式分类。点击卡片可选中建筑，但不得自动确认建造。不得为每个任务临时增加一级Tab。不得自动切换玩家当前操作，除非玩家主动点击任务推荐卡。

### 免费重新放置功能

#### 入口
- 符合条件的建筑在建筑详情面板中显示独立按钮："免费重新放置"
- 不得用普通"移动"按钮替代，也不得把两个操作混为同一按钮

#### 启用条件
- 属于本存档最先完成的3座玩家建筑之一
- 免费机会未使用
- 尚未超过10游戏分钟期限
- 未受到有效战斗伤害
- 建筑已完成施工
- 没有正在执行的互斥任务（升级、维修、普通移动等）
- 当前所在地皮有效

#### 按钮显示信息
- 启用时显示："免费重新放置" + 剩余时间（游戏分钟倒计时，格式 07:35）
- 暂停时倒计时保持不变
- 成功使用后显示："免费重新放置已使用"
- 超时后显示："免费重新放置已过期"
- 受到战斗伤害后显示："受到过战斗伤害，无法免费重新放置"
- 同一地皮内可用，不消耗资源
- 非前三座建筑不需要长期显示灰色按钮

#### 操作流程
1. 选择符合条件的建筑
2. 点击"免费重新放置"
3. 原位置显示半透明原位标记
4. 建筑进入目标位置虚影预览
5. 目标合法时显示绿色虚影，非法时显示红色虚影和具体原因
6. 左键或确认键完成
7. 右键、Esc或取消按钮退出，不消耗机会
8. 成功后刷新建筑详情面板

确认时必须再次检查：剩余时间、战斗伤害状态、目标位置合法性、建筑是否仍然存在、是否出现新的互斥任务。

#### 成功反馈
- 使用非模态提示："建筑已免费重新放置"
- 不得弹出阻断式确认窗口
- 不得暂停游戏
- 不得发放奖励
- 不得创建任务
- 不得显示为拆除返还

#### 普通移动与拆除UI
- 保留"移动"入口，说明"需要工人搬运并消耗时间"
- 拆除继续显示二次确认，明确"拆除后返还75%建造资源"
- 不得显示"前5日100%返还""前三座100%返还""免费拆除"或"免费重新建造"

### 施工加速状态与互斥显示

#### 首建加速自动应用
- 符合首建加速条件时，建造确认区域显示："首次建造加速"和"已自动生效"以及"无需消耗额外资源"。
- 同时将额外资源加速入口设为禁用，并显示原因："首次建造加速生效期间不可叠加额外资源加速"。
- 不得显示为资源不足。
- 不得允许玩家勾选"放弃免费加速""改用额外资源加速"或"同时使用两种加速"。

#### 预约状态
- 施工中显示："首次建造加速：已预约"和"成功完工后消耗机会"。
- 任务暂停时显示："首次建造加速：已预约"和"施工暂停不会返还机会"。

#### 取消提示
- 任务取消前的确认说明："取消施工后，本次首次建造加速预约将被释放。"

#### 完工与恢复反馈
- 成功完工后使用非模态提示："首次建造加速机会已使用"。
- 任务失败或完工前被摧毁后使用非模态提示："施工未完成，首次建造加速机会已恢复"。
- 不得弹出阻断式奖励窗口。

#### 异常双加速修复提示
- 出现异常双加速状态时显示非模态提示："已保留首次建造加速，并返还重复投入的加速资源"。

### 建造面板分类教程节点
- 教程标识：tutorial.build.panel_tabs
- 教程名称：建造面板分类
- 前置条件：建造功能已经解锁
- 触发条件：玩家第一次打开建造面板
- 完成条件：玩家主动切换过至少一次一级Tab
- 展示目标：五个一级Tab区域
- 展示方式：非模态锚点提示
- 普通关闭后的重触发：玩家关闭并再次打开建造面板，且从未切换过一级Tab
- 允许跳过：是
- 不得：自动切换Tab、自动选择建筑、自动扣除资源、自动确认建造、因打开面板就标记完成、因教程展示而重排Tab

### 存档与异常恢复
- 每个存档或玩家UI偏好保存：上次选中的一级Tab、搜索内容是否清空、收藏建筑、最近使用建筑。
- 五个一级Tab的数量、名称和顺序不得由存档覆盖。
- 读档后Tab顺序保持固定；上次选择的Tab不存在异常值时恢复为"常用"；非法Tab值回退到"常用"，不得导致建造面板无法打开。
- 建筑没有合法一级分类时，临时显示在"设施"中的开发诊断区域，记录开发日志，不创建新的一级Tab，不向玩家显示技术字段。
