## 一、全局布局规范

### 屏幕分辨率基准
- 目标分辨率：1920×1080（16:9）
- 最小支持：1280×720（适配缩放）
- UI缩放：支持 75%/100%/125% 三档

### 色板
| 角色 | 颜色 | 色值 | 用途 |
|------|------|------|------|
| 主背景 | 深灰蓝 | #1A1A2E | 面板背景 |
| 次级背景 | 深蓝灰 | #16213E | 卡片/列表背景 |
| 强调色 | 金色 | #FFD700 | 选中/稀有/传奇 |
| 危险色 | 红色 | #E74C3C | 警告/恐惧/敌对 |
| 安全色 | 绿色 | #2ECC71 | 合法/友好/充足 |
| 信息色 | 蓝色 | #3498DB | 可交互/信息 |
| 文字主色 | 白色 | #FFFFFF | 标题/正文 |
| 文字次色 | 浅灰 | #B0B0B0 | 辅助说明 |


**交互流程**：
1. 玩家按B或点击底部B按钮 → 弹出建造面板（悬浮，可拖拽）
2. 点击分类Tabs过滤建筑类型
3. 鼠标悬停在建筑卡片上 → 高亮+显示消耗预览
4. 点击建筑卡片 → 面板底部显示详情
5. 鼠标移到3D场景 → 跟随鼠标显示虚影
6. 虚影绿色（合法）→ 点击左键 → 确认建造
7. 虚影红色（非法）→ 显示非法原因文字

## 四、关键交互流程状态图

### 4.1 建造流程
```
按B/点击按钮
    ↓
弹出建造面板 ←→ 点击分类Tabs过滤
    ↓
点击建筑卡片 → 面板底部显示详情
    ↓
鼠标移入3D场景 → 跟随显示虚影(绿色/红色+原因)
    ↓
红色虚影? → 显示具体原因(资源不足/承重超限等) → 返回选择
    ↓
绿色虚影? → 点击左键 → 确认建造弹窗 → 确认
    ↓
建筑进入施工状态(半透明+进度条) → 工人自动分配
    ↓
施工完成 → 建筑实体化 → 开始运作
```

### 4.2 BOSS战流程
```
夜晚 → 威胁值判定 → BOSS出现
    ↓
全屏红色警告+慢镜头+BOSS名称显示
    ↓
BOSS从地皮边缘向中心推进
    ↓
3分钟倒计时开始
    ↓
玩家操作：手动瞄准部位/紧急修复城墙
    ↓
↗ 倒计时内击杀 → 掉落物飞入仓库+全屏胜利特效
    ↓              ↓
  BOSS击败成就    威胁值重置为0
    ↓
↘ 超时未击杀 → BOSS摧毁3座建筑后撤退
                  ↓
              显示损失总结
```

### 4.3 传承与新游戏流程
```
主菜单 → 新游戏
    ↓
输入种子(可选) → 选择难度 → 选择成就解锁选项 → 选择传承遗产+遗物
    ↓
预览世界地图(5×4网格，初始4块高亮)
    ↓
确认 → 游戏开始
    ↓
加载界面 → 显示种子码+生物群系名称+初始提示
    ↓
进入游戏 → 初始4块地皮已解锁(未勘探)
```

## 五、响应式UI组件规范

### 面板通用规则
| 组件 | 规则 |
|------|------|
| 所有面板 | 悬浮式，可拖拽，可关闭(×) |
| 面板宽度 | 最大不超过屏幕宽的40%（480px~720px） |
| 面板高度 | 最大不超过屏幕高的80% |
| 超出滚动 | 面板内容超长时右侧出现滚动条 |
| 多面板 | 支持同时打开多个面板，层叠排列 |
| 面板固定 | 点击📌可固定面板位置（不随视角旋转移动） |

### 按钮规范
| 类型 | 尺寸 | 样式 |
|------|------|------|
| 主要按钮 | 120×36px | 金色填充#FFD700+白色文字 |
| 次要按钮 | 100×32px | 蓝色边框#3498DB+蓝色文字 |
| 危险按钮 | 100×32px | 红色填充#E74C3C+白色文字 |
| 图标按钮 | 32×32px | 透明背景+图标 |
| Tab按钮 | 自适应 | 选中时底部金色下划线3px |

### 交互反馈
| 操作 | 反馈 |
|------|------|
| 悬停 | 颜色微变+1.05倍放大 |
| 点击 | 0.95倍缩放动画(100ms) |
| 拖拽 | 半透明跟随+阴影 |
| 无效操作 | 红色闪烁+震动 |
| 成功 | 绿色闪烁+轻声音效 |
