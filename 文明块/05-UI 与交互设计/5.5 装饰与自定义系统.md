# 装饰与自定义系统
> 方案C：建筑染色 + 装饰性地标 + 成就雕像

---

## 一、建筑染色系统

### 基础规则
- [x] 玩家可消耗 **染料** 给已建成的建筑更换配色
- [x] 每个建筑有 6 种预设配色方案（含默认色）
- [x] 染色操作：选中建筑 → 详情面板 → "染色"按钮 → 选择配色 → 确认消耗
- [x] 染色不影响建筑功能、等级、耐久

### 6种预设配色方案
| 配色名 | 主色调 | 适合建筑类型 |
|--------|--------|-------------|
| 默认色 | 标准（生产棕黄/居住米白/防御灰蓝） | 全部 |
| 温暖橙 | 橙红+米黄 | 生产类 |
| 冷冽蓝 | 深蓝+银灰 | 防御类 |
| 自然绿 | 草绿+木棕 | 生产类/居住类 |
| 皇家紫 | 紫色+金色 | 科技类/功能类 |
| 暗影黑 | 黑灰+暗红 | 防御类 |

### 染料获取
| 获取方式 | 产出量 | 备注 |
|---------|--------|------|
| 垃圾处理场模式C新增产量途径 | 1染料/5垃圾 | 与肥料并行产出 |
| 商店购买 | 10星辉/份 | 限购3份/日 |
| 正向爆点事件"天降财宝" | 附带+5染料 | 随机奖励 |
| 商人来访 | 8星辉/份 | 限购2次/来访 |

### UI表现
- [x] 染色确认时：建筑短暂闪烁→渐变过渡到新配色（1秒过渡动画）
- [x] 染色后建筑保留新配色直到再次染色
- [x] NPC服装不随建筑染色改变（独立）

## 二、装饰性地标建筑

### 可建造装饰品列表
| 名称 | 建造成本 | 占格 | 描述 |
|------|---------|------|------|
| 🗿 石质雕塑 | 10石头+5木材 | 1×1 | 朴素石雕，基础装饰 |
| ⛲ 喷泉 | 20石头+10水 | 1×1 | 持续喷水粒子效果，周围地皮好感度+5温暖效果 |
| 🚩 旗帜 | 5木材+3染料 | 0.5×0.5 | 自定义配色旗帜，随风飘扬动画 |
| 🌳 庭院树 | 5木材+2水 | 1×1 | 小型景观树，四季常绿 |
| 💡 路灯 | 5金属+2燃料 | 0.5×0.5 | 夜晚自动发光，照亮周围1格（装饰性照明，无生产加成） |
| 🪦 纪念碑 | 30石头+10金属 | 1×1 | 最大型装饰，基础版免费，史诗版需消耗BOSS令牌 |

### 建造规则
- [x] 装饰品无耐久损耗（永不损坏）
- [x] 装饰品可建在地皮任意空格上（包括已有建筑的空隙）
- [x] 装饰品无承重值（不参与承重计算），也不能支撑上层建筑
- [x] 装饰品可随时拆除（返还100%建造资源，不消耗工人）
- [x] 装饰品可染色（同建筑染色系统）
- [x] 染料加入商店常驻商品：10星辉/份，日限购3份

- [x] 雕像默认存于"存储"中，点击"放置"→选择空地皮格→确认摆放
- [x] 每个雕像占地 1×1 ~ 1×2 不等
- [x] 雕像不可染色（保持原始材质色）
- [x] 雕像可随时收起（保留，不消耗资源）
- [x] 已获得的雕像永久存在，可重复摆放
