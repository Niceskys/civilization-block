# 建筑详情UI（通用规范）

### 面板打开与关闭
- [x] 面板打开方式：左键点击建筑
- [x] 关闭方式：右上角叉号 / 点击空白处 / 再次点击建筑
- [x] 是否支持同时打开多个建筑面板：是（悬浮式，可并排查看）

### 建筑内部视图系统（画中画 + 全屏切入）
- [x] **右下小窗预览**：选中建筑后，右下角自动弹出建筑内部3D预览小窗
  - 小窗尺寸：屏幕 1/4 大小，半透明边框
  - 内容：建筑内部3D切面视图（剖开外墙看到内部）
  - NPC工作动画实时播放（见下方动画表）
  - 资源流转粒子效果可见
  - 小窗可拖拽位置、可关闭
- [x] **双击全屏切入**：双击建筑或小窗 → 画面推进到建筑内部
  - 全屏显示建筑内部3D空间（替代全局视角）
  - 可在全屏模式下拖拽工人调整站位
  - 可手动操作内部设备（如切换垃圾处理场模式）
  - ESC / 再次双击 → 退回全局视角
  - 全屏切入时，游戏时间继续流转

### 建筑内部视图系统增强（全屏切入后）

- [x] **多分区内部空间**：每种建筑2-3个功能分区，NPC在不同区间移动工作
- [x] **自由旋转视角**：全屏切入后鼠标拖拽360°旋转观察，滚轮缩放
- [x] **墙壁透视模式**：点击任意墙壁→该面半透明化，可透视后方分区
- [x] **手动操作点**：关键设备可点击手动交互，获得即时效果
- [x] **效率可视化**：NPC工作效率以头顶百分比+身体光晕强度双重显示

### 17种建筑多分区 + 交互操作点

#### 1. 房屋（3分区：睡眠区/起居区/储物区）
- 睡眠区内显示床位×N
- 鼠标悬停床位：显示入住NPC姓名、性格、状态
- 点击床位：显示调整NPC入住/迁出操作
- 左键点击NPC可跳转NPC详情面板
- 状态面板显示：床位占用（已占用数量/总床位数量）

##### 床位与住户面板

- 面板标题："床位占用"
- 显示格式：已占用数量 / 总床位数量
- 每个床位槽显示：
  - 床位序号
  - 当前NPC头像（如有住户）
  - NPC姓名
  - 当前状态
  - 空床时显示"空闲"
- 自动分配产生的住户与手动分配住户使用同一显示格式

示例：
```
床位占用：2/2

床位1：NPC甲
床位2：NPC乙
```

有空床时：
```
床位占用：1/2

床位1：NPC甲
床位2：空闲
```

交互规则：
- 保留原有"点击床位调整NPC床位分配"入口
- 自动入住完成后，面板立即刷新
- 房屋升级增加床位后，面板立即增加对应槽位
- 房屋失效时，不再显示失效床位为可分配床位
- 手动分配界面的默认显示顺序使用NPC创建顺序
- 玩家可主动切换其他排序，仅影响界面查看，不影响自动入住规则
- 自动入住完成后可使用非模态提示（例如"2名NPC已自动入住"），不得弹出需要玩家确认的模态窗口

#### 2. 农田
- 3区：耕种区/休整区/储存区
- 可拖拽调整工人站位（第1/2格）
- 显示：土壤水分、生长进度UI条

#### 3. 水井（2区：采水区/储水区）
- 显示：当前水位、出水量动画
- 手动交互：打水（加快单次产出）

#### 4. 树场（2区：种植区/砍伐区）
- 显示：树苗生长进度、预计成熟时间
- 手动操作：快速砍伐（缩短当前次产出间隔）

#### 5. 熔炉房（3区：入料/冶炼/出料）
- 显示：炉温、融化进度
- 手动操作：鼓风（临时提升效率）

#### 6. 采掘场（2区：开采区/堆放区）
- 显示：矿脉剩余量、开采进度
- 手动交互：换层开采（切换不同矿种）

#### 7. 太阳灯（无分区，单设备）
- 光锥范围可视化
- 开/关状态显示；玩家手动开关入口需等待Runtime提供正式命令后再启用
- 覆盖范围内建筑高亮标记
- 以上显示和切换入口必须读取或调用后续Runtime权威字段/命令，不得由UI自行计算覆盖或改变光照效果

#### 8. 农舍
- 3区：种植区/圈养区/储存区
- 显示：动物数量、饲料存量
- 手动交互：投喂动物（加速产出）

#### 9. 哨塔
- 2区：瞭望区/弹药储备区
- 守卫在瞭望区自动巡逻
- 瞄准模式：全屏切入选定目标部位（详见7.2部位破坏系统）
- 弹药数量显示

#### 10. 陷阱
- 单格部署，无内部分区
- 状态显示：已部署/已触发/冷却中
- 点击触发可回收部分材料（如有）

#### 11. 研究站
- 2区：实验区/理论区
- 研发进度条（科技点累计）
- 手动交互：灵感注入（消耗资源加速当前研发）

#### 12. 城墙
- 无内部分区，外部显示当前耐久
- 受损时裂痕粒子特效

#### 13. 仓库
- 3区：入库区/仓储区/出库区
- 显示：当前存储量/总容量、各类资源存量列表
- 传送带/管道出入口动画

#### 14. 垃圾处理场
- 4个独立操作区：压缩/焚烧/回收/堆肥
- 手动切换工作模式
- 每种模式显示不同内部画面

#### 15. 疗愈所
- 2区：诊疗区/康复区
- 显示：当前病人数、治疗进度
- 手动交互：使用药品加速治疗

#### 16. 管道
- 内部显示资源粒子流
- 无分区，不可交互

#### 17. 工坊
- 2区：锻造区/修理区
- 手动交互：制造工具（选择工具类型）
- 相邻建筑耐久增益标记

### 通用组件
- [x] 所有建筑详情面板左上角显示：
  - 建筑名
  - 等级（Lv1~Lv4）
  - 当前状态标记（7种状态视觉反馈）
- [x] **工人列表**：显示当前分配的工人头像+姓名+性格标签
  - 点击工人 → 跳转NPC详情面板
- [x] **生产效率面板**：显示当前产出周期百分比进度条
  - 含夜间/资源不足/暂停状态特殊标记
- [x] **升级按钮**（满足条件时亮起）
  - 显示当前等级、下一级所需资源
- [x] **移动/拆除按钮**（确认操作）
- [x] **信息卡**：固定显示建造成本、工人上限、基础产量、耐久、承重

### 专业化效率拆解

对使用建筑效率的建筑，在生产效率面板中显示效率拆解：

```
同种建筑：4座
当前序位：第4座
同种数量倍率：80%
同种水平邻居：3座
专业化相邻加成：+40%
专业化组合倍率：112%
```

生产效率详情至少显示：
- 基础效率
- 建筑升级倍率
- NPC/工人效率
- 同种数量倍率
- 同种相邻加成
- 其他状态修正
- 当前最终效率或最终产出

### 高度分区反馈

- 信息卡固定显示建筑最高占用层与当前分区：遮蔽区、迎风区、高风区或天际区。
- 所有建筑显示自然耐久损耗增幅；农田和农舍额外显示当前高层日照奖励及未生效原因；哨塔显示高度提供的额外射程。
- 建造预览和重新放置预览必须在确认前显示目标位置对应的高度收益与维护代价。
- 高度规则只引用1.2建筑堆叠规则，不在UI文件中维护第二套分区数值。

#### UI规则
- 第1、2座的100%倍率可使用中性灰色显示。
- 第3座90%和第4座以后80%显示为"规模倍率"，不得使用强烈红色警报。
- 同种相邻加成使用正向颜色和加号。
- 鼠标悬停可查看计算说明。
- UI显示顺序不得改变正式序位。
- 玩家自定义列表排序不得改变序位。
- 建造完成、拆除、摧毁、移动和重新放置后立即刷新。
- UI至少显示到小数点后1位。
- 面板显示计算结果，不在UI文档维护另一套数值。

#### 无效率建筑的处理
- 不显示误导性的效率损失。
- 房屋床位、仓库容量等不得显示被数量倍率降低。
- 可只显示"同种建筑数量"，但不得把床位或容量乘以80%。

#### 数值权威来源
- 同种数量倍率数值：6.1 建筑数值表。
- 排序与统计规则：2.1 建筑通用规则。
- 相邻判定规则：2.2 堆叠交互规则。

## 生产控制与状态显示

### 配方选择
生产面板显示当前选中配方和所有可选配方列表。玩家可点击切换配方，切换规则遵循4.2通用生产规则。

### 单批与连续开关
提供"开始一批"按钮和"连续生产"开关。单批模式下"开始一批"创建一批后禁用，完成后再启用。连续模式下显示当前开关状态。

### 生产状态
显示生产槽的当前状态：未配置、空闲、等待条件、生产中、批次暂停、产物待转移。每种状态使用明确标识和颜色区分。

### 输入总量、锁定量和可用量
显示每个输入资源的：库存总量、已锁定数量、当前可用数量。不得把锁定资源显示为可用。

### 进度显示
"生产中"状态时显示：已完成工作量 ÷ 基础所需工作量、百分比进度条、预计剩余游戏时间（基于当前效率推算）。

### 等待与暂停原因
等待条件或批次暂停时显示具体原因：缺少输入（注明具体的资源类型和缺口数量）、缺少工人、建筑不可运行、配方暂不可用、工人离岗、建筑被手动停用等。

### 诊断详情抽屉

- 诊断详情抽屉由城市脉搏条或建筑详情面板中的状态标记打开，显示在当前建筑详情面板内侧，不新开全屏页面。
- 抽屉内容固定为"现象 -> 直接原因 -> 建议操作 -> 预期恢复"四段，不显示隐藏公式、内部类名、测试名或未解锁系统。
- 抽屉只能读取Runtime状态、`StateDiagnostics`诊断码、建筑实例状态、连续生产状态、太阳灯燃料覆盖状态和已有事件；不得在UI层重新计算生产、物流、遮挡、承重、燃料或昼夜。
- 定位/高亮只允许使用诊断或Runtime状态明确关联的建筑ID；没有关联建筑ID时，只显示"打开相关面板"或"查看事件日志"，不猜测供应链、路径或责任建筑。
- 修复反馈只在对应诊断消失或状态恢复时以非模态文字显示一次，不发放奖励、不创建任务、不暂停游戏。

首批诊断展示格式：

| 案例 | 现象 | 直接原因 | 建议操作 | 预期恢复 | 禁止UI自行判断 |
| --- | --- | --- | --- | --- | --- |
| 农田缺光 | 农田生产暂停 | 读取`paused_no_light`或`continuous_production.farm.no_light` | 定位农田，查看太阳灯或等待恢复光照 | 农田从后续Tick继续生产，不补算暂停时间 | 遮挡扫描、昼夜判断、太阳灯覆盖计算 |
| 太阳灯缺燃料 | 太阳灯没有可用覆盖 | 读取`sunlamp.fuel.empty`或`FuelCoverageTicks = 0` | 定位太阳灯，查看燃料库存或供能状态 | 获得燃料覆盖后可参与农田光照裁决 | 扣燃料、开关太阳灯、推导燃料运输路径 |
| 产物待转移或满仓 | 建筑产物等待转移 | 读取生产/连续生产待转移状态或容量诊断 | 打开建筑库存、仓库或物流相关面板 | 待转移清空后生产可继续推进 | 重算仓库容量、推导搬运路径 |
| 建筑结构失稳 | 建筑处于失稳或停用风险 | 读取结构状态或结构诊断码 | 定位建筑，查看支撑和维修入口 | 支撑恢复后解除失稳状态 | 重算承重、接触比例、崩塌顺序 |
| 缺少工人或输入资源 | 建筑等待条件 | 读取岗位、输入锁或生产状态 | 打开工人分配或资源详情 | 条件满足后从保留进度继续 | 自动分配NPC、锁定资源、重算生产条件 |

### 光照状态读取契约

农田缺光图标和状态文案只能读取Runtime权威结果：

- 农田持续生产状态为`paused_no_light`；
- 或诊断码为`continuous_production.farm.no_light`。

太阳灯燃料提示只能读取Runtime权威结果：

- `GameState.Sunlamps.Buildings[buildingId].FuelCoverageTicks`用于显示该太阳灯已预付覆盖剩余Tick；
- 诊断码`sunlamp.fuel.empty`用于提示可运行太阳灯当前没有预付燃料覆盖；
- 农田是否因缺燃料而停产仍通过农田自身的`paused_no_light`或`continuous_production.farm.no_light`表达，UI不得自行把缺燃料解释成另一套生产状态。

UI不得自行扫描建筑遮挡、计算太阳灯覆盖、判断昼夜、扣除燃料、创建太阳灯开关状态或关闭太阳灯效果。太阳灯覆盖高亮和夜晚提示若需要格级结果，必须等待Runtime提供权威字段、状态或诊断摘要后再读取。

该契约只定义读取边界，不新增图标资源、光照渲染规则、昼夜规则、玩家开关命令、燃料扣除规则或树场光照规则。

### 产物待转移
产物待转移状态时显示：产物名称、数量、需要多少存储空间。并提供"处理存储空间"提示或快捷入口。

### 关闭连续生产
显示当前连续生产状态（开/关）。玩家可点击切换。关闭时确认提示："关闭后当前批次继续完成，后续批次不再自动启动"。

### 取消当前批次
生产中或批次暂停时提供"取消批次"按钮。点击后显示确认对话框："取消后本批进度清零，已锁定输入资源解除锁定，不生成任何产物"。确认后执行取消。

### 配方切换确认
尝试切换到新配方时，如果当前有活动批次，提示两种选项："完成当前批次后切换"（默认）或"立即取消当前批次并切换"（需二次确认）。不允许转移进度。

### 存档恢复反馈
读档后如果有旧批次恢复，在对应建筑面板显示提示信息。如果旧批次因异常被重置为空闲或未配置，显示一条不影响游戏进行的非阻断提示。
