# 全局界面设计

## UI整体风格
- [x] 布局风格：**极简悬浮式**
  - 所有面板悬浮于3D游戏画面上方
  - 面板可自由拖拽移动、可随时关闭
  - 保持视野最大化，减少固定UI占用

## 主菜单界面
- 按钮：新游戏、继续游戏、设置、退出游戏
- 其他元素：游戏Logo、版本号

## 游戏主界面元素

### 顶部资源栏（可自定义）
- [x] 默认显示核心资源：食物、水、木材、石头、星辉
- [x] 点击展开显示全部11种资源
- [x] 玩家可自定义置顶显示哪些资源

### 快捷功能区
- 时间控制：暂停 / ▶播放 / ▶▶加速（1倍/2倍）
- 当前游戏日 + 昼夜指示器
- 功能按钮：商店、任务、NPC列表、科技、事件日志、设置、存档
  - 商店（快捷键S）
  - 任务（快捷键Q）
  - NPC列表（快捷键N）
  - 🔬科技树（快捷键T，与挑战共用T键切换子页）
  - 🏆挑战（快捷键T，与科技树共用T键切换子页）
  - 📦贸易路线（快捷键R）
  - 📋事件日志（快捷键L）
  - 设置（快捷键Esc）
  - 保存/存档

### 时间控制状态显示
- 时间控制区固定显示三个状态：暂停、1倍速、2倍速。
- 当前生效状态必须具有明确选中效果，不依赖玩家根据图标猜测当前速度。
- 新存档首次进入世界时，"暂停"状态默认选中。
- 首次合法建筑放置触发自动启动后，"1倍速"状态自动变为选中。
- 自动切换不得弹出模态窗口，不得暂停玩家操作。
- 玩家手动切换时间状态后，界面必须立即同步显示当前实际状态。
- 若玩家在首次放置建筑前已经手动启动时间，首次放置不得覆盖玩家当前选择的档位。
- 读取已有存档时，界面显示该存档保存的实际时间状态，不强制显示暂停。

### 城市脉搏条

- 城市脉搏条位于主HUD底栏上方，属于非模态提示，不遮挡3D场景，不暂停游戏，不自动打开面板。
- 同一时刻只显示1条最高优先级诊断，不堆叠多条警告，不使用红点、闪烁、倒计时或音效制造压力。
- 信息来源只能是Runtime状态、`StateDiagnostics`诊断结果、建筑状态和已有事件；UI不得自行扫描遮挡、计算太阳灯覆盖、判断昼夜、扣除燃料、推导物流路径或创建第二套诊断优先级算法。
- 脉搏条文案表达当前可观察事实，例如"农田缺光，生产暂停""太阳灯没有预付燃料覆盖""产物等待转移""建筑结构失稳""建筑缺少工人"；不得暴露内部类名、状态码或隐藏公式。
- 点击脉搏条只允许打开诊断详情抽屉或定位到诊断关联建筑；若诊断没有关联建筑ID，只显示"打开相关面板"入口，不猜测供应建筑、路径或责任建筑。
- 若`DiagnosticIssue.TargetIds`存在且不为空，城市脉搏条可使用其中的稳定ID执行定位或高亮；若为空，不得解析`Message`文本中的建筑ID。
- 若`DiagnosticIssue.Priority`存在，城市脉搏条按Runtime提供的优先级选择展示项；若不存在，仍按收到的诊断顺序显示第一条，不在UI层维护第二套排序。
- 若Runtime提供`StateDiagnostics.GetHighestPriorityIssue`，城市脉搏条应直接读取该结果；UI不得复制或改写诊断优先级、严重度或目标ID排序规则。
- 问题消失时可显示一次非模态修复反馈，例如"农田恢复生产""太阳灯获得燃料覆盖""待转移产物已清空"；不发放奖励、不创建任务、不弹出确认窗口。
- 若多条诊断同时存在，v1优先级由Runtime或诊断聚合层提供；在该字段不存在前，UI只能按收到的诊断顺序显示第一条，不在界面层维护另一套排序规则。

首批展示案例：

| 案例 | 显示位置 | 玩家可执行动作 | UI禁止行为 |
| --- | --- | --- | --- |
| 农田缺光 | 城市脉搏条 + 农田详情抽屉 | 定位农田、打开农田详情、查看`paused_no_light`原因 | 自行扫描遮挡、判断昼夜或计算太阳灯覆盖 |
| 太阳灯缺燃料 | 城市脉搏条 + 太阳灯详情抽屉 | 定位太阳灯、打开太阳灯详情、查看燃料覆盖剩余Tick | 自行扣燃料、关闭太阳灯或推断供能路径 |
| 产物待转移或满仓 | 城市脉搏条 + 生产建筑详情抽屉 | 定位建筑、打开库存或仓储相关面板 | 自行推导物流路径或重算仓库容量 |
| 建筑结构失稳 | 城市脉搏条 + 建筑详情抽屉 | 定位失稳建筑、打开结构状态详情 | 自行计算承重、接触比例或崩塌顺序 |
| 缺少工人或输入资源 | 城市脉搏条 + 建筑详情抽屉 | 打开工人分配或资源详情 | 自行分配NPC、锁定资源或重算生产条件 |

城市脉搏条首批文案：

| 案例 | 脉搏条短文案 | 点击后主动作 | 修复反馈 |
| --- | --- | --- | --- |
| 农田缺光 | 农田缺光，生产暂停 | 定位农田 | 农田恢复生产 |
| 太阳灯缺燃料 | 太阳灯没有燃料覆盖 | 定位太阳灯 | 太阳灯获得燃料覆盖 |
| 产物待转移或满仓 | 产物等待转移 | 打开库存 | 待转移产物已清空 |
| 建筑结构失稳 | 建筑支撑异常 | 查看结构 | 建筑支撑恢复 |
| 缺少工人或输入资源 | 建筑等待条件 | 查看需求 | 建筑恢复运转 |

## 设置界面
- 选项：音量、分辨率、窗口/全屏、语言
- 游戏设置：时间流速默认值、自动存档间隔
- 自定义：资源栏显示项配置

## 数据仪表盘（快捷键D）
- [x] 独立的全局统计悬浮面板
- 内容：
  - 总人口统计（成人/婴儿 男女比例）
  - 全局生产效率总览（当前总产出速率 vs 总消耗速率）
  - 资源增长趋势折线图（最近10游戏日）
  - 建筑总览（各类型建筑数量/等级分布）
  - NPC职业分布饼图

## 事件日志
- 需要记录的事件：NPC出生、NPC死亡、生产完成、异常警告
- 建筑损坏、灾害事件、任务完成、恐惧值变化
