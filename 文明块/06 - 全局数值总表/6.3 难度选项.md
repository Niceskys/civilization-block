# 难度选项

- [x] 是否做多难度：是
- [x] 难度档位：简单 / 普通 / 困难 / 极难 + 自定义难度
  - 自定义难度：玩家可自由调整各参数倍率

### 不同难度调整的数值项
| 参数 | 简单 | 普通 | 困难 | 极难 |
|------|------|------|------|------|
| 资源产出倍率 | 150% | 100% | 75% | 50% |
| NPC消耗倍率 | 75% | 100% | 125% | 150% |
| 恐惧值增幅 | 50% | 100% | 150% | 200% |
| 恐惧值衰减速度 | 150% | 100% | 75% | 50% |
| 建造时长倍率 | 75% | 100% | 125% | 150% |
| NPC寿命倍率 | 125% | 100% | 75% | 50% |
| 怪物生成概率倍率 | 75% | 100% | 125% | 150% |

### 结构事故宽限（按难度）

| 参数 | 简单 | 普通 | 困难 | 极难 |
|------|------|------|------|------|
| 1倍速悬空宽限 | 60现实分钟 | 30现实分钟 | 15现实分钟 | 5现实分钟 |
| 2倍速实际等待 | 30现实分钟 | 15现实分钟 | 7.5现实分钟 | 2.5现实分钟 |
| 连续崩塌间隔（1倍速） | 10现实秒 | 5现实秒 | 3现实秒 | 1现实秒 |
| 超时处理 | 自动崩塌 | 自动崩塌 | 自动崩塌 | 自动崩塌 |

- 上表“现实分钟/秒”只表示1倍速下的预计现实等待；权威状态保存为游戏Tick，暂停不推进，2倍速按两倍速度消耗。
- 难度配置属于世界存档，联机只使用服务器配置；客户端不得覆盖宽限或崩塌间隔。
- 自定义难度允许设置0至120个1倍速现实分钟的宽限、1至10个1倍速现实秒的崩塌间隔。
- 自定义难度可将超时处理改为“停用但不自动摧毁”；此模式下悬空建筑停止功能且不提供承重，必须修复或手动拆除，不得借此维持生产、库存访问或上层支撑。

### 自适应支援设置
| 项目 | 规则 |
|------|------|
| 默认值 | 开启 |
| 作用对象 | 普通敌袭最终怪物数量 |
| 默认倍率 | 100% |
| 最低倍率 | 50% |
| 最高倍率 | 100% |
| 是否修改触发概率 | 否 |
| 是否修改怪物属性 | 否 |
| 是否作用于BOSS | 否 |
| 是否作用于大型危机 | 否 |
| 是否可关闭 | 是 |
| 关闭后的倍率 | 100% |
| 重新开启 | 从100%和零计数开始 |
| 保存范围 | 世界存档 |

固定难度决定基础游戏规则；自适应支援只在基础规则之下提供向下托底；不得突破固定难度向上增加强度。
具体规则详见7.2 § 自适应支援。

### NPC全灭恢复规则（按难度）
| 难度 | 恢复方式 |
|------|----------|
| 简单 | 免费救援2名基础NPC + 商店可购买更多 |
| 普通 | 免费救援2名基础NPC（无商店购买） |
| 困难 | 无免费救援；支付400星辉立即招募1名基础成年NPC |
| 极难 | 本局结束；可查看总结、读取旧档或新开档 |

- 本表只维护各难度的恢复方式与价格；触发顺序、暂停范围、事务和存档边界以7.1为准。
- 简单与普通的免费救援只把人口恢复至2名，不作为常规扩人口手段。
