# 建筑数值总表

> 基准锚点：1标准NPC（无职业）在农田工作 = **4份食物/游戏日**
> 消耗基准：每人每游戏日消耗 0.5食物 + 0.5水

## 本表权威范围

### 本表负责的字段

本表是以下建筑基础字段的权威源：
- 正式建筑名称；
- 建造成本（材料类型与数量）；
- 正常施工时长；
- 首次加速施工时长；
- 工人上限；
- 基础产出；
- 耐久度与承重值；
- 备注列中的额外说明。

本表以当前表格列为准。

### 本表不负责的字段

本表不作为以下字段的唯一权威源：
- **星辉解锁价格与商店前置**：以9.2商品清单为准；
- **配方输入、输出、批次流程**：以4.2生产链路总览为准；
- **存储容量、输入锁定、产物缓冲**：以4.3存储规则为准；
- **怪物、敌袭和防御战斗规则**：以7.2怪物与防御系统为准；
- **建筑通用行为、升级通则、自动运转**：以2.1建筑通用规则为准；
- **堆叠交互**：以2.2堆叠交互规则为准；
- **建筑独有特殊机制、独有升级能力和例外规则**：以对应建筑详情为准；
- **模拟或派生计算结果**：6.4数值平衡模拟表仅为派生结果。

### 与建筑详情的关系

建筑详情可以保留本表基础字段的受控摘要（根据00.1 §6.3受控摘要规则），但摘要不构成第二权威源。冲突时以本表对应基础字段优先。建筑详情中的独有机制、特殊模式、专属升级能力和例外规则仍以建筑详情为权威源。

### 名称映射说明

- **采掘场**（本表正式名称），旧名或商店显示名：**矿场**，对应详情文件：矿场.md。

### 首次建造加速说明

- "正常施工时长"是建筑永久基础数值，不因本规则被替换。
- "首次加速施工时长"只在对应存档加速机会尚未使用时生效。
- 房屋使用独立的首次房屋加速机会。
- 农田、水井、树场共享首次基础生产建筑加速机会。
- 首次加速只改变施工时长，不改变建造成本、工人需求、耐久、产量、容量或升级数据。

### 施工加速优先级与互斥矩阵

| 首建加速资格 | 额外资源加速 | 最终处理 |
| ------- | ------ | --------------- |
| 有，机会未使用 | 未选择 | 自动使用首建加速 |
| 有，机会未使用 | 玩家尝试选择 | 禁止选择，不扣资源 |
| 已预约 | 任意 | 保持首建加速，额外资源入口禁用 |
| 已消耗或无资格 | 未选择 | 使用正常施工时间 |
| 已消耗或无资格 | 合法选择 | 按现行额外资源加速规则 |
| 异常同时存在 | 已投入 | 保留首建加速并返还额外资源 |

- 首建机会在施工任务创建时预约，在成功完工时消耗。
- 完工前永久终止时恢复。
- 暂停和临时阻塞不恢复。
- 首建加速与额外资源施工加速禁止叠加。
- 免费首建加速优先于额外资源施工加速。
- 状态定义详见 2.1 § 首建加速机会状态。

### 同种建筑专业化组合

#### 组合公式
```
专业化组合倍率 = 同种数量倍率 ×（1 + 同种水平相邻总加成）
```

#### 同种水平相邻总加成
| 有效同种水平邻居 | 相邻总加成 |
| -------- | ----: |
| 0座       |   +0% |
| 1座       |  +10% |
| 2座       |  +25% |
| 3座及以上    |  +40% |

- 空间相邻判定规则详见 2.2 堆叠交互规则 § 同种建筑水平相邻加成。

#### 计算示例
```
第3座建筑、2个同种邻居：
90% × 125% = 112.5%

第4座建筑、3个同种邻居：
80% × 140% = 112%
```

### 消耗链路参考
- NPC生存消耗：**0.5 食物 + 0.5 水 / 游戏日 / 人**
- 1 农田工人（无职业）可维持 8 人食物（4 ÷ 0.5）
- 1 水井工人（无职业）可维持 8 人供水（4 ÷ 0.5）
- 1 房屋可住 2 人
