# NPC社交事件系统
> 方案C：文字小剧场 + 功能事件链

---

## 一、每日社交消息

- [x] 每游戏日自动生成1-2条NPC社交短消息
- [x] 消息显示在事件日志中（白色/粉色文字标记）
- [x] 消息基于NPC实时状态生成（好感度、职业、性格、所在建筑类型）

### 消息池示例
| 类型 | 消息示例 | 触发条件 |
|------|---------|---------|
| 🗣️ 闲聊 | "🧑‍🌾张三和👩‍🌾李四在农田边聊了一下午收成" | 同建筑工作好感≥40 |
| 💕 好感 | "👷王五今天给💧赵六带了一壶水" | 好感度单方≥60另一方向≥40 |
| 🎉 庆祝 | "👩‍🔧所有人围着喷泉庆祝福利的生产线！" | 当日某项产出创纪录 |
| 😤 摩擦 | "🧑‍🌾张三抱怨👩‍🌾李四干活太慢" | 同建筑工作效率差距>50% |
| 🎵 独处 | "🛡️守卫钱七在哨塔上哼了一整夜的歌" | 守卫夜班+孤独性格 |
| ☕ 休息 | "🧑‍🔬研究者孙八端着咖啡在屋顶看夕阳" | 研究站休息阶段+乐观性格 |

### 生成规则
- [x] 优先选择好感度最高/最低的NPC对生成消息
- [x] 消息自动写入事件日志（不弹窗，不打断游戏）
- [x] 玩家可在事件日志面板筛选"社交"标签

## 二、功能事件链

### 事件类型与效果

| 事件名 | 触发条件 | 概率 | 效果 | 持续时间 |
|--------|---------|------|------|---------|
| 💑 恋爱 | 一对异性NPC好感度均≥80 + 同建筑或同房屋≥5日 | 每对每10日判定一次，概率15% | → 双方同居（合并床位，1床位住2人）→ 生育冷却-1日 | 永久(直到一方死亡) |
| 💔 分手 | 恋爱NPC好感度降至<50 | 降至时立即 | → 恢复独立床位，双方心情-10(恐惧值+5临时) | 即时 |
| 😤 吵架 | 同建筑NPC好感度一方<20另一方<30 | 每5日判定一次，概率20% | → 其中一方辞职（自动变为空闲，该建筑出现空位）→ 双方好感-10 | 即时 |
| 🍽️ 聚餐 | 同房屋NPC≥3人且食物库存≥10 | 每3日判定一次，概率30% | → 全体该房屋NPC好感+5，食物消耗+2 | 即时 |
| 🎂 生日 | NPC出生日（每年一次，每30日） | 固定触发 | → 该NPC↓：全员好感+3，该NPC当日效率+50% | 1游戏日 |
| 🏅 表彰 | 某NPC连续工作≥10日无休息 | 第10日固定触发 | → 全体好感+2，该NPC当日效率+20% | 1游戏日 |
| 🚶 散步 | 空闲NPC好感≥60的另一NPC | 每日判定5% | → 双方好感+3 | 即时 |

### 恋爱/同居规则
- [x] 恋爱NPC入住同一房屋后：1个床位可住2人（合并居住）
- [x] 恋爱双方工作效率+10%（同居加成）
- [x] 恋爱状态在NPC详情面板显示：💕 恋人（对方名字）
- [x] 一方死亡时：另一方悲伤（效率-20%，持续3日）

### 生日规则
- [x] 每个NPC出生时记录其出生日（游戏日序号）
- [x] 每30游戏日触发一次生日（30天=1年）
- [x] 生日自动弹出小通知横幅"🎂 XXX 今天生日！"
- [x] 生日当天该NPC工作效率+50%，全员好感+3
- [x] NPC死亡后不再庆祝生日

### 事件通知
- [x] 功能事件触发时：弹出对应横幅（粉色💕/红色😤/绿色🍽️/金色🎂）
- [x] 纯社交消息写入事件日志不弹窗
- [x] 事件日志添加"社交"筛选标签
