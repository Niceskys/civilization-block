# 职业系统通用规则

> 权威边界：本文件维护职业学习、职业数量和转职规则；职业与多职业效率倍率以6.2为唯一数值源，岗位分配与重新分配流程以3.3为准。

### 核心效率规则（基础设定）
- [x] 无职业、职业匹配和职业不匹配使用不同效率倍率，具体数值以6.2为准
- [x] 职业加成与属性加成的计算方式：**相乘**
  - 岗位基础效率 = 职业匹配倍率 × 属性适配倍率
  - 性格情境效果由6.2单独定义，不属于职业倍率
  - 公式数值见6.2；分配应用流程见3.3

### 职业获取规则
- [x] 职业确定时机：玩家手动分配
- [x] 初始可选择职业数量：8种（全部可用，部分需建筑解锁后触发）
- [x] 高级职业解锁条件：科技解锁 / 商店购买建筑后可用
- [x] 每名NPC最多拥有 3 个职业
  - 多职业规则：
    - 第1、第2、第3职业使用不同匹配倍率，具体数值以6.2为准
    - 同一时间只能从事 1 个职业的工作
- [x] 学习新职业的消耗/条件：消耗50星辉 + 学习时间1游戏日（期间不能工作）

### 转职规则
- [x] NPC是否可以更换职业：是（玩家手动操作）
- [x] 转职是否有冷却时间：是，冷却时长：2 游戏日
- [x] 转职是否消耗资源/货币：是，消耗：50 星辉/次
- [x] **转职后原职业是否保留：是**（已学职业永久保留，NPC最多积累3个职业）

### 职业与属性关联
- [x] 属性是否影响职业适配度：是（每职业有3偏好属性，按权重计算适配倍率）
- [x] 高属性NPC是否可解锁特殊职业：否（所有职业均可通过学习获得）
- [x] 职业是否会随工作经验升级：是 → 科技树社会Lv5解锁"大学"后，NPC可自学提升职业等级
