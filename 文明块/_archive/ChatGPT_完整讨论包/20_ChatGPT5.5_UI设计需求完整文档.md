## ☝ 第一部分：UI设计需求总览

---

### 一、整体UI风格定位

| 项目 | 要求 |
|------|------|
| **风格** | **极简悬浮式** — 所有面板悬浮于3D游戏画面上方 |
| **面板背景** | 高斯模糊效果（15px模糊强度），不是纯色背景，让玩家始终看到背后的城市 |
| **交互方式** | 面板可自由拖拽、可关闭、可固定、可最小化 |
| **分辨率基准** | 1920×1080 (16:9)，支持1280×720最小适配 |
| **UI缩放** | 支持 75%/100%/125% 三档 |
| **设计原则** | 每一个操作≤2次点击，每一个信息≤1.5秒找到，所有按钮≥44×44px（Fitts's Law） |
| **面板上限** | 同时打开≤5个面板，超过时最早打开的面板自动最小化 |
| **打开动画** | 所有面板打开动画≤100ms（Doherty Threshold） |

### 二、核心UI色板

| 角色 | 颜色 | 色值 | 用途 |
|------|------|------|------|
| 主背景 | 深灰蓝 | #1A1A2E | 面板背景 |
| 次级背景 | 深蓝灰 | #16213E | 卡片/列表背景 |
| 强调色 | 金色 | #FFD700 | 选中/稀有/传奇 |
| 危险色 | 红色 | #E74C3C | 警告/恐惧/敌对 |
| 安全色 | 绿色 | #2ECC71 | 合法/友好/充足 |
| 信息色 | 蓝色 | #3498DB | 可交互/信息 |
| 文字主色 | 白色 | #FFFFFF | 标题/正文 |
| 文字次色 | 浅灰 | #B0B0B0 | 辅助说明 |

### 三、字体与图标规范

- **字体**：像素风格字体（如Press Start 2P, 04b03），但最小不小于16px（1080p基准）
- **图标**：所有资源/建筑使用方块风格的像素图标，与3D体素风格一致
- **分隔线**：使用发光晶脉线条（淡蓝色细线），呼应世界观
- **最小文字大小**：16px（低于16px阅读疲劳）
- **最小按钮大小**：44×44px
- **颜色对比度**：至少4.5:1（WCAG AA标准）
- **面板圆角**：8px
- **面板阴影**：柔和阴影（层级感）

```
│ 底部"📐规划"TAB：显示城市天际线奖励完成度          │
└─────────────────────────────────────────────┘
```

**交互流程**：
1. 按B → 弹出悬浮建造面板
2. 点击分类Tabs过滤建筑类型
3. 鼠标悬停卡片 → 高亮+显示消耗预览
4. 点击卡片 → 面板底部显示详情
5. 鼠标移到3D场景 → 跟随显示虚影
6. 绿色虚影 → 左键确认建造；红色虚影 → 显示非法原因
7. 施工状态：半透明建筑+进度条环绕

**建造交互反馈**：
- 合法位置：绿色虚影
- 非法位置：红色虚影 + 显示具体原因（资源不足/承重超限/地皮未解锁等）
- 建造时显示建筑成本、建造时长、所需工人

#### 5.2 建筑详情面板（3级深度 - Progressive Disclosure）

| 级别 | 触发 | 内容 |
|------|------|------|
| **Lv1-悬浮** | 鼠标悬停建筑1秒 | 建筑名+等级+当前状态图标+进度条 |
| **Lv2-点击** | 左键单击建筑 | 简洁面板：名称/耐久条/产出/工人/3个操作按钮(≥44px) |
| **Lv3-详情** | 点击Lv2的"▶详情" | 全部信息：属性/历史/社交/升级 |

**Lv3 详细模式新增**：
- 产量历史折线图（最近5游戏日）
- 工人列表：头像、姓名、属性、职业、效率（可拖拽排序调整优先级）
- 资源存储状态（本地缓存/全局仓库）
- 升级所需资源清单
- 异常状态视觉提示：红色文字警告+黄色感叹号图标
- 异常类型：缺少工人、资源不足、光照遮挡、承重超限、耐久不足

#### 5.3 建筑内部视图系统（画中画 + 全屏切入）— 这是本游戏最大视觉特色！

**右下小窗预览**（默认）：
- 尺寸：屏幕1/4大小，半透明边框
- 内容：建筑内部3D切面视图（剖开外墙看到内部）
- NPC工作动画实时播放
- 资源流转粒子效果可见
- 可拖拽位置、可关闭

**双击全屏切入**：
- 画面推进到建筑内部3D空间（替代全局视角）
- 可在全屏模式下拖拽工人调整站位
- 可手动操作内部设备
- ESC/再次双击 → 退回全局视角
- 全屏切入时游戏时间继续流转

**每个建筑内部有2-4个功能分区**（共17种建筑各有不同）：
- 例：农田 → 播种区/生长区/收获区
- 例：熔炉房 → 原料区/熔炼区/成品区
- 例：研究站 → 书桌区/实验区/观测区
- 例：哨塔 → 底层(武器架)/中层(楼梯)/顶层(瞭望台)

**全屏切入后功能**：
- 自由旋转视角：鼠标拖拽360°旋转观察，滚轮缩放
- 墙壁透视模式：点击任意墙壁 → 该面半透明化
- 手动操作点：关键设备可点击手动交互，获得即时效果
  - 水井绞盘：点击手动旋转3圈，本次产出瞬间完成
  - 熔炉风箱：点击手动拉3次，加速熔炼10秒
  - 研究站望远镜：点击手动观测，科技点产出+0.2
- 效率可视化：NPC工作效率以头顶百分比+身体光晕强度双重显示

## ✌ 第二部分：游戏核心机制（供设计UI时理解上下文）

---

### 一、游戏世界观与背景

在一个由"文明块"构成的体素世界中，玩家作为文明引领者，从零开始建立一座垂直堆叠的城市。世界由随机种子生成，包含7种生物群系。玩家需要管理NPC（非玩家角色）的生存、工作、社交与繁衍，建造并升级建筑，抵御夜间怪物的入侵，并向下挖掘探索地下10层的秘密资源。游戏无预设胜利目标，纯沙盒模式，玩家自由定义目标。

---

### 二、核心循环（Game Loop）

```
白天(10分钟) → NPC工作产出资源 → 玩家建造/升级/管理
    ↓
黄昏 → 怪物入侵判定（概率30%-90%）
    ↓
夜晚(10分钟) → 怪物攻击建筑 → 哨塔守卫防御 → 玩家紧急修复
    ↓
次日清晨 → 每日结算简报 → 新一天开始
```

### 三、建筑系统（共17种建筑）

#### 分类
| 分类 | 建筑 |
|------|------|
| 🏠 居住类 | 房屋 |
| 🌾 生产类 | 农田、水井、树场、采掘场、熔炉房、农舍 |
| 🔬 科技类 | 研究站、工坊 |
| 🛡️ 防御类 | 哨塔、陷阱、城墙 |
| 💡 功能类 | 太阳灯、垃圾处理场、仓库、管道、疗愈所 |

#### 堆叠规则
- 单块地皮最大堆叠64层
- **承重系统**：下层建筑有承重上限，上层总重量不得超过
- 重量公式：`(木材×1 + 石头×2 + 金属×3) ÷ 10`
- 第8层起出现温度/风力惩罚（每层效率-2%，耐久消耗+3%）
- 拆除中间层后上层悬空30分钟，超时崩塌

#### 建筑状态视觉反馈
- 正常工作：建筑常亮，窗口暖光
- 停产/条件不足：建筑变暗+顶部红色感叹号
- 建造中：半透明虚影(50%)+底部进度环
- 拆除中：红色脉冲闪烁+碎片粒子
- 耐久不足(<30%)：墙体裂纹纹理+黄色警告三角
- 升级中：金色脚手架框架动画
- 选中状态：金色描边(3px)+底部光圈

#### 相邻加成
- 同类型相邻：每座+10%效率，上限+30%
- 特定组合：采掘场+熔炉房=各+20%，水井+农田=各+15%
