# 商店通用规则

### 基础规则
- [x] 商店解锁条件：初始直接解锁（游戏开始时商店即可用）
- [x] 商品刷新规则：混合制
  - 基础商品（地皮、基础建筑解锁等）：固定永久在售
  - 稀有商品（高级建筑、增益道具等）：每游戏日刷新一次
  - 周常商品：每 5 游戏日刷新一次
- [x] 通用货币名称：**星辉**
- [x] 是否支持资源出售换货币：是
  - 基础资源（食物/水）：出售折价 25%（买入5星辉→卖出1星辉）
  - 木材/石头：出售折价 25%（买入8-10星辉→卖出2星辉）
  - 矿石：出售折价 50%（买入15星辉→卖出7星辉）
  - 金属：出售折价 75%（买入30星辉→卖出22星辉）

### 货币获取途径
- [x] 出售多余资源获得星辉
- [x] 达成里程碑/成就奖励星辉
- [x] 完成任务奖励星辉（见 10.2 任务清单）
- [x] 随机商人交易（见 1.6 随机事件系统）
- [x] 初始开局星辉（按难度：简单200/普通100/困难50/极难0）

### 购买限制
- [x] 常驻商品是否有购买数量上限：是
  - 资源类每日限购（食物10份/日、水10份/日、木材10份/日等）
  - 建筑解锁类一次性购买永久生效
  - 增益类商品无日限，按游戏日刷新
- [x] 每日购买额度刷新时间：每游戏日开始（0:00）刷新
- [x] 高级商品是否有前置解锁条件：是
  - 部分建筑解锁需前置建筑（如熔炉房需已拥有采掘场）
  - 部分受科技等级影响（如科技树Lv5解锁新建筑）
  - 详见 **9.2 商品清单** 各商品解锁条件列

### 售价调整规则（按难度）
| 参数 | 简单 | 普通 | 困难 | 极难 |
|------|------|------|------|------|
| 商店购买价格倍率 | ×0.7 | ×1.0 | ×1.3 | ×1.6 |
| 资源出售价格倍率 | ×1.4 | ×1.0 | ×0.7 | ×0.5 |

# 商店UI设计

### 入口与布局
- [x] 商店入口位置：主界面快捷键S / 顶部功能栏按钮
- [x] 界面整体布局：**列表式商店**
  - 垂直滚动列表，每行显示商品信息
  - 顶部显示当前星辉余额
  - 每项商品行：图标 + 名称 + 售价 + 库存/限购状态 + 购买按钮
- [x] 分类标签（顶部Tab切换）：地皮 / 建筑解锁 / 资源 / 增益道具

### 商品交互
- [x] 商品卡片显示内容：图标、名称、售价、限购状态、购买按钮
- [x] 购买确认方式：点击直接购买（资源类）/ 弹出二次确认窗口（地皮和建筑解锁类）
- [x] 是否支持批量购买：是，资源类支持加减按钮调整数量
- [x] 售罄/不可购买的视觉提示：灰色置灰 + 红色文字说明原因
  - 原因示例：星辉不足、已达限购上限、前置条件未满足

### 出售功能
- [x] 出售入口：商店面板内切换"出售"标签
- [x] 出售操作：选择资源 → 输入数量 → 确认出售
- [x] 是否显示出售前后的货币变化预览：是
- [x] 出售折价比例：不同资源不同折价
  - 基础资源（食物/水/木材/石头）：按购买价 25% 回收
  - 中级资源（矿石/垃圾）：按购买价 50% 回收
  - 高级/稀有资源（金属/肥料/燃料）：按购买价 75% 回收
