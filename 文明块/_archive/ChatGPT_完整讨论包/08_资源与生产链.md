# 生产链路总览

## 生产链路
- 树场 → 木材
- 矿场 → 矿石
- 熔炉房：矿石 → 金属
- 农田 → 食物
- 水井 → 水
- 垃圾处理场（多模式可切换）：
  - 模式A：垃圾 → 直接销毁（无产出，处理速度快）
  - 模式B：垃圾 → 随机回收资源（木材/石头/金属之一，少量）
  - 模式C：垃圾 → 肥料（可加速农田产出）
  - 模式D：垃圾 → 燃料（可提升熔炉房效率）

## 生产链路（交叉依赖）
<!-- 基础资源生产 -->
- 树场 → 木材 → 建造/维修
- 采掘场 → 矿石+石头 → 建造/加工
- 农田 → 食物 → NPC消耗
  └─ 运行消耗：0.5水/日（灌溉）
- 水井 → 水 → NPC消耗+农田灌溉+疗愈所
- 熔炉房：矿石+燃料(1/日) → 金属
  └─ 运行消耗：1燃料/日
- 农舍：6食物/工人/日（取代农田的高效农业）
  └─ 运行消耗：1水/日

<!-- 垃圾与回收系统 -->
- 每座运营建筑 → 1垃圾/日 → 垃圾处理场
- 垃圾处理场（多模式）：
  ├→ 模式A：垃圾直接销毁
  ├→ 模式B：垃圾→随机回收资源(木/石/金属)
  ├→ 模式C：垃圾→肥料→加速农田
  └→ 模式D：垃圾→燃料→供应熔炉房/太阳灯

<!-- 科技与进阶 -->
- 研究站 → 科技点 → 科技树解锁
  └─ 运行消耗：0.5食+0.5水/工人/日
- 疗愈所 → 延长寿命+防疾病
  └─ 运行消耗：1食+1水/日
- 工坊 → 相邻建筑耐久损耗-50%
- 管道 → 跨层资源自动传输

<!-- 防御系统 -->
- 哨塔+守卫 → 夜间消灭怪物
- 城墙 → 阻挡怪物路径(飞行怪除外)
- 陷阱 → 一次性消灭经过怪物

## 消耗链路
- NPC生存消耗：0.5 食物 + 0.5 水 / 游戏日 / 人
- 建筑建造消耗：木材、石头、金属
- 建筑运行消耗：详见 6.1 建筑运行消耗表

# 资源存储规则

### 存储体系（三层结构）
- [x] **建筑本地缓存**：每个生产建筑有独立存储空间
  - 建筑产出先存入自身缓存
  - 各建筑本地存储上限：
    - 农田/水井/树场/房屋/太阳灯：20 单位
    - 采掘场/熔炉房/农舍/垃圾处理场：30 单位
- [x] **全局仓库**：所有资源共享的存储池
  - 建筑本地缓存满后，溢出资源自动进入全局仓库
  - 全局仓库基础上限：2000 容量
- [x] **仓库建筑扩容**：可建造"仓库"建筑扩展全局存储上限
  - 每个仓库增加全局存储容量：+200 容量

### 资源分类存储上限（全局仓库内细分）
| 资源类型 | 单类上限 | 备注 |
|---------|---------|------|
| 食物 | 1000 | 超出部分溢出处理 |
| 水 | 1000 | 超出部分溢出处理 |
| 木材 | 800 |  |
| 石头 | 800 |  |
| 矿石 | 500 |  |
| 金属 | 400 |  |
| 燃料 | 300 |  |
| 垃圾 | 500 | 单独处理（见 4.4） |
| 科技点 | 无上限 | 非存储资源 |

### 存储联动规则
- [x] 跨层建筑资源是否自动连通：否（需建造管道连接上下层间资源传输）
- [x] 资源运输是否有时间/成本消耗：是
  - 跨层资源运输（通过管道）每层有 1 秒传输延迟
  - 同一层内资源流转无延迟
- [x] **资源溢出后的处理逻辑**（超出全局上限）：
  - 玩家可选两种策略（在设置面板切换）：
    1. **自动销毁**（默认）：溢出资源直接丢弃，不做任何保留
    2. **暂停生产**：当某类资源到达上限后，产出该资源的建筑自动暂停（显示"📦 存储已满"）
  - 切换策略即时生效

### 存储告警系统
- [x] 全局存储使用率达 80%：HUD资源栏变黄色 + 闪烁提示
- [x] 全局存储使用率达 90%：HUD资源栏变红色 + 持续闪烁 + 顶部弹出"⚠️ 存储空间不足"横幅
- [x] 任一类资源达到单类上限：该类资源图标显示红色边框

### 资源消耗优先级
- [x] 建造/生产消耗资源时，优先从哪个存储池扣减：
  - 优先从全局仓库扣减
  - 全局仓库不足时，从建筑本地缓存补充

# 垃圾机制

### 垃圾产生
- [x] 垃圾产生来源：生产建筑运行、NPC死亡
- [x] 单位时间垃圾产生量：每座运行中的建筑产生 **1 垃圾/游戏日** + 每名NPC产生 **0.5 垃圾/游戏日**
  - 建筑闲置（无工人）不产生垃圾
  - NPC死亡时额外产生 1 垃圾
- [x] 总垃圾产生公式：运行建筑数 + NPC数×0.5 / 游戏日
- [x] 垃圾处理参考：1名清洁工可处理约 5-8 座运行建筑的垃圾产出

### 垃圾不处理的负面效果
- [x] 垃圾未处理时：每日占用全局存储空间（垃圾=存储类资源，上限500）
- [x] 垃圾堆积超过200：NPC满意度每日 -1%（全员工作效率 -1%，可累积至 -20%）
- [x] 垃圾堆积超过400：NPC满意度每日 -2%，疾病概率 +5%/日
- [x] 垃圾满500后：溢出垃圾丢弃，但负面效果持续累积
- [x] 垃圾清零后：满意度负面效果立即消除，疾病概率次日恢复

### 垃圾处理场
- [x] 处理模式：4种模式可手动切换
  1. **直接销毁**：垃圾直接清除，无产出，处理速度最快
  2. **资源回收**：垃圾转化为少量随机资源（木材 / 石头 / 金属之一）
  3. **堆肥制肥**：垃圾转化为肥料，可加速农田产出
  4. **燃料提炼**：垃圾转化为燃料，可提升熔炉房效率
- [x] 垃圾处理场单周期处理量：**6 垃圾/游戏日**（1工人基础值）
- [x] 各模式处理效率与产出数值：

| 处理模式 | 处理速度 | 每1垃圾产出 | 备注 |
|---------|---------|------------|------|
| A-销毁 | 10 垃圾/日 | 无 | 最快清除，无收益 |
| B-回收 | 6 垃圾/日 | 回收1随机资源(木材40%/石头40%/金属20%) | 资源循环 |
| C-堆肥 | 6 垃圾/日 | 产出1肥料（可使1座农田下次产出+50%）+ 每5垃圾额外产出1染料 | 农业增益+染料副产 |
| D-提炼 | 6 垃圾/日 | 产出0.3燃料 | 工业增益 |

### 存储规则
- [x] 垃圾可堆积：是
- [x] 垃圾堆积上限：500（全局仓库内）
- [x] 垃圾超出上限后：溢出自动丢弃
- [x] 垃圾存储不影响其他资源的分类上限

### 与存储告警联动
- [x] 垃圾达400时触发存储黄色告警（同全局存储80%）
- [x] 垃圾达500时触发存储红色告警（同全局存储90%）
- [x] 详见 **4.3 存储规则** § 存储告警系统
