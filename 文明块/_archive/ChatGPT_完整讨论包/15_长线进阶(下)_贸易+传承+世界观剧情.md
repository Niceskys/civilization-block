## 二、贸易路线系统

### 开通规则
- [x] 可通过商店购买"贸易路线开通许可"：100星辉（一次性永久）
- [x] 开通后每 5 游戏日获得一次外部特产
- [x] 特产自动送达全局仓库（带运输队动画）

### 贸易路线等级
| 等级 | 升级条件 | 每5日获得 | 额外效果 |
|------|---------|----------|---------|
| Lv1 (基础) | 开通 | 随机资源×20（食物/木材/石头之一） | - |
| Lv2 | 拥有5座不同建筑 | 随机资源×30 + 5星辉 | - |
| Lv3 | 人口≥15 | 随机资源×40 + 10星辉 | 低概率获得染料/肥料 |
| Lv4 | 拥有≥3种不同职业 | 随机资源×50 + 15星辉 | 稳定获得1染料或1肥料 |
| Lv5 (满级) | 人口≥25 + 累计贸易10次 | 随机资源×60 + 25星辉 + 1BOSS令牌碎片 | 极低概率获得随机传奇特质书 |

### 升级消耗
| 升级 | 消耗 | 说明 |
|------|------|------|
| Lv1→Lv2 | 50星辉 | 扩大商路规模 |
| Lv2→Lv3 | 100星辉+10木材 | 修建驿站 |
| Lv3→Lv4 | 200星辉+10金属 | 组建护卫队 |
| Lv4→Lv5 | 500星辉+20金属+3BOSS令牌 | 建立永久商路要塞 |

### 贸易路线视觉表现

- [x] 每次特产送达：一队小推车从地图边缘进入→走向仓库→资源入库

# 传承模式（存档间联动）
> 方案C：星辉遗产 + 遗物继承

---

## 一、核心概念

- [x] 传承模式允许玩家在不同存档之间继承部分资产
- [x] 目标是让每次新开局比上一次"更强一点点"，形成"越玩越强"的正向循环
- [x] 传承数据存储在本地（跨存档共享的`legacy.dat`文件）

---

## 二、星辉遗产系统

### 遗产积累
- [x] 每个存档在**结束**（NPC全灭/玩家手动弃档）时，自动结算遗产：
  - 遗产 = 当前星辉余额 × 10%
  - 遗产自动存入全局遗产池（跨存档）
- [x] 遗产池上限：5000 星辉（超出部分不累积）
- [x] 遗产池不因任何操作减少（只进不出，直到被提取）

### 遗产提取
- [x] 新游戏开局时，可选择"提取遗产"选项
- [x] 每次提取上限：500 星辉
- [x] 每次提取后遗产池对应减少（如遗产池1000，提取500→剩余500）
- [x] 遗产提取与成就解锁选项互相独立（可同时使用）

### 示例
```
存档1：玩了50日，结束时星辉余额2000 → 存入遗产池 200
存档2：玩了30日，结束时星辉余额500 → 存入遗产池 +50，总遗产250
新存档3：开局提取250星辉 → 遗产池剩余0，开局自带250星辉（+基础配置）
```

---

## 三、遗物继承系统

### 遗物获取
- [x] 每个存档在**结束**时，随机保留1件"遗物"
- [x] 遗物从以下来源中随机抽取（优先级：从高到低）：
  1. 背包中未使用的BOSS令牌 → 30%概率保留
  2. 背包中未使用的古代数据碎片 → 25%概率保留
  3. 当前拥有传奇特质但已死亡的NPC → 15%概率保留其"传奇遗物"
  4. 地下挖掘第10层的"地下金库"图纸 → 10%概率保留
  5. 其他随机稀有物品 → 20%概率保留

### 遗物池
- [x] 遗物池最多存储 10 件
- [x] 达到上限后不再获得新遗物（或玩家手动删除旧遗物腾出空间）
- [x] 遗物在"传承面板"中展示（图标+名称+来源存档名+简短描述）

### 遗物类型与效果
| 遗物 | 来源 | 新档效果 | 描述 |
|------|------|---------|------|
| 🏅 BOSS令牌 | 击败BOSS | 开局获得1个BOSS令牌 | "前任文明的战利品" |
| 💿 古代数据碎片 | 地下城/击败混沌核心 | 开局科技点+5 | "远古知识的残片" |
| 📜 传奇遗物 | 传奇特质NPC死亡 | 新档开局一名NPC获得该传奇特质 | "英雄的遗志" |
| 🏗️ 金库蓝图 | 地下10层 | 开局解锁地下金库建造 | "深埋地下的宝藏图" |
| 🛡️ 远古盾牌 | 击败石甲巨兽 | 开局城墙初始+500耐久 | "经历过无数战斗的盾" |
| ⚗️ 生命药剂 | 击败暗影蝠王小概率 | 开局全NPC寿命+5日 | "蕴含着生命力的药水" |

### 遗物继承流程
- [x] 新游戏开始前 → 进入"传承面板"
- [x] 显示当前遗产池星辉数 + 遗物池列表
- [x] 玩家选择是否提取遗产（每次最多500）
- [x] 玩家选择是否携带遗物（最多1件）
- [x] 选择确认后 → 开始新游戏 → 初始背包包含所选遗产+遗物

# 世界观与隐藏剧情
> 原创故事线 | 基于叙事心理学研究 | 避免"模拟器""观察者"等过度使用设定

---

## 一、原创性审查声明

### 已删除的过度使用设定
| 旧设定 | 原因 | 替换方案 |
|--------|------|---------|
| "四次文明轮回的实验场" | 与《三体》"文明实验"高度重合 | → 单一真实世界，历史被"遗忘"而非"被操控" |
| "观察者从外部监视" | 与《楚门的世界》核心设定雷同 | → 改为"来自地心的共鸣信号"—不是外部监视而是内部传递 |
| "按钮选择终结世界" | 与《尼尔》结局A/B结构相似 | → 改为"两种建造哲学的选择"，不涉及毁灭/逃脱 |
| "沙盒中的种子" | 与《黑客帝国》"电池农场"设定撞车 | → 改为"继承者"——玩家是被选中的历史挖掘者，而非实验品 |
| "银白色眼球BOSS" | 视觉与《塞尔达·旷野之息》守护者雷同 | → 改为"晶石巨构"——纯粹矿物水晶结构，无生物特征 |

### 叙事原则（基于游戏叙事研究成果）
- **原则1：世界必须是真实的** — 玩家建造的城市是真实的定居点，不是"模拟"。真实感产生情感重量。
- **原则2：玩家的选择定义身份** — 不是"逃脱"或"毁灭"，而是"你建造了什么"决定你是谁。
- **原则3：未被发现的真相，而非被隐藏的真相** — 历史没有被"掩盖"，而是被"遗忘"。发现真相的过程是考古而非解谜。
- **原则4：结局必须可延续** — 通关存档后玩家应该想"再来一局"而非感觉"结束了"。

---

## 二、世界背景

### 核心设定
- [x] **时间**：不确定的未来。人类历史出现过一次"大断层"——史书记录在某个世纪之后出现了约400年的空白。无人知道那400年间发生了什么，只知道从那之后，世界被一层能量场——**"晶脉"**所覆盖。
- [x] **晶脉**：一种从地心涌出的结晶能量，在空气中凝结成**方块**。方块可以被开采、加工、重组为建筑。它是这个世界最基础的构成单位。
- [x] **前文明**：在大断层之前，存在过一个极度发达的"熔铸文明"。他们掌握了晶脉能量的核心秘密，建造了遍布全球的晶能网络。但他们在大断层中消失了——没有战争痕迹，没有大规模死亡遗迹，仿佛**整个文明集体选择了某个方向，然后走了进去**。
- [x] **玩家角色**：你不是开拓者。你是**掘墓人（Tomb-Digger）**——一个专门搜索前文明遗迹的孤独职业。你在一次常规挖掘中，发现了一块不属于任何已知文明的晶石。这块晶石**会回应你的触碰**。故事从此开始。

### 与旧设定的关键区别
| 旧版 | 新版（原创） |
|------|------------|
| 多次文明轮回 | → 单次"大断层"，历史连续但中断 |
| 外部观察者在监视 | → 内部"共鸣信号"从地心发出 |
| 玩家是被实验的种子 | → 玩家是主动的"掘墓人"，历史由他发现 |
| 结局是逃脱/终结 | → 结局是"两大建造阵营"的选择 |

---

## 三、碎片化叙事——"回声文本"

### 什么是"回声"
- [x] 那块特殊的晶石被命名为**"共鸣晶石"**
- [x] 它储存着前文明最后一代人的记忆片段——不是文字，不是声音，而是**"情感+画面"的脉冲**。玩家触碰即"看到"。
- [x] 这些片段是不完整的、跳跃的、每个人的视角都不同的。
- [x] 玩家收集的越多，拼凑出的画面就越完整——但永远不会100%清晰，留白是设计的一部分。

| 外观 | 纯粹由半透明水晶构成的多面体结构，高4格，宽度约2.5格。**没有任何生物特征**——没有眼睛、没有口器、没有四肢。它在空中缓慢旋转，每一面晶面都折射出不同颜色的光芒。它移动时不发出声音。它不是"活"的。它是**地质构造**。 |
| 生命值 | 普通怪物×25 |
| 攻击方式 | ① **晶刺**——地面升起水晶尖刺（对单格建筑造成150伤害），可提前看到地面发光预判躲避 |
|  | ② **共鸣波**——释放环形冲击波（全建筑减10耐久），无预兆但伤害较低 |
|  | ③ **晶体化**——锁定一座建筑，3秒内将其覆盖水晶（建筑暂时失效，持续15秒），可以被其他建筑阻挡视线 |
|  | ④ **无撤退机制**——它不会撤退。要么你击败它，要么它摧毁你全部防御后消失。 |
| BOSS战时限 | 6分钟（最长BOSS战） |
| 击败后掉落 | 🔷 **共鸣晶石·完整形态**（共鸣晶石升级为可摆放装饰，持续发出低频共鸣声） |
|  | 📀 **回声·终章**（无法收集，击败后自动播放一次） |
|  | 💠 **晶脉碎片×5**（可用于科技树特殊升级） |

### BOSS战叙事 + 战胜后的回声·终章
- [x] 守卫出现时：没有红色警报。天空没有变色。只是所有建筑表面的晶石装饰同时开始发光。屏幕中央显示一行小字：*"它来了。你知道它为什么来。"*
- [x] 战斗过程中，地面上的晶脉会随着守卫的攻击节奏明暗闪烁
- [x] **击败后，共鸣晶石自动播放最后一段回声：**
  > *你站在一片纯白的空间中。没有地面，没有天空。*
  > *前方有一个模糊的人影——轮廓像那个年长者，但她在发光。*
  > *她对你微笑——你感觉自己认识她。非常熟悉。像是你从未见过的家人。*
  > *她说："你做到了。你收集了所有的碎片，你面对了守卫，你没有逃走。"*
  > *"现在你有两个选择。不是关于我们——关于你自己。"*
  > *"你手里拿着我们所有人的记忆。你可以选择——"*
  > *她把一只手放在胸口："把它们埋进地心。让前文明真正安息。然后你建造属于你自己的世界，不再被我们的影子覆盖。"*
  > *她伸出另一只手，掌心朝上："或者，你可以带着我们的记忆回到地面。把我们的知识教给活着的人。不是让他们成为我们——而是让他们比我们更好。"*
  > *她收回手，微笑着说："选择没有对错。选择只是……你是谁。"*
  > *画面淡出。*

## 五、结局设计（两种建造哲学）

### 触发条件
- [x] 已击败晶脉守卫
- [x] 共鸣晶石已完成"完整形态"
- [x] 拥有至少10座建筑
- [x] 游戏日≥100日
- [x] 此时HUD顶部共鸣晶石图标变为可点击状态——不是"出口"，而是"**归途**"

### 结局A：深埋（Build Anew）
| 项目 | 描述 |
|------|------|
| 名称 | 🪨 **深埋——从零开始** |
| 操作 | 点击"归途" → 选择"将记忆埋入地心" |
| 叙事 | *你带着共鸣晶石回到发现它的那个地下洞穴。你蹲下身，把它放回它原本所在的位置。你看着晶石的光慢慢变暗——它没有熄灭，只是变成了稳定的、安静的微光。它不再说话了。它睡着了。你站起来，转身离开。回到地面上，你的城市还在那里。你的NPC还在等你分配工作。今天的风很大，农田里的作物在摇摆。还有一座哨塔需要维修……明天再说吧。* |
| 游戏效果 | ❌ 游戏不结束。存档继续运行，无任何标记。 |
| 传承奖励 | ✅ 此存档获得"深埋"标记（非通关标记，不显示🏆，但数据保留） |
|  | ✅ 传承模式获得遗物**"沉默的晶石"**：携带此遗物开局，第一次地下挖掘的前3层不消耗工具（节约金属） |
| 设计意图 | 👤 选择"自力更生"的玩家——他们不需要前文明的指引，他们要自己走出一条路 |

### 结局B：传承（Carry Forward）
| 项目 | 描述 |
|------|------|
| 名称 | 🔥 **传承——带着记忆走下去** |
| 操作 | 点击"归途" → 选择"带着记忆回到地面" |
| 叙事 | *你从口袋里拿出共鸣晶石。它感觉到你的决心——它的光变亮了。你走到城市最高的建筑顶上（即使它只有2层高）。你把晶石举过头顶。晶石的光芒向四面八方扩散，像一波涟漪划过天空。地面上，所有建筑表面的晶石同时亮起——一瞬——然后熄灭。从这一天起，你的城市没有发生任何可见的变化。但你知道某些东西变了。你知道前文明的故事。你知道他们的伟大和他们的教训。你不再只是一个掘墓人了。你是一座桥梁。* |
| 游戏效果 | ✅ 存档标记为🏆 **"传承者"**（通关标记） |
|  | ✅ 所有已解锁的成就选项永久保留至传承模式（即使你删除此存档） |
|  | ✅ 此存档中仍在世的NPC获得一个永久隐藏效果：全属性+3（不会被玩家看到，但数据在） |
| 传承奖励 | 🏅 遗物**"共鸣碎片"**：携带此遗物开局，研究站科技点产出+1/日 |
| 设计意图 | 👤 选择"承前启后"的玩家——他们尊重过去，但更关注未来 |

### 与旧版的关键区别
| 旧版结局 | 新版结局 | 原创性改进 |
|---------|---------|-----------|
| "成为新的观察者"（延续实验） | → "深埋"（自主建造，不依赖前文明） | ✅ 没有变成"新的控制者"，聚焦于玩家自己的建设 |
| "让一切终结"（毁灭） | → "传承"（带历史前行，不让历史消失也不被历史束缚） | ✅ 没有毁灭选项，所有人都是建设性的选择 |
| 钥匙从BOSS掉落 | → 钥匙一直在玩家自己手中的共鸣晶石里 | ✅ 主动权在玩家手中，不是等待外部给予 |
| 结局后游戏结束 | → 两个结局存档都可继续游玩 | ✅ 沙盒精神保留，不强制"终局" |

## 六、隐藏建筑"归墟之碑"（原"神像"的重设计）

### 设计变更原因
- 旧版"神像"概念过于通用（"先民神像"类似的造像在无数游戏里出现）
- 新版"归墟之碑"的定义：**它不是一个崇拜对象。它是一个信息中继站。**

### 解锁方式
- [x] 集齐3个BOSS令牌后，商店出现选项"建造归墟之碑"（免费，消耗3令牌）
- [x] 每局游戏只能建造1个

### 归墟之碑效果
| 项目 | 描述 |
|------|------|
| 名称 | 🏛️ **归墟之碑** |
| 命名含义 | "归墟"——传说中百川归海之处，引申为"一切记忆的汇聚点" |
| 占地 | 2×2 |
| 建造成本 | 3个BOSS令牌 + 50石头 + 20金属 |
| 外观 | 一个由七块不同颜色的晶石板构成的圆环结构。没有人物形象，没有动物图腾。只有晶石。每一块晶石板代表一段回声。圆环中央空无一物——因为观看者站在中央时，看到的是**自己的倒影**。 |
| 功能 | ① 每日产出5星辉（自动入账） |
|  | ② 周围1格内建筑效率+5% |
|  | ③ 触发核心碎片·钥匙的获取（完成全部回声收集） |
| 特殊 | 碑不可染色，不可拆除，不可移动。夜间七块晶石板发出不同颜色的微光 |

## 八、核心主题总结

| 主题 | 表达方式 |
|------|---------|
| 🏗️ **建造者的尊严** | 你的城市是你建造的，不是任何人给你的。前文明的知识是参考，不是蓝图 |
| 💎 **记忆的重量** | 知道历史不会让你变强，但会让你在选择时更清醒 |
| 🔄 **传承不复制** | "传承"不是模仿前文明，而是避免他们犯过的错 |
| ❓ **没有标准答案** | 两个结局没有"好"与"坏"，只有"适合你" |
| 🌱 **开放结局** | 两个选择都不结束游戏。沙盒不会因为"剧情通关"而失去意义 |
