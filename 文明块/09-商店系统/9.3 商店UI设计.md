# 商店UI设计

### 入口与布局
- [x] 商店入口位置：主界面快捷键S / 顶部功能栏按钮
- [x] 界面整体布局：**列表式商店**
  - 垂直滚动列表，每行显示商品信息
  - 顶部显示当前星辉余额
  - 每项商品行：图标 + 名称 + 售价 + 库存/限购状态 + 购买按钮
- [x] 分类标签（顶部Tab切换）：地皮 / 建筑解锁 / 资源 / 增益道具

### 商品交互
- [x] 商品卡片显示内容：图标、名称、售价、限购状态、购买按钮
- [x] 购买确认方式：点击直接购买（资源类）/ 弹出二次确认窗口（地皮和建筑解锁类）
- [x] 是否支持批量购买：是，资源类支持加减按钮调整数量
- [x] 售罄/不可购买的视觉提示：灰色置灰 + 红色文字说明原因
  - 原因示例：星辉不足、已达限购上限、前置条件未满足

### 出售功能
- [x] 出售入口：商店面板内切换"出售"标签
- [x] 出售操作：选择资源 → 输入数量 → 确认出售
- [x] 是否显示出售前后的货币变化预览：是
- [x] 出售折价比例：不同资源不同折价
  - 基础资源（食物/水/木材/石头）：按购买价 25% 回收
  - 中级资源（矿石/垃圾）：按购买价 50% 回收
  - 高级/稀有资源（金属/肥料/燃料）：按购买价 75% 回收
