# 商店通用规则

### 价格权威边界

- `9.2 商品清单`维护常驻商店的基础单价、商品效果和常驻限购规则。
- 本文件维护常驻商店的难度倍率、资源回收价和刷新规则。
- `1.6 随机事件系统 § 商人交易`维护随机商人的固定礼包价与单次来访限购；该价格是独立事件价，不要求与9.2按单位换算后相同。
- 常驻商店难度倍率不得再次应用到随机商人价格；随机商人固定价也不得回写覆盖9.2基础单价。
- 同一购买事务必须记录价格来源为“常驻商店”或“随机商人”，只能选择一套价格和限购规则。
- 地皮最终价格按固定顺序结算：读取9.2基础价→应用8.3已解锁的地皮折扣→应用本文件的商店购买难度倍率。不得调换顺序或重复应用。

### 基础规则
- [x] 商店解锁条件：初始直接解锁（游戏开始时商店即可用）
- [x] 商品刷新规则：混合制
  - 基础商品（地皮、基础建筑解锁等）：固定永久在售
  - 稀有商品（高级建筑、增益道具等）：每游戏日刷新一次
  - 周常商品：每 5 游戏日刷新一次
- [x] 通用货币名称：**星辉**
- [x] 是否支持资源出售换货币：是
  - 基础资源（食物/水）：出售折价 25%（买入5星辉→卖出1星辉）
  - 木材/石头：出售折价 25%（买入8-10星辉→卖出2星辉）
  - 矿石：出售折价 50%（买入15星辉→卖出7星辉）
  - 金属：出售折价 75%（买入30星辉→卖出22星辉）

### 货币获取途径
- [x] 出售多余资源获得星辉
- [x] 达成里程碑/成就奖励星辉
- [x] 完成任务奖励星辉（见 10.2 任务清单）
- [x] 随机商人交易（见 1.6 随机事件系统）
  - 随机商人的限购折扣与资源回售差额属于低频事件收益，允许玩家在限购范围内获得星辉净收益
- [x] 初始开局星辉（按难度：简单200/普通100/困难50/极难0）

### 购买限制
- [x] 常驻商品是否有购买数量上限：是
  - 资源类每日限购（食物10份/日、水10份/日、木材10份/日等）
  - 建筑解锁类一次性购买永久生效
  - 增益类商品无日限，按游戏日刷新
- [x] 每日购买额度刷新时间：每游戏日开始（0:00）刷新
- [x] 高级商品是否有前置解锁条件：是
  - 部分建筑解锁需前置建筑（如熔炉房需已拥有采掘场）
  - 部分受科技等级影响（如科技树Lv5解锁新建筑）
  - 详见 **9.2 商品清单** 各商品解锁条件列
