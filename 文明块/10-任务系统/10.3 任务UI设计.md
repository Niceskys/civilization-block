# 任务UI设计

### 入口与布局
- [x] 任务入口位置：主界面快捷键Q / 顶部功能栏按钮
- [x] 界面整体布局：
  - 左侧分类标签 + 中间任务列表 + 右侧任务详情与奖励预览
- [x] 分类标签：主线 / 日常 / 周常 / 里程碑 / 随机事件 / 已完成

### 任务追踪
- [x] **无常驻追踪UI** — 保持画面简洁
  - 仅在任务面板（Q键）中查看所有任务
  - 主线任务触发时弹出一次通知提示
  - 任务完成时弹出奖励领取提示

### 任务列表显示
- [x] 单条任务显示内容：任务名称、进度条、奖励图标、状态标记
  - 状态：🔵进行中 / 🟢可领取 / ⚪未接取 / ✅已完成
- [x] 进行中任务是否显示实时进度数值：是
- [x] 已完成任务是否保留显示：是，保留至当前游戏日结束

### 交互规则
- [x] 点击任务可展开详情：完整目标描述、奖励明细、倒计时（限时任务）
- [x] 可领取任务高亮显示，点击「领取奖励」按钮直接到账
- [x] 限时任务显示剩余时间，倒计时结束自动标记失败
- [x] 日常/周常任务：点击"接取"按钮激活，每日手动接取
