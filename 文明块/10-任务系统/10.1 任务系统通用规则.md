# 任务系统通用规则

### 任务分类
- [x] 发展篇章：一次性，引导玩家熟悉完整生存循环，自动记录但可以隐藏
- [x] 城市委托：玩家从委托池主动选择的短中期目标，不按现实或游戏日自动过期
- [x] 成就/里程碑：自动记录长期成果，可对应平台成就，但不构成通关要求
- [x] 随机事件任务：随机触发，限时完成，特殊奖励
- [x] 季节主题委托：首次遇到对应季节时解锁，永久保留，不计入普通任务上限，具体生命周期以1.6为准

### 触发与领取规则
- [x] 任务触发方式：
  - 发展篇章：满足条件时自动记录；通知和教程去重，可隐藏追踪
  - 成就/里程碑任务：达成条件时自动触发
  - 城市委托：玩家手动在委托面板接取
  - 随机事件任务：随机自动触发
- [x] 同时进行任务数量上限：城市委托最多3个；发展篇章、里程碑和季节记忆不占该上限
- [x] 任务是否可以放弃：是，放弃后视为自动失败
  - 城市委托放弃无资源或声望惩罚，目标返回委托池后可再次出现
  - 发展篇章不能被删除，但可以关闭通知和停止追踪
- [x] 限时任务超时处理：自动失败（显示"⏰ 任务失败"），无惩罚
- [x] 季节主题委托不是限时任务，不适用自动失败、刷新清空或等待下一年规则

### 奖励规则
- [x] 奖励发放方式：完成自动发放（到账弹出通知）
- [x] 奖励类型：资源、星辉（货币）、地皮、解锁建筑、NPC、增益效果
- [x] 任务难度与奖励倍率对应关系：城市委托10-200星辉 / 里程碑300-1000星辉
- [x] 任务奖励是否受游戏难度档位影响：是（简单难度奖励×1.2，困难×0.8，极难×0.6）

### 刷新规则
- [x] 城市委托不会因日期切换清空；完成或放弃后，空位从适合当前城市状态的委托池补充
- [x] 未接取的委托候选可以随城市阶段更新，但不得显示现实时间或“今日不做就损失”提示
- [x] 发展篇章和里程碑永久保留进度
- [x] 季节主题委托的目标、进度、奖励领取状态永久保存，季节切换不清空

### 沙盒边界

- 任务奖励可以提供资源、星辉、装饰或可选便利，但不得成为基础建造、生产、NPC管理、防御或探索的唯一入口。
- 玩家隐藏全部任务UI后，模拟与所有已满足前置条件的核心操作仍可正常进行。
- 任务完成状态不得被解释为存档“通关”；开放式叙事选择和传承规则见1.4与8.5。
