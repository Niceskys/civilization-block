# 任务系统通用规则

### 任务分类
- [x] 主线引导任务：一次性，引导新手熟悉核心玩法，逐步解锁系统
- [x] 日常任务：每日刷新，重复完成，提供小额奖励
- [x] 周常任务：每5游戏日刷新，奖励高于日常
- [x] 成就/里程碑任务：一次性，达成永久奖励，可对应Steam成就
- [x] 随机事件任务：随机触发，限时完成，特殊奖励

### 触发与领取规则
- [x] 任务触发方式：
  - 主线引导任务：自动触发推送
  - 成就/里程碑任务：达成条件时自动触发
  - 日常任务：玩家手动在任务面板接取
  - 周常任务：玩家手动在任务面板接取
  - 随机事件任务：随机自动触发
- [x] 同时进行任务数量上限：日常最多3个 + 周常1个 + 主线不限 + 里程碑不限
- [x] 任务是否可以放弃：是，放弃后视为自动失败
  - 日常任务放弃后：可重新接取（刷新周期内）
  - 周常任务放弃后：不可重新接取，等待下次刷新
  - 主线任务不可放弃
- [x] 限时任务超时处理：自动失败（显示"⏰ 任务失败"），无惩罚

### 奖励规则
- [x] 奖励发放方式：完成自动发放（到账弹出通知）
- [x] 奖励类型：资源、星辉（货币）、地皮、解锁建筑、NPC、增益效果
- [x] 任务难度与奖励倍率对应关系：日常10-30星辉 / 周常80-200星辉 / 里程碑300-1000星辉
- [x] 任务奖励是否受游戏难度档位影响：是（简单难度奖励×1.2，困难×0.8，极难×0.6）

### 刷新规则
- [x] 日常任务刷新时间：每 1 游戏日刷新
- [x] 周常任务刷新周期：每 5 游戏日刷新
- [x] 刷新后未完成的任务处理：日常任务清空，周常任务清空，里程碑永久保留

# 任务UI设计

### 入口与布局
- [x] 任务入口位置：主界面快捷键Q / 顶部功能栏按钮
- [x] 界面整体布局：
  - 左侧分类标签 + 中间任务列表 + 右侧任务详情与奖励预览
- [x] 分类标签：主线 / 日常 / 周常 / 里程碑 / 随机事件 / 已完成

### 任务追踪
- [x] **无常驻追踪UI** — 保持画面简洁
  - 仅在任务面板（Q键）中查看所有任务
  - 主线任务触发时弹出一次通知提示
  - 任务完成时弹出奖励领取提示

### 任务列表显示
- [x] 单条任务显示内容：任务名称、进度条、奖励图标、状态标记
  - 状态：🔵进行中 / 🟢可领取 / ⚪未接取 / ✅已完成
- [x] 进行中任务是否显示实时进度数值：是
- [x] 已完成任务是否保留显示：是，保留至当前游戏日结束

### 交互规则
- [x] 点击任务可展开详情：完整目标描述、奖励明细、倒计时（限时任务）
- [x] 可领取任务高亮显示，点击「领取奖励」按钮直接到账
- [x] 限时任务显示剩余时间，倒计时结束自动标记失败
- [x] 日常/周常任务：点击"接取"按钮激活，每日手动接取
