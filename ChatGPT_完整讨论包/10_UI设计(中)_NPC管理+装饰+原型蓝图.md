# NPC管理UI

### NPC列表面板（快捷键N）
- [x] 悬浮式面板，可拖拽
- [x] 顶部筛选栏：全部 / 空闲 / 工作中 / 按职业筛选 / 按性格筛选
- [x] 列表项显示：头像 + 姓名 + 年龄 + 性格标签 + 当前状态 + 职业图标
- [x] 点击NPC条目→展开NPC详情

### 个体外观差异系统
- [x] **姓名**：随机姓+名组合（200+名库避免重复）
- [x] **外观**：
  - 性别：男/女（体素方块人风格，不同体型方块组合）
  - 体型：瘦小/标准/魁梧（影响移动速度微妙差异±5%）
  - 发型发色：6种发型×8种发色（共48种组合）
  - 肤色：3种基调
  - 职业服装：被分配工作后穿上对应职业服装（农民草帽、铁匠皮围裙、守卫铠甲等8种）
- [x] **声音反馈**：
  - 选中时：简短语音（"是！""好的""明白"）
  - 工作音效：挥锄/打铁/砍树/推车滚动声（距离决定音量）
  - 情绪音效：开心（丰收时欢呼）、紧张（怪物入侵时惊叫）

### 性格系统（6种性格标签，出生时随机分配）
| 性格 | 正面影响 | 负面影响 | 标签颜色 |
|------|---------|---------|----------|
| 勤奋 | 工作间歇缩短30% | 疲劳积累快20% | 🟢绿色 |
| 懒惰 | 食物消耗-20% | 工作间歇延长40% | 🟡黄色 |
| 乐观 | 恐惧值恢复速度+50% | 效率略微波动±5% | 🔵蓝色 |
| 悲观 | 高警觉(怪物感知+1格) | 恐惧值恢复速度-30% | 🟣紫色 |
| 勇敢 | 守卫职业效率+25% | 更容易冲上前线受伤 | 🔴红色 |
| 谨慎 | 建筑维修消耗-20% | 移动速度-10% | ⚪灰色 |

### NPC详情面板
显示信息：
- 基础信息：姓名、性别、年龄、剩余寿命、**性格标签+性格描述**
- **外观小头像**（动态生成，反映当前体型+发型+服装）
- 7维属性：力量/敏捷/智力/体力/创造力/社交/幸运（数值+柱形图）
- 职业信息：已学职业列表（最多3个）+ 当前从事职业
- 工作状态：当前分配建筑、工作效率百分比（性格影响已计入）
- 生育状态：是否可生育、冷却剩余时间
- **日常轨迹**：最近5个游戏日的行动日志（起床→工作→吃饭→睡觉循环）
- 快捷按钮：转职 / 促育 / 分配工作 / 切换白班夜班

### NPC日常行为动画循环
| 时段 | 行为 | 动画 |
|------|------|------|
| 白天工作 | 在建筑内执行职业动作 | 依职业不同（见5.3动画表） |
| 白天空闲 | 在地皮公共区域闲逛 | 慢速走动/坐下休息/与其他NPC短暂交谈 |
| 白天休息(夜班NPC) | 在房屋床上睡觉 | 躺卧闭眼，zzZ气泡 |
| 夜晚工作(夜班NPC) | 在建筑内工作 | 同白天动画，但灯光效果更暖 |
| 夜晚睡眠 | 在房屋床上睡觉 | 躺卧闭眼 |
| 怪物入侵时 | 非守卫NPC跑向最近房屋躲避 | 奔跑动画+惊恐表情气泡 |
| 生育时 | 两名NPC短暂靠近 | 心形粒子+婴儿NPC生成 |

### 操作功能
- [x] 转职按钮：消耗50星辉 + 2游戏日冷却
- [x] 促育按钮：选中目标NPC配对
- [x] 分配工作按钮：跳转建筑选择界面
- [x] 白班/夜班切换按钮

### 排序与统计
- [x] 按属性排序（如找最高力量NPC分配给采掘场）
- [x] 按性格排序（如找勇敢NPC分配给哨塔）
- [x] 总人口统计：成年/婴儿/男性/女性/空闲/各性格数量

# 装饰与自定义系统
> 方案C：建筑染色 + 装饰性地标 + 成就雕像

---

## 一、建筑染色系统

### 基础规则
- [x] 玩家可消耗 **染料** 给已建成的建筑更换配色
- [x] 每个建筑有 6 种预设配色方案（含默认色）
- [x] 染色操作：选中建筑 → 详情面板 → "染色"按钮 → 选择配色 → 确认消耗
- [x] 染色不影响建筑功能、等级、耐久

### 6种预设配色方案
| 配色名 | 主色调 | 适合建筑类型 |
|--------|--------|-------------|
| 默认色 | 标准（生产棕黄/居住米白/防御灰蓝） | 全部 |
| 温暖橙 | 橙红+米黄 | 生产类 |
| 冷冽蓝 | 深蓝+银灰 | 防御类 |
| 自然绿 | 草绿+木棕 | 生产类/居住类 |
| 皇家紫 | 紫色+金色 | 科技类/功能类 |
| 暗影黑 | 黑灰+暗红 | 防御类 |

### 染料获取
| 获取方式 | 产出量 | 备注 |
|---------|--------|------|
| 垃圾处理场模式C新增产量途径 | 1染料/5垃圾 | 与肥料并行产出 |
| 商店购买 | 10星辉/份 | 限购3份/日 |
| 正向爆点事件"天降财宝" | 附带+5染料 | 随机奖励 |
| 商人来访 | 8星辉/份 | 限购2次/来访 |

### UI表现
- [x] 染色确认时：建筑短暂闪烁→渐变过渡到新配色（1秒过渡动画）
- [x] 染色后建筑保留新配色直到再次染色
- [x] NPC服装不随建筑染色改变（独立）

- [x] 装饰品无耐久损耗（永不损坏）

**交互流程**：
1. 玩家按B或点击底部B按钮 → 弹出建造面板（悬浮，可拖拽）
2. 点击分类Tabs过滤建筑类型
3. 鼠标悬停在建筑卡片上 → 高亮+显示消耗预览
4. 点击建筑卡片 → 面板底部显示详情
5. 鼠标移到3D场景 → 跟随鼠标显示虚影
6. 虚影绿色（合法）→ 点击左键 → 确认建造
7. 虚影红色（非法）→ 显示非法原因文字

## 四、关键交互流程状态图

### 4.1 建造流程
```
按B/点击按钮
    ↓
弹出建造面板 ←→ 点击分类Tabs过滤
    ↓
点击建筑卡片 → 面板底部显示详情
    ↓
鼠标移入3D场景 → 跟随显示虚影(绿色/红色+原因)
    ↓
红色虚影? → 显示具体原因(资源不足/承重超限等) → 返回选择
    ↓
绿色虚影? → 点击左键 → 确认建造弹窗 → 确认
    ↓
建筑进入施工状态(半透明+进度条) → 工人自动分配
    ↓
施工完成 → 建筑实体化 → 开始运作
```

### 4.2 BOSS战流程
```
夜晚 → 威胁值判定 → BOSS出现
    ↓
全屏红色警告+慢镜头+BOSS名称显示
    ↓
BOSS从地皮边缘向中心推进
    ↓
3分钟倒计时开始
    ↓
玩家操作：手动瞄准部位/紧急修复城墙
    ↓
↗ 倒计时内击杀 → 掉落物飞入仓库+全屏胜利特效
    ↓              ↓
  BOSS击败成就    威胁值重置为0
    ↓
↘ 超时未击杀 → BOSS摧毁3座建筑后撤退
                  ↓
              显示损失总结
```

### 4.3 传承与新游戏流程
```
主菜单 → 新游戏
    ↓
输入种子(可选) → 选择难度 → 选择成就解锁选项 → 选择传承遗产+遗物
    ↓
预览世界地图(5×4网格，初始4块高亮)
    ↓
确认 → 游戏开始
    ↓
加载界面 → 显示种子码+生物群系名称+初始提示
    ↓
进入游戏 → 初始4块地皮已解锁(未勘探)
```

## 五、响应式UI组件规范

### 面板通用规则
| 组件 | 规则 |
|------|------|
| 所有面板 | 悬浮式，可拖拽，可关闭(×) |
| 面板宽度 | 最大不超过屏幕宽的40%（480px~720px） |
| 面板高度 | 最大不超过屏幕高的80% |
| 超出滚动 | 面板内容超长时右侧出现滚动条 |
| 多面板 | 支持同时打开多个面板，层叠排列 |
| 面板固定 | 点击📌可固定面板位置（不随视角旋转移动） |

### 按钮规范
| 类型 | 尺寸 | 样式 |
|------|------|------|
| 主要按钮 | 120×36px | 金色填充#FFD700+白色文字 |
| 次要按钮 | 100×32px | 蓝色边框#3498DB+蓝色文字 |
| 危险按钮 | 100×32px | 红色填充#E74C3C+白色文字 |
| 图标按钮 | 32×32px | 透明背景+图标 |
| Tab按钮 | 自适应 | 选中时底部金色下划线3px |

### 交互反馈
| 操作 | 反馈 |
|------|------|
| 悬停 | 颜色微变+1.05倍放大 |
| 点击 | 0.95倍缩放动画(100ms) |
| 拖拽 | 半透明跟随+阴影 |
| 无效操作 | 红色闪烁+震动 |
| 成功 | 绿色闪烁+轻声音效 |
