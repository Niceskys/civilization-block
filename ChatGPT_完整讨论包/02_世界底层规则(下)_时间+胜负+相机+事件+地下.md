# 时间系统规则

### 基础时间设定
- [x] 1 个游戏日 = 20 现实分钟
- [x] 是否有昼夜交替：是
  - [x] 昼夜时长比例：各占 1/2（白天10分钟 + 夜晚10分钟）
  - [x] 昼夜对生产的影响：
    - 依赖阳光的建筑（农田、农舍）在夜晚产出暂停（有太阳灯覆盖则不受影响）
    - 持续运行建筑（熔炉房、采掘场等）可继续运作，前提是有工人值守
  - [x] 昼夜对NPC行为的影响：
    - 默认：白天工作，夜晚进入休息/睡眠状态
    - **夜班机制**：玩家可将NPC设为"夜班"（夜晚工作，白天睡觉）
    - **Lv3自动化**：建筑升至Lv3后解锁"自动运转"，夜晚无工人时以 50% 效率运作
  - [x] 昼夜对安全的影响：
    - 夜晚可能生成怪物攻击建筑（详见 7.2 怪物与防御系统）
- [x] 游戏时间控制档位：暂停 / 1倍速 / 2倍速
- [x] 暂停时是否停止所有生产与NPC行为：是
- [x] 游戏时间是否可倒退：否

### 昼夜详细规则
- [x] 夜晚是否需要照明建筑维持部分运作：是（太阳灯覆盖范围可维持农田/农舍夜间产出）
- [x] 是否有季节变化影响昼夜比例：否（EA版本无季节，但有天气系统影响生产效率，详见 1.6 随机事件系统）

### 天气关联
- [x] 天气系统影响昼夜产出：阴天/阴雨/暴风雨对农田减产，详见 **1.6 随机事件系统** § 天气

# 相机与3D视角规则
> 参考方向：《我的世界》模组「下界之治」的斜俯上帝视角，方块化视觉风格

### 基础视角设定
- [x] 核心视角模式：水平360°自由旋转，俯仰角锁定（固定斜45°俯角，不可上下调整）
- [x] 初始默认视角高度与角度：高度15格，斜45°俯角，正北朝向
- [x] 是否支持视角水平旋转：是，旋转角度限制：360°无限制
- [x] 是否支持俯仰角调整：否

### 缩放与平移规则
- [x] 缩放操作方式：鼠标滚轮缩放
- [x] **缩放范围**：最近距离 2 格，最远距离 100 格（5级LOD，详见2.3视觉规范）
  - Lv1 2格：全细节 — 可见内部NPC动画/粒子效果
  - Lv2 2-30格：标准渲染 — 完整外观+清晰纹理
  - Lv3 30-60格：简化渲染 — 纯色块+简单阴影
  - Lv4 60-80格：色块模式 — 仅显示轮廓色块
  - Lv5 80-100格：图标模式 — 缩小为职业图标标记
- [x] 平移操作方式：鼠标右键拖拽地图为主，键盘WASD方向键为辅
- [x] 缩放至最大距离时，是否自动简化建筑显示：是
  - 80-100格范围自动切换至色块/图标模式
  - 中等缩放(30-60格)显示色块+阴影简化

### 3D世界基础规范
- [x] 单个方块建筑的标准尺寸：2m × 2m × 2m 正方体（详见2.3视觉规范）
- [x] 地皮网格是否严格对齐世界3D坐标：是（每格地皮对应一个2m×2m底座）
- [x] 上下堆叠建筑的视觉衔接：无缝拼接（每层严丝合缝，无间隙）
- [x] 地下/被遮挡建筑的显示规则：默认完全隐藏 + 支持层级单独查看切换
  - 默认视图：仅显示当前选中层及以上的建筑，下层被遮挡建筑隐藏
  - 层级切换：可一键隐藏/显示指定层以上的建筑，查看任意层

### 视角交互辅助
- [x] 选中建筑是否显示高亮描边：是，金色描边(3像素宽，#FFD700) + 底部光圈
- [x] 建造预览是否显示3D虚影：是
  - 合法位置：绿色虚影
  - 非法位置：红色虚影 + 显示具体非法原因（如：承重不足、资源不足、地皮未解锁等）
- [x] 是否支持层级单独查看：是（可隐藏上层建筑，只看指定层）
- [ ] 是否支持第一人称视角切换：是 / 否（EA版本可暂缓）

# 随机事件系统
> 方案D：天气 + 周期性危机 + 随机商人/流浪NPC/地陷
> 扩展B：正向爆点（超稀有正面事件）

## 一、天气系统

### 天气类型与概率
| 天气 | 概率 | 持续时长 | 效果 |
|------|------|---------|------|
| ☀️ 晴天 | 55% | 3-8游戏日 | 无影响（默认状态） |
| ☁️ 阴天 | 25% | 2-5游戏日 | 农田/农舍产量 -10%，太阳能灯夜间覆盖范围-1层 |
| 🌧️ 阴雨 | 15% | 1-3游戏日 | 农田/农舍产量 -30%，建筑耐久日损耗×2，树场木材产量 +15% |
| ⛈️ 暴风雨 | 5% | 0.5-1游戏日 | 农田停产，建筑耐久日损耗×3，所有NPC工作效率 -20%，树木有概率被吹倒(随机破坏一座木建筑) |

### 天气切换规则
- [x] 每游戏日结束时判定天气是否切换（20%基础概率）
- [x] 暴风雨结束后强制转为阴天持续至少1日
- [x] 天气切换时有全屏视觉效果（阴天变暗、雨天粒子效果）
- [x] 天气图标显示在HUD左上角

### 天气与怪物
- [x] 暴风雨夜晚：怪物生成概率 +30%
- [x] 阴雨夜晚：怪物生成概率 +10%

---

## 二、周期性危机事件

### 危机类型
| 危机 | 触发条件 | 效果 | 持续时长 |
|------|---------|------|---------|
| 🦠 瘟疫爆发 | 每15-30游戏日随机 | 全体NPC寿命消耗×2，工作效率-40% | 3-5游戏日 |
| 🦗 虫灾 | 每10-25游戏日随机(仅影响有农田/农舍时) | 所有农田/农舍产出降至25%，树场-30% | 2-3游戏日 |
| 🔥 火灾 | 每20-40游戏日随机 | 随机点燃1座木质建筑(房屋/农田/树场/农舍)，需立即维修否则蔓延 | 0.5游戏日(蔓延前) |

### 危机应对
- [x] **瘟疫**：拥有疗愈所→持续时间减半(1.5-2.5日)，效率惩罚减半(-20%)
- [x] **虫灾**：消耗2水→手动驱虫(缩短1日)。哨塔守卫可协助驱虫
- [x] **火灾**：消耗3水灭火。工坊工人自动参与灭火。无灭火→火势蔓延至相邻建筑(每0.5日蔓延1座)
- [x] 危机期间游戏顶部显示红色危机横幅 + 倒计时

### 危机触发规则
- [x] 同一时间只触发一种危机
- [x] 危机结束后有10游戏日冷却期
- [x] 危机触发概率随游戏日递增：前30日无危机，30-60日概率减半，60日后正常

# 地下挖掘与探索系统
> 方案C：深层地下城（10层）+ 工具层级联动 + 加速坍塌难度

---

## 一、基础规则

- [x] 每块地皮可向下挖掘，深度上限：10 层
- [x] 挖掘方式：建造"挖掘入口"（消耗5木材+3铁锭，占该地皮1格）
- [x] 挖掘入口建好后，该地皮出现"向下楼梯"图标，点击可切换视角至地下层
- [x] 地上建筑与地下层互不干扰（地上承重不受地下挖掘影响）
- [x] **必须使用对应等级的工具才能开始挖掘对应层**（见下表）
- [x] 工具在挖掘过程中持续消耗耐久（每层消耗约30%耐久，请提前备好足够工具）

---

## 二、挖掘流程（增加难度版）

### 工具要求与挖掘消耗（修正版）
| 深度 | 所需工具 | 所需时间 | 工具耐久消耗 | 坍塌风险 | 加固消耗 |
|------|---------|---------|------------|---------|---------|
| 第1层 | 铁镐（或更高级） | 1游戏日 | 30% | 0% | - |
| 第2层 | 铁镐 | 1.5游戏日 | 30% | 5% | 5木材+1铁锭 |
| 第3层 | 铁镐 | 2游戏日 | 30% | 10% | 5木材+2铁锭 |
| 第4层 | **铜镐**（铁镐不可用） | 2.5游戏日 | 30% | 15% | 8木材+2铁锭 |
| 第5层 | 铜镐 | 3游戏日 | 40% | 22% | 8木材+3铁锭 |

## 五、挖掘难度说明

```
D1-D3：新手区（铁镐即可，基本无风险）
   ↓
D4-D6：进阶区（需铜镐，坍塌概率开始显著，建议加固）

   ↓
D10：终极区（建议晶能镐，否则效率减半，有隐藏BOSS可能）
```

### 设计意图
- [x] 玩家不能"随便就挖到地下10层"，必须先发展科技+工具制造
- [x] 第4层开始需要铜镐 → 必须先建矿脉挖掘点+熔炉产出铜锭→制造铜镐
- [x] 第7层需要水晶镐 → 必须先积累水晶碎片，再经过工坊制造
- [x] 第10层晶能宝石只能地下挖掘获得 → 形成闭环：挖前几层→得晶能宝石→造晶能镐→挖第10层
- [x] 第10层必出1晶能宝石，保证玩家不会永远卡死
