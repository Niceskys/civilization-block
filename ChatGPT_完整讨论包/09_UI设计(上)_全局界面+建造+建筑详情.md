# 全局界面设计

## UI整体风格
- [x] 布局风格：**极简悬浮式**
  - 所有面板悬浮于3D游戏画面上方
  - 面板可自由拖拽移动、可随时关闭
  - 保持视野最大化，减少固定UI占用

## 主菜单界面
- 按钮：新游戏、继续游戏、设置、退出游戏
- 其他元素：游戏Logo、版本号

## 游戏主界面元素

### 顶部资源栏（可自定义）
- [x] 默认显示核心资源：食物、水、木材、石头、星辉
- [x] 点击展开显示全部11种资源
- [x] 玩家可自定义置顶显示哪些资源
- 悬停显示详细：当前值/存储上限/增减速率

### 快捷功能区
- 时间控制：暂停 / ▶播放 / ▶▶加速（1倍/2倍）
- 当前游戏日 + 昼夜指示器
- 功能按钮：商店、任务、NPC列表、科技、事件日志、设置、存档
  - 商店（快捷键S）
  - 任务（快捷键Q）
  - NPC列表（快捷键N）
  - 🔬科技树（快捷键T，与挑战共用T键切换子页）
  - 🏆挑战（快捷键T，与科技树共用T键切换子页）
  - 📦贸易路线（快捷键R）
  - 📋事件日志（快捷键L）
  - 设置（快捷键Esc）
  - 保存/存档

### 悬浮面板（可拖拽）
- 建筑菜单面板（快捷键B）
- 任务面板（快捷键Q）
- NPC管理面板（快捷键N）
- 商店面板（快捷键S）
- 事件日志面板（快捷键L）

## 设置界面
- 选项：音量、分辨率、窗口/全屏、语言
- 游戏设置：时间流速默认值、自动存档间隔
- 自定义：资源栏显示项配置

## 数据仪表盘（快捷键D）
- [x] 独立的全局统计悬浮面板
- 内容：
  - 总人口统计（成人/婴儿 男女比例）
  - 全局生产效率总览（当前总产出速率 vs 总消耗速率）
  - 资源增长趋势折线图（最近10游戏日）
  - 建筑总览（各类型建筑数量/等级分布）
  - NPC职业分布饼图

## 事件日志
- 需要记录的事件：NPC出生、NPC死亡、生产完成、异常警告
- 建筑损坏、灾害事件、任务完成、恐惧值变化

# 建筑详情UI（通用规范）

### 面板打开与关闭
- [x] 面板打开方式：左键点击建筑
- [x] 关闭方式：右上角叉号 / 点击空白处 / 再次点击建筑
- [x] 是否支持同时打开多个建筑面板：是（悬浮式，可并排查看）

### 建筑内部视图系统（画中画 + 全屏切入）
- [x] **右下小窗预览**：选中建筑后，右下角自动弹出建筑内部3D预览小窗
  - 小窗尺寸：屏幕 1/4 大小，半透明边框
  - 内容：建筑内部3D切面视图（剖开外墙看到内部）
  - NPC工作动画实时播放（见下方动画表）
  - 资源流转粒子效果可见
  - 小窗可拖拽位置、可关闭
- [x] **双击全屏切入**：双击建筑或小窗 → 画面推进到建筑内部
  - 全屏显示建筑内部3D空间（替代全局视角）
  - 可在全屏模式下拖拽工人调整站位
  - 可手动操作内部设备（如切换垃圾处理场模式）
  - ESC / 再次双击 → 退回全局视角
  - 全屏切入时，游戏时间继续流转

### 建筑内部视图系统增强（全屏切入后）

- [x] **多分区内部空间**：每种建筑2-3个功能分区，NPC在不同区间移动工作
- [x] **自由旋转视角**：全屏切入后鼠标拖拽360°旋转观察，滚轮缩放
- [x] **墙壁透视模式**：点击任意墙壁→该面半透明化，可透视后方分区
- [x] **手动操作点**：关键设备可点击手动交互，获得即时效果
- [x] **效率可视化**：NPC工作效率以头顶百分比+身体光晕强度双重显示
