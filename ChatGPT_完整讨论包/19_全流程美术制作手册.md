## 一、总体策略

### 三阶段执行路线
```
第一阶段：MVP（1-3天） → 第二阶段：完善（1-2周） → 第三阶段：精品（1-2月）
  免费素材凑合用         AI工具批量生成            外包/细化
  系统字体             自定义像素字体             精修纹理
  纯色色块             基础纹理贴图               全套UI主题
```

## 二、UI制作（A部分）

### A1. 第一阶段：MVP UI（纯色块+系统字体）

#### 颜色取值表（直接复制使用）
| 用途 | 色值 | RGB | 在引擎中设置 |
|------|------|-----|-------------|
| 面板主背景 | #1A1A2E | (26,26,46) | Panel.Background |
| 面板次背景 | #16213E | (22,33,62) | List.ItemBackground |
| 强调色（选中/金色） | #FFD700 | (255,215,0) | Highlight |
| 危险色（警告/红） | #E74C3C | (231,76,60) | Danger |
| 安全色（绿色） | #2ECC71 | (46,204,113) | Success |
| 信息色（蓝色） | #3498DB | (52,152,219) | Info |
| 文字主色 | #FFFFFF | (255,255,255) | Text.Primary |
| 文字次色 | #B0B0B0 | (176,176,176) | Text.Secondary |

#### MVP UI制作步骤
```
第1步：在引擎中创建UI Canvas
   → 设置 Canvas Scaler 为 "Scale With Screen Size"
   → Reference Resolution: 1920×1080

第2步：创建底栏 Panel
   → Image (纯色) #1A1A2E, 透明度 85%
   → 尺寸: 全宽 × 48px, 锚点底部
   → 添加5个 Button: [🏗️B] [🏛️C] [📋Q] [🗺️M] [⚙️Esc]
   → 每个Button尺寸: 120×40px, 间距12px
   → Button文字: Press Start 2P 字体, 18px, 白色

第3步：创建顶栏资源条
   → 全宽 × 40px, 锚点顶部
   → 背景 #16213E, 透明度 80%
   → 从左到右排列资源图标+数值

第4步：创建悬浮面板
   → 尺寸: 480×640px（默认）
   → 背景: #1A1A2E + Alpha 90%
   → 圆角: 8px
   → 边框: 1px #FFD70040（金色弱化）
   → 右上角三个按钮: 📌 — × (40×40px)
```

### A2. 第二阶段：AI生成UI贴图

#### Midjourney Prompt模板（直接复制，替换[内容]）

**1. 按钮纹理**（生成后在PS中抠出可平铺部分）
```
game UI button, dark fantasy pixel style, glowing blue edges, 
64x64 pixel art, flat design, no text, #1A1A2E background, 
gold accent #FFD700 --ar 1:1 --v 6
```

**2. 面板边框装饰**
```
game UI panel frame corner ornament, blue crystal pixel art style, 
glowing thin lines, corner decoration only, transparent background, 
64x64 pixels, for dark theme UI --ar 1:1 --v 6
```

**3. 资源图标（16×16像素风）**
```
pixel art icon for [food/water/wood/stone/ore/etc], 16x16,
rpg game inventory style, flat colors, transparent background,
clean pixel edges, high contrast --ar 1:1 --v 6
```

#### 图标批量生成策略
```
一次性生成所有资源图标：
Prompt: "16x16 pixel art icons set for 16 resources, 
[food, water, wood, stone, iron_ore, copper_ore, crystal, 
iron_ingot, copper_ingot, crystal_shard, gem, star_coin, 
fertilizer, fuel, science_point, trash], 
rpg inventory style, flat colors, transparent bg, grid layout" --ar 16:1 --v 6

→ 生成后切成16个独立图标（用GIMP或Figma切片）
```

## 五、AI图像工具精确使用指南

### Midjourney快速上手（最佳选择）

#### 第1步：注册
1. 打开 discord.com → 注册账号
2. 访问 midjourney.com → 点击 "Join the Beta" → 关联Discord
3. 订阅（约¥150/月，可生成数百张图，用完取消）

#### 第2步：生成第一张图
在Midjourney的任意频道输入：
```
/imagine prompt:64x64 pixel art seamless tileable wood plank texture, warm brown, game asset, voxel style --ar 1:1 --v 6
```
等待约30秒 → 看到4张结果

#### 第3步：选择变体
- U1~U4：放大某一张
- V1~V4：以某一张为基础生成变体
- 🔄：重新生成

#### 第4步：下载保存
点击放大后的图片 → 右键 → 保存为PNG

### DALL·E 3（备选，免费版有限额）

通过 Bing Image Creator 免费使用：
1. 打开 https://www.bing.com/create
2. 输入相同Prompt
3. 每天有25次免费生成额度

### Stable Diffusion（免费本地运行，需要显卡）

如果有一张6GB+显存的NVIDIA显卡：
1. 下载 Automatic1111 WebUI: https://github.com/AUTOMATIC1111/stable-diffusion-webui
2. 下载 PixelArt 模型: https://civitai.com 搜索 "pixel art" 模型
3. 本地运行，无限免费生成
