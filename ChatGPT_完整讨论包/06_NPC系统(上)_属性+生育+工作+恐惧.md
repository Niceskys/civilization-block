# NPC属性与生命周期

### 基础属性列表（共7项）
- [x] **力量**：影响体力劳动产出（矿场、伐木等），影响承重相关工作
- [x] **敏捷**：影响工作速度和移动效率
- [x] **智力**：影响技术型工作产出（熔炉、研究等），影响职业学习速度
- [x] **体力**：影响工作耐力，高体力NPC可持续工作更久才需要休息
- [x] **创造力**：影响特殊产出概率（稀有资源、高品质产品暴击）
- [x] **社交**：影响生育成功率、恐惧值恢复速度
- [x] **幸运**：轻微影响各方面产出，触发随机暴击几率

### 属性数值规则
- [x] 属性范围：0 ~ 100
- [x] 属性平均值：50
- [x] 正常波动范围：50 ± 20%（即 40~60 占主体约68%）
- [x] 极强属性(>85)出现概率：5%，极值范围：86~100
- [x] 极弱属性(<15)出现概率：5%，极值范围：0~14
- [x] 属性各档位对工作效率的影响（叠加计算）：
  - 属性 71~85：该属性相关产出 +10%
  - 属性 86~95：该属性相关产出 +20%
  - 属性 96~100：该属性相关产出 +30%
  - 属性 15~29：该属性相关产出 -10%
  - 属性 0~14：该属性相关产出 -20%

### 属性获取与遗传
- [x] 属性获取方式：遗传 + 随机变异
  - 子代属性 =（父母属性平均值）+ 随机变异波动
  - 变异概率：60%（每项属性独立判定）
  - 变异幅度：±5~15（均匀分布），有10%概率出现大幅变异±20~30
  - 极端情况下可能出现远超/远低于父母的天赋异禀/缺陷个体

### 传奇特质系统 🌟
- [x] **每个NPC出生时有5%概率获得1个传奇特质**
- [x] 传奇特质列表（随机抽取，不可叠加，已有特质不重复抽取）：

  | 特质名称 | 效果 | 金色边框显示 |
  |---------|------|------------|
  | 🌟 天选之人 | 全属性 +20 | ✅ |
  | 🌟 铁胃 | 食物消耗 -50% | ✅ |
  | 🌟 天才工匠 | 金属/矿石产出额外 +1/次 | ✅ |
  | 🌟 神速 | 移动速度 +50%，工作速度 +30% | ✅ |
  | 🌟 钢铁意志 | 恐惧值免疫（永远为0） | ✅ |
  | 🌟 丰收之体 | 农业产出（食物） +50% | ✅ |
  | 🌟 社交蝴蝶 | 好感度增长 ×2，生育成功率 +30% | ✅ |
  | 🌟 长生 | 基础寿命 +20 游戏日 | ✅ |
  | 🌟 幸运星 | 幸运属性固定95，暴击概率 +10% | ✅ |
  | 🌟 学者 | 智力固定90，科技点产出 +1/日 | ✅ |
  | 🌟 无畏 | 守卫职业效率 ×3（而非×2） | ✅ |
  | 🌟 亲和 | 管理者加成范围×2，幅度+10%→+15% | ✅ |

- [x] **传奇特质遗传**：传奇特质拥有者的子代有 30% 概率继承该特质
  - 若双亲都有传奇特质：子代获得其中1个（各50%概率），不会叠加
- [x] **传奇特质在NPC详情面板显示**：头像旁边金色✨特效 + 特质名金色文字标签
- [x] **NPC死亡显示生平总结**：死亡时弹出面板显示：
  - NPC头像 + 姓名 + 性格标签 + 传奇特质（如有）
  - 一生统计：总产出各项资源数量、总工作时长、生育子女数、击杀怪物数
  - 示例：🧑‍🌾 农民张三 | 一生种出食物 1200 份 | 育有 3 子 | 享年 32 日
  - 生平面板持续显示5秒后自动关闭，玩家也可手动关闭

### 婴儿阶段规则
- [x] 是否占用房屋床位：否（婴儿跟随父母同住，不额外占床位）
- [x] 是否可以工作：否
- [x] 是否消耗资源：是，消耗：0.25食物/游戏日（减半消耗）

### 成长规则
- [x] 婴儿长大时长：2.5 游戏日
- [x] 成长计时起点：出生时刻
- [x] 成年后是否自动加入劳动力池：是
- [x] 成年时是否自动分配/选择职业：否（需玩家手动分配职业）

### 性格系统（详见 5.4 NPC管理UI）
- [x] 出生时随机分配6种性格之一：勤奋/懒惰/乐观/悲观/勇敢/谨慎
- [x] 性格影响工作效率、消耗、恐惧值恢复、移动速度等

### 寿命与死亡
- [x] 是否有老年/自然死亡机制：是
  - [x] 有寿命且可通过后期科技/物品延长
  - [x] 基础平均寿命：30 游戏日（30日×20分钟=10小时现实时间）
  - [x] 寿命浮动范围：±10 游戏日（即 20~40 游戏日）
  - [x] **体质属性影响寿命**：每10点体质 +2 游戏日寿命（体质100→+20日，体质0→-0日）
  - [x] **疗愈所效果延长寿命**：寿命消耗 -30%（详见疗愈所单页）
  - [x] **社会科技Lv1**：基础寿命 +5 游戏日
  - [x] 延长寿命的方式汇总：
    - 商店道具「NPC寿命延长」：+10游戏日（300星辉）
    - 疗愈所建筑：NPC寿命消耗速度 -30%
    - 社会科技Lv1：基础寿命+5日
    - 体质属性：每10体+2日
    - 传奇特质「长生」：基础寿命+20日
    - 未来科技树解锁高级延寿科技

# NPC生育规则

### 生育方式
- [x] 主要方式：自然配对
  - 有空闲房屋床位时，异性成年NPC自动生育
- [x] 辅助方式：玩家可手动促育
  - 玩家可点击2名NPC发起"促育"指令，加速人口增长
  - 手动促育消耗：50星辉/次，冷却3游戏日（同对NPC）

### 生育前置条件
- [x] 是否需要空闲房屋床位：是（至少1空位，婴儿不占额外床位）
- [x] 是否需要食物充足：是（食物库存≥5，水库存≥5）
- [x] 是否需要男女成年且配对：是（异性成年NPC）
- [x] 其他条件：人口未达上限50人

### 生育周期与规则
- [x] 生育周期：每 2 游戏日，一对成年男女可生育一名婴儿
- [x] 生育冷却：一对NPC生育后，冷却 3 游戏日
- [x] 新生儿性别规则：随机 50%/50%
- [x] 多对NPC同时生育：每对独立计算周期，互不影响
- [x] 人口上限是否影响生育：是（达50人停止）

### 其他人口获取途径
- [x] 商店购买NPC：400星辉/人（立即招募）
- [x] 任务奖励NPC：见任务清单（10.2）
- [x] 流浪NPC自然加入：详见 1.6 随机事件系统 § 流浪NPC
- [x] NPC全灭救援：按难度分档（见 7.1 极端情况处理）

# 工作与分配逻辑

### 工作分配方式
- [x] 工作分配方式：点击建筑→选择NPC分配
  - 玩家点击建筑打开工作槽位面板
  - 从空闲NPC列表中选择分配给该建筑
  - 类似RimWorld风格的工作面板交互

### 工人与岗位
- [x] 工人顺序的具体作用：第1位工人100%效率，第2位及之后每人效率递减5%（如2人时：100%+95%）
- [x] NPC是否会自动更换工作：否（需玩家手动调整）
- [x] 空闲NPC默认行为：自动做低效杂活
  - 空闲NPC以 25% 效率随机帮助附近有人的建筑
  - 不会主动抢占已满工人槽位
  - 一旦被玩家手动分配工作，立即进入正常效率状态

### 夜班机制
- [x] 玩家可在NPC详情面板将NPC切换为"夜班"模式
  - 夜班NPC：夜晚工作，白天睡觉
  - 白班NPC（默认）：白天工作，夜晚睡觉
  - 夜班NPC白天不响应工作分配
  - 建议维持至少50%人口为白班（白天防御建筑需工人）

### 属性与工作适配
- [x] 属性与工作适配规则：**偏好属性权重加权**
  - 每个职业有3个偏好属性，按50%/30%/20%权重计算综合效率倍率
  - 公式：综合倍率 = (属性1÷100×0.5) + (属性2÷100×0.3) + (属性3÷100×0.2)
  - 例：守卫(力量/体力/敏捷)，属性值80/70/90 → (0.8×0.5)+(0.7×0.3)+(0.9×0.2) = 0.79 → 效率×1.79
  - 综合倍率范围：0.01~1.0（与职业效率加成相乘）
  - 无职业NPC：取7属性平均值÷100为综合倍率

### NPC死亡规则
- [x] 死亡原因：
  - 自然死亡（寿命耗尽）
  - 玩家击杀
  - 怪物击杀
  - 随机事件致死（灾害）
- [x] 死亡后是否产生尸体/垃圾：是（产生 1 单位垃圾）
- [x] 死亡对其他NPC的影响：全体恐惧值 +5（任何非自然原因死亡）

### 动画播放速度与效率挂钩
- [x] **基础播放速度** = 1倍速（普通NPC，100%效率）
- [x] **高效率NPC**（>150%效率）：动画播放速度 ×1.3
- [x] **属性加权NPC**（属性倍率>0.8）：动画播放速度 ×1.5
- [x] **懒惰性格**：工作阶段之间有额外"发呆"暂停（延长10%总时长）
- [x] **勤奋性格**：阶段切换无停顿（缩短15%总时长）
- [x] **夜班惩罚**：夜晚工作动画速度 ×0.85（除非有太阳灯/哨塔探照灯覆盖）

### 职业效率加成规则
- [x] 职业匹配对应建筑：效率 × 200%
- [x] 职业不匹配对应建筑：效率 × 75%
- [x] 无职业NPC：效率为基准 100%
- [x] 职业加成与属性加成的计算方式：相乘
  - 最终效率 = 职业匹配倍率 × 属性适配倍率 × 性格修正
  - 例：职业匹配×200% + 属性倍率×1.79 + 勤奋性格间歇-30% = 2.0×1.79×1.15 ≈ 4.12倍

### NPC个体交互操作
- [x] **点击NPC**：显示小头像气泡 + 当前状态文字 + 当前工作阶段名

# 恐惧值系统

- [x] 玩家击杀1名NPC，全体恐惧值 +15
- [x] 恐惧值上限：100
- [x] 恐惧值达到100时，虚弱效果具体数值：
  - 工作效率降低 50%
  - 其他负面效果：👉 待补充
- [x] 恐惧值衰减规则：
  - 连续 2 日无玩家击杀，全体恐惧值 -20
  - 衰减计算方式：每日降10（即两日一次性降20）
  - 中途击杀是否重置衰减倒计时：是
- [x] 恐惧值其他增减途径：
  - 怪物击杀NPC：全体恐惧值 +10（怪物入侵事件中）
  - 守卫成功击杀怪物：全体恐惧值 -5（增强安全感）
  - 商店道具「恐惧值清除」：即时清零
- [x] 恐惧值档位效果：
  - 0-49：无负面效果
  - 50-79：工作效率降低 25%
  - 80-99：工作效率降低 35%
  - 100：工作效率降低 50%
- [x] 恐惧值与怪物系统关联：
  - 恐惧值越高，夜晚怪物生成概率和强度越大
  - 恐惧值≥50：怪物生成概率 +20%
  - 恐惧值≥80：怪物生成概率 +40%，可能出现精英怪物

### 恐惧值与好感度联动
- [x] NPC好感度最高的朋友死亡：全体恐惧值 +5（叠加于其他增减）
- [x] NPC好感度最高的朋友被玩家击杀：该NPC好感度清零 + 全体恐惧值额外 +10
- [x] 详见 **3.7 NPC好感度系统** § 与恐惧值联动
