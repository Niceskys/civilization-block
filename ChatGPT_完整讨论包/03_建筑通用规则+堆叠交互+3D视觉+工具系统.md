# 建筑通用规则

### 建造规则
- [x] 所有建筑建造是否消耗资源：是
- [x] 建筑建造是瞬间完成还是需要时长：需要施工时长
- [x] 建造是否需要工人参与：是（需分配工人进行施工）
- [x] 可消耗额外资源加速建造（如消耗双倍资源，建造时间减半）
- [x] 具体建造时长数值：见 6.1 建筑数值总表

### 耐久与维护
- [x] 建筑是否有耐久度/损耗报废机制：是
  - 耐久随使用时间自然损耗
  - 随机灾害/事件/怪物攻击可能导致建筑损坏
  - 损坏建筑需消耗资源维修
- [x] 维修规则：消耗同种建造资源，维修量为原建造成本的 30%
  - 例：农田建造成本=5木材+1水，维修需 2木材+1水
  - 维修后耐久恢复至满值

### 升级与操作
- [x] 建筑是否可以升级：是
  - 可多级升级（每升1级提升产出）
  - 升级后建筑外观发生变化
  - 升级消耗资源，需工人参与施工
  - **Lv3解锁自动运转**：建筑升至Lv3后，夜晚无工人时自动以50%效率继续运作
  - [x] 各级升级数值：见 6.1 建筑数值总表
- [x] 建筑是否可被移动：是（消耗时间，需工人搬运）
- [x] 同一建筑是否可重复建造：是（可重复，但同种建筑越多效率越低）
  - 第1座同种建筑：100%效率
  - 第2座同种建筑：80%效率
  - 第3座同种建筑：60%效率
  - 第4座及更多：40%效率
  - 鼓励垂直堆叠多样化搭配

### 解锁方式
- [x] 建筑解锁方式：
  - 初始直接解锁（基础建筑）：房屋、农田、水井、树场
  - 其余建筑通过商店购买或任务奖励解锁
- [x] 商店解锁的建筑是否为永久权限：是

# 堆叠交互规则

### 光照遮挡
- [x] 光照遮挡判定标准：正上方1格内存在任何实体建筑即完全遮挡（含非标准尺寸建筑）
- [x] 太阳灯生效是否需要消耗资源：是
  - 消耗：1 燃料/游戏日
- [x] 太阳灯最多向下覆盖层数：3 层（含自身所在层）
- [x] 太阳灯水平覆盖范围：本格 + 相邻4格（上下左右十字形）
- [x] **多层太阳灯是否效果叠加**：是（2个太阳灯可覆盖6层深度，效果累加不衰减）

### 相邻加成/惩罚
- [x] 水平相邻加成（同层）：
  - 相同建筑相邻（上下左右），每座+10%效率，上限+30%
  - 采掘场相邻熔炉房：双方+20%效率
  - 水井相邻农田：双方+15%效率
  - 哨塔相邻（连线围成防御圈）：每座+10%攻击范围
- [x] **上下层相邻加成**：暂不设上下层相邻加成（EA阶段可通过管道实现资源连通）

### 跨层资源连通
- [x] 底层建筑与上层建筑的资源是否自动连通：否
  - 默认情况下，每层建筑资源独立
  - 需建造"管道"建筑才能在上下层之间自动传输资源
- [x] 管道连接规则：
  - 一根管道连接上下相邻2层建筑（输入口在下层，输出口在上层）
  - 多层堆叠需逐层铺设管道
  - 管道不占用建筑工人槽位
  - 管道方向一旦建好不可更改（需拆除重建）

### 管道建筑
- 功能：连接同一块地皮的上下层，使资源自动流通
- 建造成本：5木材+2金属
- 建造条件：需在上下层之间有空位（管道尺寸0.5×0.5×2，不占用全格）
- 连接范围：仅连接直接相邻的上下2层

# 建筑3D视觉规范

### 整体风格
- [x] 美术风格：**体素方块风**（对标下界之治）
  - 2m×2m×2m标准正方体为基本单位
  - 所有建筑由方块体组合构成，保持统一视觉语言
  - 材质采用像素化纹理（64×64像素贴图）
- [x] 所有建筑是否为标准正方体：是
  - 单个方块建筑标准尺寸：2m × 2m × 2m
  - 特殊造型建筑突破标准尺寸的清单：
    - 哨塔：1×1×3（细高塔形）
    - 城墙：2×0.5×1.5（扁长墙体）
    - 陷阱：0.5×0.5×0.3（扁平地面装置）
    - 管道：0.5×0.5×2（垂直细管）
    - 研究站：2×2×1.5（略扁，天文穹顶）
    - 太阳灯：1×1×2（高杆灯架）
- [x] 建筑区分方式：**颜色+材质纹理+外形细节 三重区分**
  - 颜色区分：生产类暖色调(棕黄)、居住类中性调(米白)、防御类冷色调(灰蓝)
  - 材质纹理：木材横纹/石材粗糙面/金属光滑反光 各不同
  - 外形细节：屋顶形状、门窗位置、烟囱/天线等辨识元素

### 缩放与视角
- [x] 缩放操作方式：鼠标滚轮缩放
- [x] **5级LOD缩放简化规则**（对应1.5相机规则）：
  - Lv1 最近2格：全细节渲染，可见内部NPC/粒子动画
  - Lv2 2-30格：标准渲染，建筑外观完整，纹理清晰
  - Lv3 30-60格：简化渲染，建筑降为纯色块+简单阴影，隐藏内部细节
  - Lv4 60-80格：色块模式，仅显示建筑轮廓色块
  - Lv5 80-100格：图标模式，建筑缩小为职业图标标记

### 渲染与性能规则（遮挡剔除优化）
- [x] 重复建筑使用实例化渲染优化性能：是
  - 同种建筑使用GPU Instancing批量渲染
- [x] 同屏最大渲染建筑数量上限：500座（超过则最远建筑简化或隐藏）
- [x] 远距离建筑降级显示：是，按距离分级（见上方5级LOD规则）
- [x] **视觉遮挡剔除**：玩家未单独查看的建筑（被其他建筑包围遮挡），内部只进行数据运算：
  - 人物动画不在后台加载
  - 资源粒子效果暂停渲染
  - 仅保留核心数值更新（产出/消耗/耐久计算）
  - 防止内存消耗过高，确保大型堆叠城市流畅运行

### 状态视觉反馈
- [x] 正常工作状态：建筑常亮，窗口透出暖光（白天无色差，夜晚可见）
- [x] 停产/条件不足状态：建筑变暗 + 顶部红色感叹号图标悬浮
- [x] 建造中状态：半透明虚影(50%不透明度) + 进度条环围绕建筑底部
- [x] 拆除中状态：建筑红色脉冲闪烁 + 碎片粒子掉落
- [x] 耐久不足(<30%)：墙体出现裂纹纹理覆盖 + 黄色警告三角
- [x] 升级中状态：建筑外包覆金色脚手架框架动画
- [x] 选中状态：建筑外侧金色描边(宽度3像素) + 底部光圈

## 三、工具使用规则

### 分配方式
- [x] 工具制造完成后存入全局仓库（工具为非堆叠物品，每种工具最多持有5件）
- [x] 分配：打开建筑详情面板 → "工具"标签 → 选择工具分配给该建筑的工人
- [x] 一个工人同时只能使用1件工具
- [x] 工具分配后绑定该建筑（不是绑定NPC），NPC更换时工具留在建筑中
- [x] 工具可随时收回（返还仓库，耐久不变）

### 耐久与消耗
- [x] 工具每次工作（每产出1个游戏日的资源）消耗1点耐久
- [x] 耐久归零时工具进入"损坏"状态（不可使用，但未消失）
- [x] 损坏的工具可在工坊**修理**：修理消耗制造材料的50%
  - 例：铁镐修理=1铁锭（原2铁锭的50%）
- [x] 修理后耐久恢复至满值
- [x] 损坏的工具不占工具持有上限（5件）
- [x] 玩家也可选择"拆解"损坏工具（返还制造材料的25%）

## 四、工具升级系统

### 升级规则
- [x] 每种工具可升级：Lv1→Lv2→Lv3
- [x] 升级消耗：1件同种工具 + 对应等级材料
- [x] 升级后效果：每级额外 +10%效率

| 升级 | 消耗 | 铁镐效率 | 铜镐效率 | 水晶镐效率 |
|------|------|---------|---------|----------|
| Lv1 (基础) | 制造材料 | +25% | +35% | +50% |
| Lv2 | 同种工具×1 + 1铁锭 | +35% | +45% | +60% |
| Lv3 | 同种工具×1 + 2对应材料 | +45% | +55% | +70% |

### 效率叠加公式
- [x] 最终效率 = 基础效率 × (1 + 工具加成%) × 性格修正 × 职业匹配
- [x] 例：农民Lv1 + 铁锄Lv3(+45%) + 勤奋性格(间歇缩短30%) + 职业匹配×2
  = 4食物/日 × 1.45 × (间歇缩短) × 2 = 约11.6食物/日（近似）
