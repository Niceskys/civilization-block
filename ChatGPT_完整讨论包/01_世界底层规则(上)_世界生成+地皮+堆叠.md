# 世界生成与生物群系规则
> 适用如《Minecraft》的随机种子+程序化地形生成

---

## 一、随机种子系统

### 种子生成规则
- [x] 玩家创建新游戏时，可输入**种子码**（字母/数字，最长32位）
- [x] 种子码为空则自动生成随机种子（基于系统时间戳SHA-256截断）
- [x] 同一种子码每次生成**完全相同**的世界布局（包括：地皮类型排列、特殊地皮位置、地下层资源分布）
- [x] 种子码不区分大小写
- [x] 种子码在加载界面显示，支持分享

### 种子对哪些内容生效
| 生成内容 | 种子影响 | 固定内容 |
|---------|---------|---------|
| 初始4块地皮的生物群系类型 | ✅ 完全随机 | - |
| 商店可购买地皮的生物群系类型排列 | ✅ 种子决定排列 | 总地皮数固定（共20块） |
| 特殊地皮（6种）的位置 | ✅ 种子决定 | 总数量固定（≈总地皮数×5%） |
| 地下10层每层的资源内容 | ✅ 种子决定 | 每层总量固定（见1.7） |
| 初始资源量 | ❌ 固定按难度 | - |
| 初始NPC属性 | ❌ 每次独立随机 | - |
| 天气/事件序列 | ❌ 运行时随机 | - |
| 怪物生成 | ❌ 运行时随机 | - |

## 二、生物群系系统（共7种）

### 生物群系生成算法（仿Minecraft）
- [x] 使用Perlin噪声算法生成温度图+湿度图（分辨率：1格地皮=1采样点）
- [x] **温度值** 0.0~1.0（0=极寒，1=极热）
- [x] **湿度值** 0.0~1.0（0=极干，1=极湿）
- [x] 温度+湿度交叉映射到7种生物群系
- [x] 初始4块地皮的生物群系保证可生存（不会全部是极端群系）

### 生物群系映射表
| 生物群系 | 温度范围 | 湿度范围 | 视觉色调 | 特殊地皮出现概率 |
|---------|---------|---------|---------|----------------|
| 🏔️ 雪山 | 0.0~0.2 | 0.3~1.0 | 白色冷色调 | 8%（祥和+高风为主） |
| 🌲 寒带森林 | 0.2~0.4 | 0.4~1.0 | 深绿+灰 | 5%（均衡分布） |
| 🏞️ 平原 | 0.4~0.6 | 0.3~0.7 | 翠绿 | 4%（富饶为主） |
| 🌊 海滨 | 0.4~0.7 | 0.7~1.0 | 蓝绿渐变 | 6%（水源+富饶为主） |
| 🌄 高原 | 0.3~0.5 | 0.1~0.4 | 土黄+棕 | 7%（矿脉+高风为主） |
| 🌴 雨林 | 0.6~1.0 | 0.7~1.0 | 深绿+彩花 | 5%（均衡分布） |
| 🏜️ 沙漠 | 0.7~1.0 | 0.0~0.3 | 黄沙色 | 3%（熔岩为主，极少富饶） |

## 三、世界地图布局

### 地图结构
- [x] 世界地图为 **5×4 网格**（20块地皮），初始玩家拥有中心4块（位置B2/B3/C2/C3）
- [x] 地图边界不可扩展（世界大小固定）
- [x] 未解锁的地皮以灰色迷雾覆盖，购买后揭晓

### 典型地图示例（种子"ABC123"）
```
  A1🏜️  A2🌴  A3🌊  A4🏔️  A5🌲
  B1🌲  B2🏞️  B3🏞️  B4🌊  B5🏜️
  C1🏞️  C2🏞️  C3🏞️  C4🌄  C5🌴
  D1🌴  D2🌄  D3🏔️  D4🏜️  D5🌊
```
- B2/B3/C2/C3 = 初始4块，出生在平原（默认基准群系）
- 玩家向不同方向扩展会进入不同生物群系
- 种子不同则排列完全不同

### 生物群系对游戏策略的影响
- [x] **沙漠开局最困难**：水-10+食物-10，但石头+15 → 适合高手挑战
- [x] **平原开局最平衡**：无加减 → 适合新手
- [x] **雨林开局资源最丰富**：木材+15食物+5，但垃圾速度+50% → 需要快速建造垃圾处理场
- [x] **高原开局防御优势**：哨塔+1格 → 防怪物入侵更容易
- [x] **随机种子保证可玩性**：初始4块地皮的生物群系组合保证至少1块是平原或雨林（可产出食物）

## 四、新游戏流程

### 新游戏创建步骤
1. 输入种子（可选，空=随机）
2. 选择难度（简单/普通/困难/极难）
3. 选择已解锁的成就选项（如有）
4. 选择传承遗产+遗物（如有）
5. **预览世界地图**（显示5×4网格，初始4块高亮，其余灰色）
   - 灰色地皮显示轮廓+编号
   - 无法提前知道未解锁地皮的群系类型
   - 可通过点击灰色地皮花费200星辉侦察（见1.1特殊地皮侦察）
6. 确认 → 游戏开始

### 初始界面加载
- 显示种子码 + 世界地图缩略图
- 显示当前生物群系名称 + 该群系效果摘要
- 如果出生在非平原群系，显示特殊提示
  - "你降落在🌴雨林中 → 木材充足但注意垃圾处理"
  - "你降落在🏜️沙漠中 → 这将是一场生存挑战！"

### 生物群系视觉表现
- 每个生物群系有不同色调的天空盒+地皮纹理+环境粒子
- 雪山：白色地面+飘雪粒子
- 雨林：深绿地面+雨滴粒子+彩色花朵
- 沙漠：黄色地面+热浪扭曲
- 海滨：蓝色水边+波浪声
- 背景音乐随群系切换不同变奏

# 地皮基础规则

### 地皮数量与获取
- [x] 初始玩家拥有地皮数量：4 个
- [x] 地皮获取方式与解锁条件：商店购买、达成成就、任务直接奖励
- [x] 解锁单块地皮的非货币消耗（资源/时间）：4 人开发真实时间半小时
- [x] 商店购买单块地皮的消耗成本：**1000 星辉**

### 地皮属性
- [x] 单块地皮的标准尺寸（单位：格/米）：单块地皮占用一格
- [x] 地皮坐标规则：固定网格，可以旋转和移动
- [x] 地皮之间是否有相邻加成/限制：是（地皮间有加成和限制）
- [x] 相邻加成具体规则：
  - **同类型加成**：相同建筑相邻（上下左右），每座+10%效率，上限+30%
  - **特定组合奖励**：特定配对有额外加成
    · 采掘场相邻熔炉房：双方+20%效率
    · 水井相邻农田：双方+15%效率
    · 哨塔相邻（连线围成防御圈）：每座+10%攻击范围

### 地皮操作
- [x] 地皮是否可被拆除/回收：不可拆除，不可回收
- [x] 可向上建造，可向下挖掘
- [x] 地皮获取方式：手动解锁 / 商店购买 / 达成条件自动解锁

### 地皮勘探系统
- [x] 每块新解锁/购买的地皮在首次使用前，必须经过**勘探阶段**
- [x] 勘探消耗：1名空闲工人 + 1游戏日
- [x] 勘探期间地皮上显示进度条（工人+勘探帐篷动画）
- [x] 勘探完成后随机发现以下之一：

  | 勘探结果 | 概率 | 效果 |
  |---------|------|------|
  | 🟢 无障碍物 | 60% | 可正常建造，无额外奖励 |
  | 🟡 野生资源堆 | 25% | 一次性+10木材/石头（随机） |
  | 🟠 小型遗迹 | 10% | 一次性+30星辉 |
  | 🔴 古代宝藏 | 4% | 一次性+80星辉+2金属 |
  | 💎 特殊地脉 | 1% | 该地皮获得永久特殊类型（见下文） |

- [x] 未勘探的地皮显示"未勘探"标记，不可建造，不可堆叠
- [x] 已勘探完成的地皮显示结果图标（不消失，永久标记）

### 特殊地皮类型（勘探1%概率发现 + 世界地图预设少量）
- [x] 世界中预设存在少量特殊地皮（数量=总地皮数×5%）
- [x] 特殊地皮在商店中购买时**灰色显示**（提示"未知类型地皮"），价格不变（1000星辉），购买后才揭晓类型
- [x] 玩家也可以花200星辉"派遣侦察队"提前揭晓该地皮的特殊类型（侦察消耗0.5游戏日，无需工人）

|---------|------|---------|

| ⛏️ 矿脉地皮 | 该地皮上所有采掘场/熔炉房产出 +30% | 灰色矿石图标 |
| 🌬️ 高风地皮 | 该地皮上所有哨塔攻击范围 +1格 | 蓝色旋风图标 |
| 🕊️ 祥和地皮 | 该地皮上建筑中的NPC寿命消耗 -20% | 白色鸽羽图标 |
| 💧 水源地皮 | 该地皮上所有水井产出 +30% | 蓝色水滴图标 |
| 🔥 熔岩地皮 | 该地皮上所有熔炉房产出 +50%，但建筑耐久日损耗×1.5 | 红色火焰图标 |

- [x] 特殊地皮的加成**永久有效**，与建筑升级/科技/性格加成**乘法叠加**
- [x] 每块地皮最多拥有1种特殊类型（不可叠加）

# 建筑堆叠基础规则

### 堆叠上限
- [x] 单块地皮最大堆叠层数上限：64 层

### 堆叠对齐
- [x] 堆叠对齐规则：允许悬空（建筑不需要严格正上方对齐）

### 承重系统
- [x] 建筑承重规则（下层建筑是否有承重上限）：有承重上限
  - 每层建筑的承重能力取决于建筑类型（见 6.1 建筑数值表 承重值列）
  - 上层建筑总重量不得超过下层承重上限
- [x] **建筑重量计算公式**：重量 = 建造成本 ÷ 10（四舍五入取整）
  - 木材权重1、石头权重2、金属权重3：重量 = (木材×1 + 石头×2 + 金属×3) ÷ 10
  - 例：房屋10木材 → (10×1)÷10 = 1 重量
  - 例：熔炉房15石头+5金属 → (15×2+5×3)÷10 = (30+15)÷10 = 4.5 → 5 重量
  - 例：农田5木材+1水 → (5×1)÷10 = 0.5 → 1 重量
  - 例：陷阱5木材+5金属 → (5×1+5×3)÷10 = (5+15)÷10 = 2 重量
- [x] **堆叠验证规则**：下层承重值 ≥ 上层所有建筑重量之和
  - 例：房屋(承重3)在上方建熔炉房(重量5) → 3 < 5 → **不可建造**
  - 例：城墙(承重6)在上方建熔炉房(重5)+房屋(重1) → 6 ≥ 6 → ✅ 可建造
- [x] 所有17种建筑重量已在6.1中计算可用，开发者按公式自行计算

### 拆除规则
- [x] 建筑拆除规则：拆除返还 75% 建造资源
- [x] 能否单独拆除中间层建筑：是
- [x] **拆除中间层后，上层建筑的处理逻辑**：
  - 拆除后上层建筑进入"悬空"状态
  - 玩家有30现实分钟时间在中间层放入足够承重值的建筑
  - 拆除时弹出警告横幅"⚠️ 上层建筑悬空！请在30分钟内放入承重建筑"
  - 倒计时悬浮在HUD顶部（红色计时器）
  - 超过30分钟仍未修复 → 上层建筑逐一崩塌（每次间隔5秒，从上往下崩塌）
  - 崩塌时NPC自动逃出变为空闲状态，建筑摧毁（**不返还资源**）
  - 成功放入承重建筑后 → 悬空标记解除，倒计时消除
- [x] 拆除时该建筑的工人：自动变为空闲状态，进入空闲NPC列表

### 高度影响
- [x] 堆叠高度是否带来额外影响：是
  - 从第 8 层起开始出现温度和风力影响
  - **温度影响**：每增加1层，建筑效率 -2%，适合种植类建筑
