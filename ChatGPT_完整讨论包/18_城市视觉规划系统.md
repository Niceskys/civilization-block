# 城市视觉规划系统
> 解决"前期堆叠杂乱、后期无法调整"的核心视觉问题
> 基于城市建造类游戏视觉设计研究

---

## 一、核心问题诊断

### 玩家前期视觉痛点
| 问题 | 原因 | 后果 |
|------|------|------|
| 建筑位置随意堆放 | 前期只求功能快速建造，不考虑布局 | 城市看起来像"随机方块堆" |
| 堆叠高度参差不齐 | 不同建筑尺寸不同（1×1×1、1×1×3等） | 天际线凌乱无节奏 |
| 无法后期调整位置 | 当前规则不支持下层建筑移动 | 玩家"将就着玩"→不满累积 |
| 缺少视觉过渡元素 | 建筑之间有空隙但没有填充物 | 城市显得"漏风"不完整 |
| 配色无法统一规划 | 染色只能单个建筑操作 | 前期懒得分批染色→后期更不想改 |

### 视觉设计研究结论应用
- **Gestalt原则**：人类视觉系统自动寻找"规律"和"对称"。杂乱=疲劳，整齐=愉悦
- **第一印象窗口**：玩家在进入游戏的**前3秒**决定是否对这款游戏的"颜值"产生好感
- **《Minecraft》教训**：自由建造虽然自由度最高，但大多数玩家建造的城市"毫无美感"→需要引导
- **《城镇叠叠乐》成功点**：自然地"生长"出美感，因为建筑会自动适配+颜色统一

## 二、解决方案体系（5大机制）

### 机制1：建筑搬迁系统（后期调整能力）

#### 搬迁规则
- [x] **搬迁操作**：选中建筑 → 详情面板 → "搬迁"按钮 → 选择目标位置（该地皮内任意空格）
- [x] **搬迁消耗**：
  - 1-3层建筑：消耗 2 游戏日 + 1名空闲工人搬运
  - 4-7层建筑：消耗 3 游戏日 + 2名空闲工人搬运
  - 8层+建筑：消耗 4 游戏日 + 2名空闲工人搬运 + 消耗5铁锭
- [x] **搬迁限制**：
  - 不可搬迁到不同地皮（只能在同一地皮内移动）
  - 不可搬迁到有上层建筑的位置（必须先拆上层或搬迁上层）
  - 搬迁过程中该建筑停止生产
  - 搬迁完成后耐久不变，等级不变，已分配的工人自动跟随

#### 搬迁动画
- [x] 建筑底部出现搬运支架 → 建筑缓缓升起（悬浮1格高度）→ 平移至目标位置 → 缓缓落下
- [x] 搬运期间工人站在建筑旁边做"推/拉"动画
- [x] 搬迁完成时建筑落地震动粒子效果

#### 设计意图
- 玩家前期可以"先随便放，后期再调整" → 降低前期决策焦虑
- 搬迁消耗足够高（时间+工人） → 不会被滥用来"刷某种便利"
- 但又不至于高到无法承受 → 中后期完全可行

---

### 机制2：网格吸附与对齐辅助

#### 吸附规则
- [x] **建造时**：鼠标拖动建筑时，建筑自动吸附到最近的半格/整格位置
- [x] **吸附精度**：基础建筑1×1吸附整格，0.5×0.5装饰吸附半格
- [x] **对齐线**：建造时显示对齐参考线（从已建建筑边缘延伸的淡蓝色半透明线）
- [x] **对称辅助**：按住Shift键时，显示以当前地皮中心为轴的两侧对称参考线
- [x] **网格参考**：建设模式下地面显示半透明网格（默认可见，可设置关闭）

#### 批量染色
- [x] 按住Ctrl选中多个建筑（框选或逐个点击），然后一次性应用同一种配色
- [x] 批量染色折扣：5座以上染色，染料消耗-20%（鼓励统一配色）

### 机制3：城市天际线奖励系统（Beautification Bonus）

#### 核心逻辑
- [x] 视觉上"好看"的城市布局获得**实际游戏收益**，不是纯装饰
- [x] 玩家不是为了"好看而好看"，而是为了"好看有回报"

#### 奖励规则

| 视觉条件 | 奖励 | 检测方式 |
|---------|------|---------|
| 🏆 **对称布局**：以地皮中心线为轴，两侧建筑类型+等级+层数镜像对称 | 该地皮所有建筑效率+5% | 系统自动检测（玩家不满足会显示缺口） |
| 🏆 **高度有序**：同一地皮上相邻建筑高度差≤1格（不出现3格塔紧挨1格小屋） | 该地皮建筑耐久损耗-10% | 系统自动检测 |
| 🏆 **色彩统一**：同一地皮上≥70%的建筑使用同一种配色方案 | 该地皮所有建筑效率+3% | 系统自动检测 |
| 🏆 **完整填充**：地皮上无空格（所有可建格都被建筑/装饰填满） | 该地皮产出+5% | 系统自动检测 |
| 🏆 **装饰丰富**：地皮上装饰品数量≥3件 | 全城NPC好感+2% | 系统自动检测 |

#### 奖励叠加
- [x] 多个条件可叠加（最高可达+5%+5%+3%+5%+2% = +20%全局效率）
- [x] 奖励在条件满足时自动生效，不满足时自动消失（实时检测）
- [x] 面板中每个条件前显示 ✅（已满足）或 ❌（未满足）和具体缺口提示
- [x] 例：❌对称布局 — 缺口：B2格需要建造与A2格相同类型的建筑

#### 设计意图
- 让"美观"有了量化标准，玩家知道"怎么做才好看和有收益"
- 奖励足够高（最高+20%）→ 值得玩家花心思规划
- 实时检测+缺口提示 → 玩家知道"还差什么"

### 机制5：视角美学优化

#### 默认视角调整
- [x] **初始视角**：调整为从45°斜上方+略微偏移（不是正对网格线），使建筑呈现更立体的"阶梯感"
- [x] **自动聚光灯**：当玩家一段时间无操作（5秒），自动切换到最佳观赏视角（俯瞰城市全景+缓慢旋转）
- [x] **截图模式**（已在问题2C中定义为F12，补充细节）：
  - 进入后隐藏所有UI
  - 可使用WASD自由平移+滚轮缩放
  - 4种滤镜可选（标准/暖色/冷色/复古）
  - 按空格切换"白天/黄昏/夜晚"光照
  - 按C键切换是否显示网格线
  - 截图自动保存至 `文明块\Screenshots` 文件夹

#### 建筑高度视觉引导
- [x] **天际线预览**：建造时按T键切换"天际线视图"（仅显示建筑轮廓剪影，隐藏细节纹理），帮助玩家判断高度是否整齐
- [x] **填色标记**：不同层高的建筑自动在底部显示不同颜色的光晕（1-3层绿色，4-7层蓝色，8+层紫色），帮助玩家快速识别"哪里高度不齐"

## 三、玩家视觉规划指南（新手引导）

### 推荐城市规划流程
```
第1步：用"网格吸附"功能，确保所有建筑对齐网格线
  ↓
第2步：前期阶段，4块地皮上各建1座房屋→ 作为"核心区"定位点
  ↓
第3步：围绕核心区填充生产建筑（农田/水井/树场对称摆放）
  ↓
第4步：每建完5座建筑→ 打开"天际线视图"检查高度是否整齐
  ↓
第5步：中期阶段（D20+），使用"批量染色"统一所有建筑配色
  ↓
第6步：放置装饰品填补空格 → 激活"完整填充"和"装饰丰富"奖励
  ↓
第7步：后期阶段（D45+），使用"建筑搬迁"调整前期随意摆放的建筑
  ↓
第8步：完成对称布局，激活"对称奖励"，获取最终+20%效率
```

### 视觉满意度时间线
```
前3秒：进入游戏 → 网格辅助默认开启 + 对齐线 + 初始视角优化 → 第一印象整洁
前10分钟：新手引导提示"按B打开建造面板，建筑会自动吸附网格"
前1小时：玩家建造了几座建筑 → 自动激活3秒空闲自动全景 → 玩家看到自己的"小城市"
中期（D20+）：使用批量染色 → 城市颜色统一 → 截图分享给朋友的冲动出现
后期（D45+）：搬迁调整布局 + 触发视觉奖励 + 装饰品填充 → 城市"完工"的成就感
```
